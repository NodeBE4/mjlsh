    三年前第一次读野夫《江上的母亲》，第一次知道“母亲”早年坎坷的经历和悲凉的身世。“这是一篇萦怀于心而一直不敢动笔的文章。是心中绷得太紧以至于怕轻轻一抚就砉然断裂的弦丝”。“整整十年都不敢重回那一段冰冷的水域”。不难想象，对作者而言，这无疑是黑暗里心魔争斗和撕心裂肺的十年。

    国庆长假回湖北老家，饭后茶余，家人围坐火炉再次聊起“母亲”，但话题相对轻松、愉悦。酒温热，话题却是老的，微醉之中，思絮仿佛回到童年，回到吊脚楼围成的小院落里。

  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    这是一个有吊脚楼的破旧小院，院子不大，但住了很多人，其中就有野夫一家。整个小院其实是我母亲祖上产业，母亲爷爷那辈兄弟分家，从中间过道
   </span>
   (
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    当地称“水巷子”，邻里担水过道
   </span>
   )
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    一分为二，母亲娘家姓陈，临上街分给福钧舅舅祖上，靠下街分给母亲祖上。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    那年头富裕人家时兴抽鸦片，福钧祖上一抽就抽了个一贫如洗，房产也卖了，结果成了“贫农”。母亲的奶奶
   </span>
   (
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    我称“嘎祖祖”
   </span>
   )
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    也抽，可惜没败光家底，结果成了“工商业者兼地主”。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    二
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    野夫名郑世平，大我一岁多，记不清他们家什么时候搬进这个小院落。在其巜童年的吊脚楼与邻居》
   </span>
   (
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    以下称巜童年》
   </span>
   )
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    里，塑造了很多栩栩如生的人物，几乎囊括了小院所有人，唯独没我，可能是当时太小没“故事”。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    两年前再次联系上野夫，称其“四平兄”。但他说我小时候叫他“四平哥”，还是此称呼更妥，我真佩服他的记忆力。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    世平哥在《童年》里描述很清晰，小院靠后街临“老拱桥”有个很小的吊脚楼，楼上住人，整栋楼已略为偏斜，阳台上木地板部分腐烂，楼梯“吱吱”直响。楼下为猪圈和厕所，公用的，其实很少养猪，厕所没门，一条布帘遮住，入厕时都老远问下“是否有人”。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    吊脚楼上住了四家，上街方向靠前住四平哥一家，靠后为黄爹爹居住，靠下街前面住外婆
   </span>
   (
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    我叫“嘎嘎”
   </span>
   )
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    一家，我跟父母住后面小房。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    三
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    四平哥的外婆是一个有气质，有文化，非常慈祥的老人，一口“荆州”口音，头发盘得整整齐齐，一副大家闺秀模样，所有人包括我们一家四代都称她“婆婆”，甚至大人教襁褓中的婴儿都如此称呼。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    在大家眼里，婆婆是天底下最善良的人，就跟我嘎嘎一样，无论生活如何艰辛，一辈子都没跟家人邻居红过脸。婆婆当年上了岁数，有一年在吊脚楼下的猪圈里养了头猪。当年人都吃不饱，当然猪就又小又瘦，年关请人宰了，除一半上税外，其余都切成一斤左右小块，肥瘦搭配，街访邻里一家一块，分到最后都不知她自己家里剩下一块没有。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    婆婆家里有台缝纫机，当年可是稀罕物件。她不是裁缝，但邻里谁家衣服破了，袜子穿孔了，都是拿给婆婆帮忙缝补。记得母亲跟我说过这么一个故事，说一年冬天，下雪，婆婆患感冒，但她仍从床上爬起来，帮人补袜子。此事被四平哥母亲知道，又心疼又好气，就把已补好的东西扔到地上，害得婆婆委屈得伤心了好几天。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    婆婆是好人，天底下最好的人，是小院和整个街坊的共识。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    四
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    四平哥母亲一头短发，人很精神、干练，在供销社当会计，据说可以双手打算盘，所有人都称呼她“成同志”。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    成同志人很好，经常帮衬乡邻。其实她家也不富裕，但凡谁家没吃的、穿的，她总会撮碗米，拿件衣服之类送与人家。小院不大，人多且穷，但没有哪家没受过其恩惠。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    因家庭成份不好，七
   </span>
   O
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    年我们和嘎嘎两家全部下乡，回到十里外祖上老家，接受贫下中农再教育，临行前成同志还拿出柜里仅剩的几十斤粮票递到我嘎嘎手里。此事《童年》有记述，我也听母亲讲过。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    五
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    四平哥父亲“郑同志”我没太多印象，因为他当年在煤矿当矿长，回家较少，电视剧《父亲的战争》就是以他为原型编写的，解放初随军“剿匪”，“文革”又成了挨批的对象。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    早些时候看望母亲，谈到郑同志，说起一事，仍唏嘘不已。说郑同志曾为一事悔恨终生。他们当年“剿匪”也涉及到我外公，外公解放前夕从“中央警校”毕业后任职警局，在“文沙暴动”中任“督查长”
   </span>
   (
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    见巜利川市志》
   </span>
   )
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    ，起义前夕，派“匪首”之一“花某某”与清剿部队接恰，人刚见面即被杀害，说是“罪大恶极”。其实当时境况，郑同志也是左右不了的，但他一直为此事而歉疚。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    外公最终“起义投诚”，出任百货公司经理，他深信政府，写了很多歌颂党、歌颂革命的政治标语，他把自已义无反顾地投身到轰轰烈烈的革命浪潮中去，像许多热血青年一样，他为自己的政治决择倍感自豪。可天不遂愿，五三年“肃反”，身为“伪政人员”的外公被贯以“反革命”罪遭镇压，从被抓到死就几天时间，时年二十七岁，留下可怜妻女于世，我母亲随即辍学。八十年代初，一张盖有“武汉军区”红印的“平反书”递到我母亲手里。“平反书”不能起死回生，一家人背负“反革命后代”恶名苟活三十余年，这就是我家的宿命。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    六
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    我们下乡接受再教育，四平哥一家搬到供销社仓库隔出的小房间住，就在我母亲奶奶
   </span>
   (
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    嘎祖祖
   </span>
   )
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    屋对面。因嘎祖祖年老，加之有个患小儿麻痹症的聋哑儿子，受照顾免于下乡，也就成了我们幼年时的寄居之所。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    四平哥虽只大我一点，但他自小就表现出了“领导”能力，很快成了孩子王，他带我们一起玩耍，撤尿和泥、打香烟盒、打糖纸、玩“警察抓强盗”游戏。再大点，四平哥上学了，偶尔也带我去学校，他们一帮小同学糊个纸灯，就唱起了“现代京剧”《红灯记》。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    四平哥家有乐器，好像是二胡，他偶尔胡乱拉两下，弄得一帮小伙伴佩服得五体投地。
