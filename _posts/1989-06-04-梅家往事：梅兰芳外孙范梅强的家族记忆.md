   我妈妈是震旦大学，也就是现在的复旦大学文理系毕业的。除了上大学，我妈妈喜欢京剧老生，舅舅梅葆玖学了男旦。这不是故意安排的，学京剧是要按照自身条件选择的。我妈妈大嗓比较好，适合唱老生。
  </span>
<o:p>
</o:p>
</p>
<img alt="17.jpg" border="0" height="426" src="https://i.imgur.com/ymLQHAc.jpg" width="590"/>
<br/>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   福芝芳与孩子们
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   我的大舅舅是同济大学建筑系毕业，毕业后一直在北京建筑设计院任工程师，北京的十大建筑之一军事博物馆他参与了设计。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   二舅舅叫梅绍武，燕京大学毕业，专业是西方文学。他是中国社会科学院美国研究所的研究员，经常写作和翻译文章，也写了很多回忆父亲梅兰芳的书。
  </span>
<o:p>
</o:p>
</p>
<img alt="18.jpg" border="0" height="326" src="https://i.imgur.com/UJwEyQk.jpeg" width="590"/>
<br/>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   梅兰芳的全家福
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   到我这一辈我们家一共有七个小孩：三个孙子、三个孙女加我一个外孙。我们家起名字挺顺应时代的。大哥哥叫卫平，他是
  </span>
  1952
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年生人，正是朝鲜战争的时候，卫平就是保卫和平的意思。二哥哥叫卫华。三哥哥出生时，爷爷正去日本访问，所以就起名卫东。大姐姐叫卫文，大妹妹叫卫红，这两个名字当时都很红，小妹妹名字是卫丽。他们都是卫字辈的。我的名字取妈妈爸爸各一个字。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  2
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   、爷爷不会说扎人家耳朵的话
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   我印象中的爷爷特别谦和。小时候我们兄弟姐妹七个经常排成一串，在晚饭后去找爷爷要糖吃。很遗憾的是，我没有一张和爷爷的合影。其他孩子在爷爷被采访的时候，都和爷爷有过合影，我因为上幼儿园之类错过了。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   那时候住在护国寺，为的就是离人民剧场近，演出方便。我还有些印象，大人演戏我就在后台玩，有一次还差点儿爬到台上去了，搞得观众笑场。还记得我小时候很喜欢到爷爷屋里，从他的展示柜里找好玩的东西，那些都是他从世界各地带回来的纪念品。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<img alt="19.jpg" border="0" height="428" src="https://i.imgur.com/oz8S7PK.jpg" width="590"/>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   梅兰芳指导梅葆玖吊嗓
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   爷爷是心脏病突发去世的。之前已经约好越南总理范文同来家里拜会，虽然爷爷突然离世了，这个约还是要进行下去。我记得是周总理陪着范文同到家里来的。我奶奶说，我那个外孙和你一个姓，也姓范，抱他出来和你见一见吧。于是把睡梦中的我叫起来，换了身衣服，出来见客。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   从梅巧玲（梅兰芳祖父）到梅兰芳，梅家都是性格温和善良的人。比如有人想请爷爷给指教一下，他从来不会说你这么演不好，那么演不对，而是说我在台上是这么演，我觉得这么演更合理，从来都是建议的口气。爷爷碰到票友向他请教，他会说，你是业余的，唱到这样已经非常棒了，我演出的时候这个地方我是这么唱的，我觉得这样行腔更合理。如果是专业的演员向他请教，他会说，今天因为我在这儿，你有些紧张，以后多多实践就好了。总是鼓励人家，不会说扎人家耳朵的话。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  3
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   、“言慧珠你没有你先生的仙气儿”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   梅兰芳画画很好，像虞姬戏服上面的兰花、梅花都是他自己设计的。其实艺术本身就是相通的。他有一度非常迷画画，花费很多工夫。齐如山就劝他，说你是唱戏唱成梅兰芳的，画画画不成梅兰芳，它可以作为你戏之外修养的辅助，画画有张大千、齐白石呢。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<img alt="20.jpg" border="0" height="354" src="https://i.imgur.com/hldix0x.jpeg" width="590"/>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   梅兰芳的画作
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   还有一个故事，也是我奶奶总津津乐道的。言慧珠是爷爷最喜爱的一个女弟子，学梅派学得最为完美。言慧珠学了一出《洛神》，没公演之前进行了一次小彩排，请了好多老先生去看，演完就请老先生们指教一二。其中一位老先生就说，扮相、身段哪儿都好，就是没有你先生的那股仙气儿。言慧珠回家一想，我演的是洛神，没有仙气儿，那不是最大的失败吗？
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<img alt="21.jpg" border="0" height="500" src="https://i.imgur.com/X5edS71.jpg" width="495"/>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   梅兰芳（左）与言慧珠同台演出
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   之后言慧珠专门去请教那位老先生，请他说说仙气儿是怎么来的。老先生让她问梅兰芳去。梅兰芳说我也没什么特殊的，就是你那么演的。后来老先生指点言慧珠，说你先生会画画，会写字，会弹钢琴，这些你会吗？你有没有你先生的这些素养？所以素养是滴水穿石的，不是立竿见影的。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  4
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   、齐如山走了，爷爷就一直坐在椅子上发愣
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   电影《梅兰芳》上映后，好多人就问我梅兰芳和齐如山的关系是怎样的，实际上他们两人没有隔阂。齐如山希望梅兰芳跟他一起去台湾，他希望梅兰芳继续创作和唱戏，但梅兰芳说京剧姓京，我跑到说闽南话的地方唱给谁听？
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   我后来看过齐如山的女儿写的齐如山回忆录，说两个人那时每天都见面，一起说戏，关系特别好。两个人因为去留问题争论了好长时间，分别时，齐如山来家里，也没太多话，呆了一会儿就要走，我爷爷也没送出来。家里佣人于妈想留齐如山吃饭，齐如山拽着于妈的手说：我要出趟远门，就把大爷交给你了。齐如山走了，爷爷就一直坐在椅子上发愣。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<img alt="22.jpg" border="0" height="263" src="https://i.imgur.com/IS1F18d.jpg" width="345"/>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   齐如山与梅兰芳（左）
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  1961
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年
  </span>
  8
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   月
  </span>
  8
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   日，梅兰芳去世。齐如山当时在台湾家里，他的女儿进门说：不好，梅兰芳去世了。据说当时齐如山“啊”了一声，也愣住了，就让屋里人都出去，他要自己呆一会儿。他在书房里通宵达旦写了一晚上，写白了头发。第二天一早，交给女儿一篇纪念梅兰芳的文章，让她拿到《中央日报》发表，《中央日报》全文登载。每次说到这些我都会很难过，这老哥俩儿是一种什么样的交情！
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  5
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   、奶奶说：可惜孟小冬没给大爷留下一男半女
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   我的大奶奶王明华曾经生养了两个孩子，生第二个的时候子宫摘除，无法再生育了，让人心酸的是两个孩子又先后夭折了。爷爷周围的人觉得梅兰芳的艺术必须得有人继承，于是撮合了他和福芝芳的婚姻。我奶奶
  </span>
  19
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   岁进梅家门，生的第一个孩子抱给王氏夫人，说大奶奶，这孩子是我给您生的。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<img alt="23.jpg" border="0" height="395" src="https://i.imgur.com/LsnUlS8.jpg" width="590"/>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   梅兰芳与福芝芳
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   福芝芳的月子是在王氏夫人房里做的，王氏夫人给小孩织了一身小衣服，出了月子又把娘儿俩送回来。这是过去的妇女，她们的关系是这样的。电影《梅兰芳》这一段拍得不好，生活中她们俩根本不是“刀光剑影”的那种样子。我奶奶也是一个性情很温和的人。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孟小冬后来在台湾出的磁带，我妈妈很爱听，但都是背着我奶奶听。有一天，我奶奶向我要，听完以后奶奶说：她这两口唱得还真不错，可惜没给大爷留下一男半女。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<img alt="24.jpg" border="0" height="426" src="https://i.imgur.com/3UltXhn.jpg" width="300"/>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孟小冬
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   我觉得奶奶能说出这几句对孟小冬的评价很不容易，包括后来她听从我爷爷的话将家里的文物全部捐出，这些事让我觉得她很伟大。
  </span>
