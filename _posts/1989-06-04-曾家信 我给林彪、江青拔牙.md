 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    江青
   </font>
  </span>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    <br/>
   </font>
  </span>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    江青的保健医生走后，
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    “中保办”迟迟不派新的医生来，估计是难派。汪东兴就让我们给她开安眠药，我们再三推辞，说不懂内科用药。汪东兴说没关系，就照她以前的处方加加减减即可。我们勉为其难开了处方，还是提心吊胆。江青说：“我看过你们
   </span>
   (
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;
mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    指洪民主任和我
   </span>
   )
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    的档案，你们学过药理学。”解放前的大学文凭背面有学生各科的成绩表，她可能就是从那儿看到的。
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    有一天晚上，汪东兴跑来，说江青“不太好”
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    ，让我们赶过去看看。原来是关于拔牙的问题。她同意拔牙，但是说我们“太紧张”
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    ，她不放心。过了几天，周总理来钓鱼台接见我们，同来的还有康生。那次是由我向周总理汇报治疗方案，周总理听后说
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    ：不要紧张嘛。于是决定拔牙。先由洪主任给她颊侧注射麻药，腭侧未注。一是腭侧进针痛，二是注射后可能有恶心。因为这是一颗松动牙，在腭侧龈沟内上碘酚，也有止痛作用。结果病牙拔除很顺利，她没有叫痛，我们也松了一口气。周总理再进来时，手术已结束。
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    在钓鱼台住了近两个月，江青请我们看了钢琴伴奏《红灯记》
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    。
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    工作人员预先告诉我们，看完一定要说“好”
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    。当然我们不会如此之傻，说“不好”。
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    给江青治病，体力上倒很轻松，每天可以在钓鱼台优美的环境里散散步，看看书。当时虽未预见到可能发生什么严重后果，但是在与江青的接触过程中，逐渐看出她的反常性及毫不掩饰的无理行为，精神上总感到紧张不安。我们就这样不知深浅地完成了那一段
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    “出诊”任务。
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    后来有一次给汪东兴看牙，从他的话语里，我才觉察到一些不寻常的“口风”
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    。汪东兴待医生如客人，十分客气，总是先坐下陪我们喝茶，聊一会儿天。他不止一次地说过
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    ：“医生犯错误是有的，但是不会害人的。
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    ”粉碎“四人帮”以后，刘院长调离
   </span>
   301
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;
mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    医院前，他才告诉洪主任
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    ：江青说，我和洪民给她吃了毒药，叫总后领导处理。总后将此事压下了，刘院长也将此事压下来了。据说是汪东兴保了我们。好在我们懵懵懂懂、糊里糊涂过来了。
