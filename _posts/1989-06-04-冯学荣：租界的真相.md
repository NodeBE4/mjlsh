    宫慕久和英国人签订的开辟上海租界的条约，名叫《上海租地章程》，签署于
   </span>
   1845
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年，当时，英国军队早就从大清国的土地上撤光很久了，与上海道台宫慕久谈判并签约的，是英国领事“巴富尔”，一个文官，不是武官，更没有英国军队参与谈判。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    这份《上海租地章程》，有这样的文字：“……为晓谕事：前于大清道光二十二年（一八四二年）奉到上谕内关：英人请求于广州、福州、厦门、宁波、上海等五处港口许其通商贸易，并准各国商民人等挚眷居住事，准如所请，但租地架造，须由地方官宪与领事官体察地方民情，审慎议定，以期永久相安……”
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    这段文字是一个重要信号：道光皇帝要求宫慕久在安排英国人入住上海的时候，要小心，不要闹出什么问题来，要“永久相安”。怎么样才能“永久相安”？那就只有划租界了：把英国人和中国人隔开来、互不往来，这样才能“永久相安”。这里，划租界的动机交待的很清楚：并非出于英国军队的胁迫，而是出于皇帝老板要求“永久相安”的政治压力。有兴趣的读者，可以去研究一下这份《上海租地章程》。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    不但上海租界是自愿划割的，旧中国二十七个外国租界，自愿划割的还有的是，我再举个例子：厦门鼓浪屿租界，也是清政府自愿、主动划割的。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    甲午战争之后，清政府害怕日本染指福建，故使出“以夷制夷”老招数、把西洋鬼子引入厦门，闽浙总督许应骙在给厦门地方官的电报里，写得很明白：“……引洋人来鼓浪屿，兼护厦门、东防日本……”。
   </span>
   <o:p>
   </o:p>
  </font>

  <o:p>

   </font>

 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    在这里，一个“引”字，已经把话说得很明白了，我就不深入罗嗦了。关于鼓浪屿租界自愿划割的这则史料，出自于厦门政协编的《厦门文史资料第
   </span>
   16
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    辑》。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>

 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>

 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    这还不算，我再举一个例子：天津美租界，也是清政府不但自愿、而且是主动划割给美国的。
   </span>
   <o:p>
   </o:p>
  </font>

 <p class="MsoNormal">
  <o:p>

   </font>
  </o:p>

 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    一开始，美国领事受宠若惊，还不敢要，举棋不定之下，美国领事拍电报回华盛顿请示，华盛顿正在忙南北战争呢，无暇回复，美国领事又不敢贸然放弃（怕华盛顿事后追究责任），于是只好暂时接管。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   1880
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年、
   </span>
   1896
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年，美国先后两次向清政府提出归还天津美租界，可是，你知道清政府的反应是什么吗？清政府的反应是：不作反应。到了
   </span>
   1902
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年，美国领事无奈之下、只好将天津美租界转送給英国，并入了英租界。有兴趣的读者，可以读一下《天津通志：租界志》，才能对天津租界的历史，有比较客观的了解。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    其实，大清国自愿给人划租界这事儿，还有一个旁证，那就是比利时、意大利这些军事小国，也和大清国签约、拿到了租界。对于比利时、意大利这些军事实力比自己弱小的国家，大清国都赋予了他们租界，所以，如果再说划租界是被迫的，实在是没有人相信的。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    再举一个旁证：
   </span>
   1843
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    年，英国人想租用广州十三行对岸的河南田地数十亩，但是该片地的地主和当地群众都不同意，不想出租，最终结果怎样呢？依据史料《广州文史资料第
   </span>
   44
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    辑：广州租界史大事记》的记载：英国人“未逞”。“未逞”的意思是：英国人也只能干瞪眼。这是证明清政府没有受到武力威吓和逼迫的一个具体的案例证据。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>

 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    事情的根本真相在于：划租界并非出于被迫，而是清政府根本不想管洋人，让洋人自己管洋人，是清政府最愿意的事。况且出租地块又有地租可收（下文会提到），所以清政府并不感觉是多大的事儿，只是爱国青年急了，这叫什么？这叫皇帝不急太监急。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    以上是大清国自愿给别人划割租界的事实，但是，还有让你更掉眼镜的：中华民国，也自愿给洋人划过租界。
   </span>
   <o:p>
   </o:p>
  </font>

 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    例如，二次革命时期，为了解决革命党人盘踞在法租界搞颠覆政府活动的难题，
   </span>
   1914
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年
   </span>
   4
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    月
   </span>
   7
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    日，袁世凯的北洋政府与法国驻上海领事签订了一份条约如下：
   </span>
   1
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    、法国承诺中华民国：法租界不得窝藏革命党，一有查出，立即逮捕法办、或驱逐出境；
   </span>
   2
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    、中华民国政府同意：法租界面积扩展至
   </span>
   15150
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    亩。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>

 <p class="MsoNormal">

  </o:p>

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
    北洋政府自愿划割（扩展）租界的这个事实，是依据《上海租界志》第一篇第一章第三节《租界扩张》。
   </span>

    租界真相二：租界并非是免费的
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    洋人租用大清国的土地，基本上，都是需要付费的，要么是付费给清政府，要么是付费给民间的地主。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    其中，《上海租地章程》第六条规定：“商定地价”。也就是说：英国人租用上海滩，不但要付租费，而且费用还是“商定”的，换言之，并非是英国人单方拍脑袋决定的。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    再举个例子，
   </span>
   1843
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    中英《虎门条约》白纸黑字写得很明白：英国人在中国租地，租金“以当地市价为准”。例如，当年
   </span>
   11
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    月
   </span>
   25
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    日，英国人在广州租了一块地，租期
   </span>
   25
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年，每年租金六千元洋银。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    不但如此，得知英国人出手阔绰、一租就租一大片地，有的广州人竟然把英国人当作“大客户”来看待。
   </span>
   1846
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年
   </span>
   6
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    月，广州市民蔡老六，购买了广州沙螺西塱堡内寺岸村安姓的围地一共十六亩多，以及其它海旁地若干亩，拟租給英国人谋利。但是，蔡老六遭到当地村民的抗议、并阻止了这笔生意。
   </span>
   <o:p>
   </o:p>
  </font>

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
    租界真相三：中国人民对租界趋之若鹜
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    首先讲讲中国人入住租界的历史由来。起初，清政府为了不滋生事端，在《上海租地章程》里明文规定：不准英国人在租界内把房子租给中国人（换言之：不准中国人入住租界）。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    后来，小刀会暴动，上海人民不顾一切、涌进租界躲避战火，使租界人口暴涨了几十倍。英国商人看到了商机，他们搭起“经济适用房”、出租给躲避内战战火的中国人牟利。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    英国商人纷纷说：“我们来中国，就是为了挣钱！中国人来租界住了，我们更好挣钱！”于是，英国领事开始吸引华人入住租界，甚至开始阻止中国人外流。以上史料，出自《上海租界志》
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    缺口一旦打开，中国人开始向租界蜂拥而进。举例说说天津英租界，这个地块在未开辟之前，仅仅住有十几个中国人，人烟荒凉。租界建立后，由于司法清明、治安良好、远离内战、没有匪患等原因，大量中国人涌进来居住。到
   </span>
   1938
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年，天津英租界已有人口
   </span>
   76815
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    人，其中中国人
   </span>
   72087
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    人，占总人口的
   </span>
   94
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    ％！以上史料，出自《天津通志：租界志》
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    不但如此，晚清、民国时期，中国多少富人、官僚、明星、革命家拼了老命往租界挤。例如，
   </span>
   1923
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年，中华民国总统黎元洪，竟然宣布：中华民国中央政府，迁往天津英国租界办公！堂堂一个大国，连它的中央政府，都要往租界挤，这是什么笑话？这不是什么笑话，这是真实的历史，是出自费成康《中国租界史》（上海社会科学院出版社）的真实历史。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    连著名的“民族脊梁”鲁迅，也长期住在上海（日本）租界里面，而且他还说过“租界好”的话。
   </span>
   1933
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年
   </span>
   1
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    月
   </span>
   26
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    日，鲁迅在日记里写下了这样的诗：“依旧不如租界好，打牌声里又新春。”。鲁迅这个自白，出自人民文学出版社
   </span>
   2005
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年
   </span>
   11
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    月版《鲁迅全集》第
   </span>
   16
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    卷第
   </span>
   356
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    页。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>

 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    不但亲日人士喜欢往租界挤，连抗日英雄、民族魂，都喜欢往租界挤。例如：张自忠曾经有相当长一段时间，住在天津英租界。为什么象张自忠这样的抗日英雄、爱国烈士，住在英国租界里面？这问题你得问问张自忠。张自忠的案例，出自天津人民出版社《近代天津名人故居》这本史料。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>

 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
    正如前文“小刀会起义”案例，中国人民喜欢往租界挤，还有躲避内战的因素。例如老天津卫的知名企业家刘锡三。刘锡三本来在华界办草帽厂，可是遇上民国初年无休止的内战，根本无法安心做生意，刘锡三无奈之下，将工厂迁到了天津法租界之内，才有了和平的经商环境，企业才因此而做大。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>

 <p class="MsoNormal">
  <o:p>

 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    刘锡三在天津法租界开办的“盛锡福帽厂”，由于经营得当，竟然打败了当时风靡中国的日本草帽、将日本草帽生生逐出了中国市场。刘锡三的案例，出处为《天津文史资料选辑第
   </span>
   31
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    辑》第
   </span>
   181
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    页，张鹏程的回忆文章《驰名中外的天津盛锡福帽厂》。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>

  <o:p>

 <p class="MsoNormal">

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    再举一个例子：
   </span>
   1932
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:

    年，红军杀入福建省漳州县。在此前听到风声的厦门人民，已经有好几千人涌进鼓浪屿公共租界避难，红军实际杀入漳州之后，又有
   </span>

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

    租界真相四：租界法治比较清明
   </span>
   <span lang="ZH-CN">

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

   </span>
   1850
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年代的时候，上海租界里的英美巡捕，每次抓到中国人犯，都移交给上海县的清朝衙门办理。每天大约
   </span>
   20
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    起刑事案件左右。上海县衙门借口看不懂英文案卷，往往草率放人。重获自由的犯人往往再次潜入租界、继续犯罪。洋人对上海衙门的做法很不满。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>

 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
    这里透露了两个信息：
   </span>
   1
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    、在上海租界，外国警察抓到中国贼，是依照条约、送到清政府衙门的；
   </span>
   2
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    、清政府衙门放纵犯罪。这是很鲜明的对比。

  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
    读者要了解上海租界的更多详细细节，请参阅上海官方编写的《上海租界志》，里面有许多客观的真相以及翔实的数据。在此不再赘述。
   </span>

    对待小贩，租界也做得不错。
   </span>
   1928
   <span lang="ZH-CN" style="font-family:

    年，厦门鼓浪屿遍地是小贩，不但堵塞道路，而且有碍市容。但是鼓浪屿租界工部局与华侨王其华合作、建了一个“鼓浪屿市场”，将路边小贩安置到市场内、统一管理，给了小贩一条活路。以上史料，请参考厦门政协编《厦门文史资料第
   </span>
   16
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

   </span>
   40
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    页。

    租界的法治直接促进了大清国的司法改革，当年外国殖民者对清政府明着说：“只要大清国改善司法公正问题，我们可以归还租界”。这件事直接刺激了满清政府、并启动了晚清司法改革，废除了凌迟、枭首等一系列野蛮的酷刑，并建立了大清国的律师制度和人权保护的初步框架。
   </span>
   <o:p>
   </o:p>
  </font>

    租界真相五：租界侵犯了官家的利益
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    租界侵占的是中国的两大权力：行政管理权、司法主权。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">

 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    大清国的行政管理权来自于皇帝的指派。侵略者侵犯的是大清国皇帝和官家的权力。治外法权，直接冲击了大清国的法律。

minor-fareast">
    对于百姓来说，租界给他们带来的就是消费、就业、市政建设等。

    租界真相六：大清国在外也有租界、治外法权
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>

 <p class="MsoNormal">

 <p class="MsoNormal">

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

    早在
   </span>
   1871
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年，大清国和日本签署了《清日修好条规》，文中有如下的约定：“……两国指定各口岸，彼此均可设置理事官，约束己国商民，凡交涉财产词讼案件皆归审理、各按己国律例核办……”
   </span>

minor-fareast">
    换言之，依据《清日修好条规》，大清国在日本享有治外法权，日本也在大清国享有治外法权。
   </span>
   <span lang="ZH-CN">

 <p class="MsoNormal">

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

    这还不算，
   </span>
   1882
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    年，大清国又在朝鲜取得了治外法权。
   </span>
   1884
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年，大清国更是在朝鲜取得了“仁川清租界”（即：中国租界）。大清国还在朝鲜驻军过呢……
   </span>

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

    租界真相七：租界的本质是保护经商安全
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
    租界，无论是外国在中国攫取的租界、还是中国在外国攫取的租界，其存在的深层次动机，都是外资对投放地司法状况的严重不信任。又要去那里投资做生意，又担心自己的人身财产安全，怎么办呢？租界作为一种解决方案，才得以横空出世。
   </span>

    事实上，中国在
   </span>
   1978
   <span lang="ZH-CN" style="font-family:宋体;

   </span>

    依照《中华人民共和国刑事诉讼法》，中国的人民检察院是不能随随便便逮捕外国人的，在逮捕外国人之前，地方检察院需要和省一级人民检察院、外事办进行沟通和磋商。有兴趣的读者可以深入了解《刑诉法》，我在此就不赘述了。
