minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   平米，绝对是当年的住房富裕户。可现在回想起住进新居头一二年的日子，还有几分后怕－－
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   我独自住在有三个大“陷阱”的二楼，实际居住的只是容纳
  </span>
  1
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   张小床的几平米地方，不是“蜗居”更是“蜗居”。一不小心，还会从二楼摔到地下，“一失足成千古恨”，甚至连性命也难保！每天晚上，我小心翼翼地走过有“陷阱”的客厅，再小心翼翼地走进有“陷阱”的小房，然后小心翼翼地爬上小床，一觉就睡到大天亮。由于每时每刻都小心谨慎，我“常在河边站，就是不湿鞋”，成功避免了发生“安全事故”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   搬进自建的新居后，全家也第一次体验到用瓦缸小便的生活。由于新居没建卫生间，而“长寿岭”的公厕在岭脚，离我家约
  </span>
  150
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   米，全家人只能在屋边自家的地盘自备一个瓦缸小便。每天，在中老年农民“尿么”的吆唤声中，将保存的尿按质论价出售。有时候尿液被雨水淋淡，白送给农民也不要。尽管尿缸子只是放在屋边，尿液难闻的气味也不时弥漫在空气中，弥漫进屋内。后来，我家建了卫生间，尿缸子理应退出历史舞台，然而好心肠的母亲依然用尿缸子存小便，无偿送给收尿的农民。家中依然不时弥漫着尿液难闻的气味。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   住进新居后，由于欠下巨额债务，全家开始了一段时间艰苦奋斗的生活。先是在屋边自家的一角地里种菜，省下部分买菜钱。看见屋前和屋边已规划为宅基地的地块暂时没人建屋，也没人种庄稼，就抓住机会，分别种上红薯和花生。几个月后，红薯和花生均取得好收成。其中最大的一只红薯重
  </span>
  1
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   斤
  </span>
  3
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   两，舍不得吃，放着窗台上展示。收获的红薯苗除选取一段作种苗外，余下部分还可作猪饲料。在赶集日，我独自将之挑到农贸市场出售，一次挑四五十斤，卖得二三元或四五元钱。我家还买回几只小鹅饲养，每天赶着小鹅到草地吃草，或买或到地里捡小红薯作鹅饲料。几个月后，小鹅终于长成大鹅。由于缺钱，除了留
  </span>
  1
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   只最小的鹅自家吃外，其余几只拿到农贸市场出售。为节省运费，我和妹妹赶着几只活生生的大鹅，沿着如今的长寿路、解放路，一直赶到农贸市场
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   如今市区同德城正门前
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，再用竹子做成的围栏把鹅围起来，等待买主。刚把鹅围好，就来了个经营烧鹅的大买主，讨价还价后以每斤
  </span>
  6
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   角多的价钱一下就买了四五只。剩下
  </span>
  1
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   只很快被一名干部或教师模样的中年人买走。算算，五六只舍不得吃的大鹅只卖了
  </span>
  20
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   多元钱，一路上赶鹅还冒着好大的风险！
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在全家为欠下的债务发愁、盘算着艰苦奋斗的岁月何日到头之际，母亲的好妹妹、我的好姨姨再次伸出援手，慷慨解囊寄回大额外汇，帮助我家还清欠下的全部债务。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   我家经济好转后，当务之急是堵上二楼楼面的三个大“天窗”。买回钢筋水泥和沙石后，就在我家附近的空地上，按原来的尺寸预制了十几块水泥板，再一块块搬到二楼。从堵上三个大“天窗”之日起，全家人才真正告别了蜗居生活。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   此后，我因上山下乡离家落户几十里外的水果场，后被抽去农村连续参加
  </span>
  4
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年“党的基本路线教育运动”，又在果场、在农村，开始了新的蜗居生活。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1974
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年至
  </span>
  1978
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，我在吴川农村共生活了
  </span>
  1400
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   多个日夜，也见证了吴川农民的真实居住状况：数十万吴川农民的居住条件比城里人更糟糕
  </span>
  ;
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   广阔农村几乎看不见两层或两层以上的民居，仅有约二三成农家建起红砖瓦屋，其余农家居住的是泥砖瓦屋或泥砖茅草屋。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   如今，吴川各地的农家已相继建起媲美城市的高层住宅楼，一些农村还保留着六七十年代的泥砖茅草屋，作为社会变化和发展的物证。如吴阳镇蛤岭村的村民，六七十年代居住的全是低矮的茅草房，居住环境恶劣。近年，该村已建成吴川乃至全国最美的新农村，并于
  </span>
  2009
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年建成举世无双的新农村展览馆。展览馆二楼是“昔日蛤岭”、“今日蛤岭”展厅，通过今昔对比，展现了蛤岭村几十年的变化和发展。而
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “昔日蛤岭”展厅，就摆放着该村村民六七十年代居住的低矮茅草房。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   朋友，要看吴川新民居，请到吴阳蛤岭村来！
