   <font size="3">
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
     所谓“公私合营”，即对资本主义工商业进行社会主义改造，按照
    </span>
    1956
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
     年高潮时的政策，国家对资本主义私股的赎买改行“定息制度”，统一规定年息五厘。生产资料由国家统一调配使用，资本家除定息外，不再以资本家身份行使职权，并在劳动中逐步改造为自食其力的劳动者。也就是说，公私合营后，每年企业向原私股持有人支付
    </span>
    5%
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
     的定息，作为分红，期限
    </span>
    20
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
     年。根据当时的情况，这一定息不但低于多数企业的盈利，也低于银行当时的定期存款利息。
    </span>
    1966
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;
mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
     年
    </span>
    9
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
     月，国家停止发放定息，公私合营企业最后转变为社会主义全民所有制。按照现在的观念，等于一夜之间私人股份被收归国有。
    </span>
    <o:p>
    </o:p>
   </font>

    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

     上海的“公私合营”是在
    </span>
    1956
    <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:

     年全部完成的，但开始则始于三年前的
    </span>
    1953
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
     年。这一年，中央下发了《关于利用、限制、改造资本主义工商业的意见》，并提出了中国共产党在过渡时期的总路线，“对资本主义工商业的社会主义改造”即成为总路线的重要内容之一，而“对资改造”最重要的实现形式就是“公私合营”。具体负责这一工作的中共中央统战部长李维汉在向中央的口头报告中提出，“上海是重点，可多搞一些”。“今年进行公私合营工作的重点应在上海，其他地区则可少搞一些。因为上海是全国经济的心脏，对全国影响也大，各地资本家也都是看着上海的，因此，上海的工作一定要做好，一定要稳重，并希望从上海取得一套工作经验。”
    </span>

    </o:p>
   </font>

    </span>

    </span>
    1954
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

    </span>

    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

    </span>

    </span>

    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

     为什么“资本家”们这么积极？根据当时市委工业生产委员会的归纳和总结，“资本家在对待公私合营态度方面就已经申请的资本家而言，动机不一，有的是认为迟早要合营，看风使舵、迟走不如早走，争取趁头班车；有的看到别人申请，自己不申请不好顾面子；有的是通过申请进行试探；有的认为申请也不一定选到他，投机取巧；有的看到工作组或派员下厂，认为政府已看准，敬酒不吃吃罚酒不漂亮，不如早申请；有的企业困难，希望通过公私合营，渡过难关；有的企图趁机放包袱。”
    </span>

    </o:p>

    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
     除了这些外，开始于
    </span>
    1952
    <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
     年的“三反”、“五反”运动也对“资本家”们造成了比较大的影响，据记录，运动中工人店员扬眉吐气，对资本家呼来吼去，揭发检举此起彼伏。资本家则人人自危，纷纷“坦白”做交待，许多人主动低头认罪争取过关。上海在“五反”运动中，四个月自杀企业主及家属就达到了
    </span>
    876
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
     人。巨大的压力使很多人失去了继续经营的信心。
    </span>
    <o:p>
    </o:p>
   </font>
  </p>
  <p class="MsoNormal">
   <o:p>
    <font size="3">
    </font>

  <p class="MsoNormal">
   <font size="3">
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
     在当时的历史条件以及阶级斗争理论的指导下，由于资本家阶级已经被列为社会主义革命要消灭的对象，因此无论他们对于合营是持什么样的态度，其根本的动机毫无例外的都被认为是出于私心的。如果积极拥护合营，有可能被认为是以积极换取政治资本；如果消极对待，就很容易被认为是唯利是图；如果是抵触甚至反对，更会被认定是落后、反动。如某机器厂资方为高级统战人士，对公私合营求之不得。工作组即认为其是“要求合营是表示进步，以进步来取得党与政府信任和巩固其地位。所以想处处走在别人前头。这样，既有地位又克服困难，真是两全其美，名利双收。”
    </span>
    <o:p>
    </o:p>
   </font>
  </p>
  <p class="MsoNormal">
   <o:p>
    <font size="3">
    </font>

  <p class="MsoNormal">

    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
     扩展公私合营中企业选择的主导权始终掌握在党和政府的有关部门手中。只要企业被有关部门选中，资方以及私营企业最后终究会“自愿”走上合营之路。
    </span>

    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

     上海永安百货公司是
    </span>
    1955
    <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:

     年自愿要求公私合营的，他们的“自愿”有着非常实际的动机。
    </span>
    1953
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
     年起，政府对关系到国计民生的重要商品，如粮食、食油、棉布等都进行了计划收购和计划供应，在批发贸易中，国营商业已经基本上代替了私营批发商，国外的东西进不来，国营批发部又限制数量，永安公司的货架上很快就空空荡荡，无货可卖。
    </span>
    1954
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
     年相比上年，永安的营业额降了一多半，到
    </span>
    1955
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
     年
    </span>

    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
     月，他们连工资都发不出来了。
    </span>
    <o:p>
    </o:p>

  </p>
  <p class="MsoNormal">
   <o:p>

    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
     在这种情况下，国家对永安采取了“安排与改造相结合”的办法，扶持永安公司，永安的批购比例达到了

    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

     ，基本上纳入了国家轨道。当年秋天，永安的老板郭琳爽提出了公私合营的请求，
    </span>
    11
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
     月底获批。经过
    </span>
    50
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
     多天的准备，
    </span>
    1956
    <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
     年
    </span>
