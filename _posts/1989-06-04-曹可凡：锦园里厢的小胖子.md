mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   弄的锦园里。不仅周边的摊贩熟识其父母，整个弄堂里的人至今也津津乐道－－
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   那个皮大王小胖子曹可凡，后来成了大家在电视上见到的著名主持人。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <b>
    <font size="4">
     锦园深处，无锡人家
    </font>
   </b>
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   锦园，得名于荣毅仁伯父、著名荣氏家族的缔造者荣宗敬的名字。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1933
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，中国近代著名民族资本家荣宗敬在上海的生意正处于巅峰时期。为了解决其在上海办面粉厂、棉纺厂时公司高级职员的住宿问题，荣家出资，在公司附近原荣家网球场的基础上，建造了
  </span>
  32
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   栋三层楼房。因为荣宗敬原名荣宗锦，所以就把这个员工宿舍区命名为“锦园”。这一名字，也与荣家
  </span>
  1929
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年在原籍无锡小箕山购地建造的私家园林“锦园”遥相呼应。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   这
  </span>
  32
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   幢楼房，全部为砖木混合结构三层楼房，分布于东、南、西三方，弄口朝北。每幢建筑面积
  </span>
  103.8
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   平方米，总计建筑面积
  </span>
  3322
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   平方米，总体成“
  </span>
  TT
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ”字形。锦园内，不仅较一般里弄宽敞，且遍植绿树，昔日在园中央还有一个喷水池，尽显荣家气派。入住的职员家庭，一户人家单独居住一幢房屋，内里钢窗蜡地、煤卫俱全，生活条件较同时代的居民优越不少。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   而有趣的是，因为入住的高级职员家庭几乎清一色全是无锡背景，因此整条弄堂里，无锡话成为通行语言，无锡的风俗礼仪畅行无阻，居住其中，宛如住在一个小小的无锡村内。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <b>
    <font size="4">
     见证民族企业的辉煌
    </font>
   </b>
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   曹可凡曾外祖父王尧臣和胞弟王禹卿同样也是来自无锡的商人。两兄弟曾在荣家的茂新面粉厂工作，并很快成为荣家的左膀右臂。成熟起来的王家兄弟，试图与同为荣家得力干将的浦家兄弟一起另立门户、自主创业。荣家闻讯后，主动出资入股，最终，荣、王、浦家的“三姓六兄弟”合作创办了福新面粉厂。因为经营得法，很快从一个厂扩展到八个厂，王家兄弟还协助荣家在沪建立了面粉交易所和纱布交易所，在上海商界创下一番天地。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   曹可凡的祖父曹启东，十几岁时来沪工作。正是在福新面粉厂里，这个家境寒微的青年从最底层做起，逐渐成为公司高层。最终，王尧臣把自己的大女儿许配给了他，还让曹启东成为福新的“财务大总管”。虽是入赘，但王尧臣十分开明，结婚以后，曹启东夫妇并不随王家同住，而是一直生活在锦园里。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   曹启东夫妇先后育有四子二女，其中，曹可凡的父亲是长子。战局动荡的年代里，颇有远见的曹启东将两个儿子都送去海外读书，但把长子和幼子留在身边。低调寡言的曹启东，先后经历上海被轰炸的惨状，经历抗战和内战的混乱，处理过复杂的各方势力要挟，又最终迎来上海解放。锦园深处，他和他代表的一群无锡籍的商人，参与并见证了那个时代民族资本家的兴衰往事。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <b>
    <font size="4">
     三岁挂牌去游街
    </font>
   </b>
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1963
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，曹可凡出生时，全家还在锦园深处，过着无忧无虑的生活。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   曹可凡的父亲当时从圣约翰大学毕业，在中国纺织机械厂担任工程师，每月工资
  </span>
  109
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   元，在当时实属高薪。祖父手里还有面粉公司、棉纺公司的股份，每年都能拿到分红。但随着文革爆发，定息成为一纸废文、父亲被迫下放劳动，工资停发，小康的生活顿时坠入困顿。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   红卫兵还曾冲入家里，将曹可凡一家三口全部抓到外面去“游斗”。一起挨批挨斗的大人，每人脖子上都挂着一块牌子，上面写着他们的“罪行”。曹可凡当时还小，只有三四岁，但即便如此，红卫兵还是替他准备了一块小一号的牌子，同样挂在脖子上，被压出去和大人们一起游街示众。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   锦园的房屋也被“就地压缩”占去了大半，全家人只能被赶到阁楼居住。然而曹可凡的父亲却依然坚持着往昔生活习惯。他每天早上四点起床，先在弄堂里打一套杨氏太极，坐
  </span>
  20
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   路公交车从愚园路赶到外滩，去中央商场里喝一杯咖啡，再看上一个钟头的书报杂志－－那时在中央商场，还能看到日文版的《人民中国》，这也是仅有的阅读外国文献资料的途径－－然后再坐公交车去杨浦区军工路上班。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   但家里很快入不敷出，直至典当度日。喜欢理工科的父亲过去在家收藏了许多日、德、英、俄文版的专业书籍，在全家经济最困难的时期，父亲变卖了三分之二的外文书籍来维持家中开销。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1972
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，美国总统尼克松访华，随着两国关系走向正常化，远在海外的亲戚们渐渐与父亲有了联系，美国的叔叔给曹家汇来美金，香港的亲戚也会时不时给上海寄些罐头火腿等。随着“文革”结束，曹可凡父亲的工资又重新调回原价。但令这个历尽劫波的家族在意的不是财富本身，而是参与建设国家的理想，并不曾因个人得失而中断。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1981
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，在落实政策后，曹可凡的祖父捐资
  </span>
  50
  <span lang="ZH-CN" style="font-family:
宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   万元，政府又拨款
  </span>
  50
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   万元，建造了现在的普陀少儿图书馆。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <b>
    <font size="4">
     举“弄”闻名的小胖子
    </font>
   </b>
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   尽管童年也曾挨批受苦，但天生的豁达乐观，让曹可凡成为弄堂里人人皆知的“皮大王”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   因为同乡和同事的双重身份，锦园里的邻里彼此十分熟悉。曹可凡从小就知道谁家的糖罐子藏在何处，谁家的茶几上放着几块巧克力，他都能协同小伙伴一起去弄到手。有一阵子，弄堂里有人家为装修在路面上堆放了黄沙，曹可凡就和小伙伴在黄沙里挖坑，上面覆盖了蛇皮袋，又撒上薄薄黄沙，做成了陷阱。等到经过的倒霉路人一脚踏空跌落，就和小伙伴们欢呼离去。受伤的人来追究时，曹可凡又溜得最快。事后他带着糖果来“安抚”挨了骂的小伙伴，可伙伴们一尝糖果又马上吐了出来。原来是曹可凡把肥皂切成小块，包上糖衣给伙伴们吃，就为了看看大家的窘样。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <b>
    <font size="4">
     喜欢吃水果的他，也曾因为水果受伤。
    </font>
   </b>
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   一次放学后，曹可凡一边吃梨一边走路，不料被身后的三轮机动车撞到，还被不知情的司机挂在车上带了一段路。直到路人惊呼，才被救下。事后曹可凡被诊断为左脚胫腓骨骨折，因着这次受伤，家里拼命给他补充营养，等到腿好了，小胖子的体型就定了调，从此成为他的一张标志名片，陪伴着他出现在荧屏上。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   一件有意思的事是，因为锦园所在的愚园路是西区高档住宅密集的地方，因此不少名人大家都曾在此居住。等到曹可凡长大开始做访谈类节目，好几次在与嘉宾的聊天和采访中，都不期而遇愚园路的“邻居”。每当这时，他总是又自豪又亲切地说“我知道这里，因为我也住在愚园路上，我的家在锦园。”
  </span>
  <span lang="ZH-CN">
  </span>
  <o:p>
  </o:p>
 </p>
