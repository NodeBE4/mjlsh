    由于贡献突出，
   </span>
</font>
<span lang="ZH-CN" style="font-size: medium; font-family: 宋体;">

  </span>
<span style="font-size: medium;">
   1959
  </span>
<span lang="ZH-CN" style="font-size: medium; font-family: 宋体;">
   年、
  </span>
<span style="font-size: medium;">

<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
    文革开始后，
   </span>
</font>
<span style="font-family: 宋体; font-size: medium;">
   容国团被指为贺龙的人，
  </span>
<span style="font-family: 宋体; font-size: medium;">
   遭到“体委红旗”等造反派的揪斗、侮辱和毒打。
  </span>
<span style="font-family: 宋体; font-size: medium;">
   因为以前经常和姜永宁等香港老乡一起吃饭，
  </span>
<span style="font-family: 宋体; font-size: medium;">
   造反派就说他们是“反革命特务小集团”。
  </span>
<span style="font-family: 宋体; font-size: medium;">
   他为了证明自己是热爱祖国的，
  </span>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

    写请战书要求参加第
   </span>
   30
   <span lang="ZH-CN" style="font-family:宋体;

    届世乒赛，
   </span>
</font>
<span style="font-family: 宋体; font-size: medium;">
   结果被批为“修正主义”。
  </span>
<span style="font-family: 宋体; font-size: medium;">

  </span>
<span style="font-family: 宋体; font-size: medium;">
   也被贴上“迷恋资产阶级腐朽生活方式”的标签。
  </span>
<span style="font-family: 宋体; font-size: medium;">
   妻子黄秀珍后来回忆说，
  </span>
<span style="font-family: 宋体; font-size: medium;">
   容国团在香港生活了十几年，
  </span>
<span style="font-family: 宋体; font-size: medium;">
   从来没有经历过政治斗争，
  </span>
<span style="font-family: 宋体; font-size: medium;">
   当运动来临时，
  </span>
<span style="font-family: 宋体; font-size: medium;">
   他根本不知道该如何应对，
  </span>
<span style="font-family: 宋体; font-size: medium;">
   有一种大祸临头的感觉。
  </span>

<p class="MsoNormal">

    在被隔离审查期间，
   </span>
</font>
<span style="font-family: 宋体; font-size: medium;">

  </span>
<span style="font-family: 宋体; font-size: medium;">

  </span>
<span style="font-family: 宋体; font-size: medium;">

  </span>

<span style="font-family: 宋体; font-size: medium;">

<span style="font-family: 宋体; font-size: medium;">

  </span>
