  </span>

minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">

  </span>
  <o:p>
  </o:p>
 </p>

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   天山芙蓉
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在伊犁的湘女每次想念家乡时，总会唱起当地的一首民谣－－“乌孙山啊，金色的摇床，英雄喜爱自己生长的地方，假如叫我在异乡做一个国王，我情愿在故乡当一名靴匠……”直到唱得泪流满面。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   如今，新疆的面貌远非昔日能比，其间又融进多少援疆儿女的血泪与汗水。在时代的大潮中，她们的命运早已与共和国的命运紧紧连在一起。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">

 <p class="MsoNormal">

minor-fareast">
   石河子市的南山陵园安眠着很多军垦战士，亦有不少湘女，陵园中有两座湘女纪念碑。想起逝去的姐妹们，湘女谢荃辉与华淑媛在纪念碑前泣不成声。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   湘女才情惠天山
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   著名作家卢一萍曾在新疆军区工作多年。从
  </span>
  2000
  <span lang="ZH-CN" style="font-family:
宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   年起，卢一萍开始关注湘女群体，他历时五年，辗转近两万公里，创作了《八千湘女上天山》一书。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “湖南女兵是新疆屯垦戍边大军中特殊的一部分。她们在苍茫的大荒原上演绎了一个又一个悲欢离合的故事，也演绎着她们的光荣与梦想。有许多人牺牲了青春和爱情，但她们以妻子和母亲的坚韧战胜了其中的艰难和困惑。”卢一萍说。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在新疆维吾尔自治区党委党史研究室编审、党史专家刘向晖看来，八千湖南女兵进疆，这个数字在人口庞大的中国似乎微不足道，但她们对新疆的影响却是巨大而持久的。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “女兵的到来，逐步解决了当地军人的婚姻问题。”刘向晖说，尽管在婚配过程中难免带有政治分配色彩，“可就是这些婚姻，让广大官兵心甘情愿扎根新疆，为新疆稳边固边打下了坚实基础。这不能不归功于当年那些把青春献给屯垦事业的湘女们。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   令人意想不到的是，这种不被后人看好的婚姻形式，不仅大大稳定了军心，还在几十年后的调查中出现了“几乎没有离异，都是原配相濡以沫走到了今天”的圆满结局。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “湘女不仅是结婚对象，她们自身也是建设者。”刘向晖说，湘女见识较广，接受力强，有一部分还是大学学历。“她们的到来，给新疆增加了一批高素质的建设力量。后来，她们中的不少人成了各行各业的佼佼者或领军人物，为新疆的发展与进步作出了重要贡献。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

  </span>
  /
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   凤网记者，当时的驻疆官兵大多来自农村，文化程度普遍很低，“有知识的湘女，无疑对部队正在进行的扫盲和文化科学水平提高工作起到了雪中送炭的作用。同时，女兵们还把湖湘大地先进的农业生产技术及其他生产技术带进新疆，促进了当地经济的进步和发展”。
  </span>

 <p class="MsoNormal">

minor-fareast">

  </span>

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   齐鲁女兵进疆，西北陕甘地区和上海一批女性青年进疆等。

 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

 <p class="MsoNormal">

Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   气门弹簧这种零件只有总库有，等战友去总库取来零件华淑媛再将机器修好时，太阳已经西下。华淑媛身上的衣服已被奶水浸透都结成硬壳，“我这才想起来，儿子已经有八九个小时没吃奶了”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   华淑媛马上从地上抓起一把土擦擦手急忙往家赶。从田间回家的路上要翻越两个沙包，当她翻过第一个沙包时，突然一个动物从她身边快速飞过。华淑媛马上意识到是一只狼，“我当年最怕的就是荒原上的狼。但当时也顾不得害怕，一心想着赶回家给孩子喂奶”。华淑媛回到家时，儿子满脸都是泪痕，连哭的力气都没有了。“等我给儿子换洗完喂奶，静下来时，才感觉到后怕，幸亏当时急着给儿子喂奶跑得快，狼也跑得飞快，那一瞬间让我逃过了生死关。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   华淑媛说，她开拖拉机经历过“三难”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1954
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年夏天，垦区第一次使用康拜因联合收割机，因为要跟雨水抢时间，争取颗粒归仓，作为班长的华淑媛每天最多只睡三四个小时，有时甚至昼夜连续工作。“有一次在随车检查质量时，我忽然感到一阵晕眩，从康拜因的扶手上直接滚下来，掉在土坑边的低凹处，这时一辆巨大的草车轮从我身边擦身而过，我险些被草车碾着。”华淑媛回忆，她当时吓出了一身冷汗。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   而另一次华淑媛就没有这么幸运了。“有一天正在抢收麦子，康拜因的链条没油了。为防止链条磨损，我叫康拜因驾驶员停下车，抓紧时间注油。”没想到驾驶员因疲劳过度睡着了，连离合器松开了也不知道，康拜因向前开了十几米，驾驶员才醒过来。一停车，发现华淑媛的右手上全是鲜血，“我仔细一看，右手拇指已被链条打掉了”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   最后一难发生在一次夏收结束后，华淑媛到七十五团机耕队去检修机器。战友们刚刚把牵引架支好，她一踏上去牵引架就打了过来。“当时把我打晕在地，头部砸了好长一道口子。”华淑媛说，至今她身上留下多处伤疤。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “好在通过几代军垦战士的奋斗，这块戈壁荒原建成了绿洲，成了我们美丽的家园。”华淑媛说，这是她最为欣慰的事。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  3
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   陶先运：那时没人知道我是陶峙岳司令的侄孙女
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="150.jpg" border="0" height="221" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3663/150.jpg" width="120"/>
 </p>
 <p class="MsoNormal">
  <br/>
 </p>
 <p class="MsoNormal">
  <img alt="151.jpg" border="0" height="369" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3663/151.jpg" width="320"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1951
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年
  </span>
  3
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   月，刚刚
  </span>
  15
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   岁的茶厂女工陶先运辞掉工作，准备参军前往新疆。“我将参军的事给父亲讲了，父亲一惊，随即点燃手中的烟锅狠狠抽了一口，缓缓地说：‘去新疆呀！’”陶先运回忆，她看见父亲叹了声长气，便斜斜地躺在床上，一言不发。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   女儿心意已决，父亲只能同意。“从长沙送我走的时候，父亲跟我说：运满
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   陶先运为家中小女儿，‘满’为最小之意
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，你明六叔公在边疆当总帅，你去了那里，可以找到他，他会关照你的。”陶先运告诉今日女报
  </span>
  /
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   凤网记者，父亲说的“明六叔公”，是时任新疆军区副司令员兼第二十二兵团司令员的陶峙岳。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “刚到新疆的时候，我们要参加集训，我们被安排住在司令部。那时兵团司令部设在景化，一大片低矮平房，四周是围墙，门口有岗哨。进门后，一侧是首长办公室的院子，另一侧是我们学生队住的地方。”陶先运说，有几次她见小汽车开进首长院子里后，有位个子不太高的老者下来，人们称他陶司令员，“我想起了父亲说的明六叔公当边疆总帅的事，但没有去找他。那时的思想十分纯洁，完全没有想过要找叔公照顾”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   集训之后，陶先运被分配到二十五师当护士。有一天，医院的协理员忽然通知陶先运，“说师部的史政委找我，要我下班后到他办公室去一趟”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   陶先运到了史政委那里后，发现史政委不在，“房子里面只有一位老人手持火钩，弯腰在铁皮火炉边生火”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   老人转脸看了陶先运一下说：“史政委不在，是我找你，你进来吧。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   坐下之后，老人一一问了陶先运的父亲、伯父等亲戚名字。“我刚说完，他就一把拉住我的胳膊说：‘我的满孙女呀！我就是你的明六叔公！’”陶先运说，叔公当时摸了摸她穿的不合身的棉袄，又指了指她脚穿的苏联大皮靴，说“靴子不合脚，磨出了两个洞”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “叔公还问了家里的亲人，又仔细问了我在新疆的生活工作。”陶先运说，叔公让她安心呆在新疆，“叔公还问我有什么困难没有时，我连忙说‘没有没有’。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “第二天早晨，我出房门，正好叔公也在门外，他对我说：‘我吃早饭后要给连队上的干部讲话，然后就回迪化
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   即乌鲁木齐市－－编者注
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   去。’叔公还给了我他家的地址，‘你若有机会去了，到家里找我。’”陶先运说，叔公说完后，马上又叹了口气，“也不容易有机会呀！”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “那时去迪化，一无公路，二无班车，
  </span>
  100
  <span lang="ZH-CN" style="font-family:
宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   多公里的苇湖荒滩中只有条便道，野狼出没。”陶先运说，当时唯一的交通工具是骑马，还有汽车团拉运农用物资的汽车。“战士要外出，只有搭拉货的便车。很难有机会出去。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   陶峙岳唯一为陶先运留下的礼物，就是一支“博士”牌钢笔。“二十多年我从来没有找过他。”陶先运说，她再次见到叔公，已经是
  </span>
  1979
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年，“叔公已离休回家乡定居”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
