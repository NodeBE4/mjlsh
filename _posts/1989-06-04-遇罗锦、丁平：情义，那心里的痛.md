  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “佛佛，
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   你这人，可恨，可恨，可恨！就因你说过，你从来不留信，看后全部立即销掉。
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   假如你留着，就去看看我刚认识你给你写信时，就给你写过：‘南京有我父母的老友丁大爷，他有个儿子丁灏大哥，文革前还来过我家，他尤其和哥哥特别谈得来。如果我回国看看一定去南京，很想去找他们，也不知道丁大爷和伯母是否还健在。’
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   当时我写了这些话，你是只字不提，看来，你早就忘了我写过什么了。
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   我都不知道丁灏还有个弟弟叫丁平，当然想认识他呀！这还用说？麻烦你告诉他我这信箱，让他写信吧。丁灏大哥早就退休了吧，他一家人都好吗？”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   回忆起那唯一一次见丁灏大哥来我家，是在
  </span>
  1966
  <span lang="ZH-CN" style="font-family:
宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   年“破四旧”之前。我从“北京工艺美术学校”毕业还在实习阶段，那是炎热的夏天，当时丁灏大哥是去内蒙出差
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   他在南京大学
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   考察地质回来，路过北京特意来看看我们。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   父母
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   尤其是母亲
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   一见他就喜欢如故，一转脸就对我们赞不绝口：“看你丁灏大哥，温文尔雅，极有才华，又有事业心，对老人多有礼貌，句句话都中听。”
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   她满脸都写着：“罗锦，我希望你和他搞对象，再没有比他更合适的人了。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   我只知父母和他父母的友谊颇深，但为何那么深，以及丁家的历史，还真不清楚，因为父母很少给我们讲述过去的事情，我们也习惯性地不问。虽然我对丁灏大哥的印象极佳，可我当时似乎对谁都没心思去搞对象；就连我爱了四年的同班同学国栋，也是空中楼阁式的幻想，尤其是父母离婚后给我的反面影响，自己总觉得这辈子都不会有什么意中人，因此对母亲的话我就装听不见。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   当时，我家的住宿条件已很拥挤：哥哥白天劳累地在“北京人民机器厂”上班，住在他那昏暗狭窄的小煤屋里，一个不宽又发潮的单人木板床。里屋，我和姥姥也不宽的双人床就让给了丁灏大哥。外屋，除了父母的双人床外，姥姥与罗文罗勉挤睡一夜，我就睡在了放衣物的大躺箱上。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   丁灏大哥说在我家只住一夜，却和哥哥坐在屋前的海棠树下，聊到大半夜，十分投机。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   天热，宽大的老旧竹帘挂着，双扇屋门开着，屋里早就关了灯。而他俩，坐在海棠树下的小板凳上，沉浸在夜色的昏暗中，面对面地低声聊着天，不时地发出笑声，毫无睡意。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   我躺在大躺箱上，迷迷糊糊地听着和想着：“他，是哥哥理想中的‘我’呀，哥哥把他看成是自己的活化身哪：他比哥哥大一岁，在南京，他俩从呀呀学语起就熟悉。哥哥所有的幻想和愿望：学习与人品的优秀、好读书、有才华、入团、南京大学、研究生、全国考察矿藏与岩石并游览大好河山、还可尽情地写作，这一切哥哥的追求与梦想，丁灏大哥全部实现和做到了，且优秀地继续在做着，并深刻细腻地体会着。哥哥的理想和梦想，丁灏大哥全都有了，他正是哥哥希望达到的活生生的样板。而恶运却都集中在哥哥身上了：学习与人品虽然优秀杰出，却三次报考大学都因出身问题而不被录取，他现在仅仅是在工厂才挣
  </span>
  16
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   元工资的体力劳动学徒工，明天，他还要劳累地去上班。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   然而，从他俩那开心的压低的笑声里，从那些我听不太清楚的谈话中，我感觉不出哥哥的任何酸楚和委屈，也感受不到丁灏大哥的任何歧视与傲慢。似乎他俩早就融为一体了：是他们谈论着社会的现象与政治？还是社会的风气？或是远方所见所感的风土人情？就算给他们十天半月，他们也聊不完呵。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   无论他俩在聊什么，他们视对方仍是彼此六岁时的纯真无暇。一个是在初中的“入学登记表”上没写“资本家和右派出身”的成功的他，一个是因写了“资本家和右派出身”的永不能成功的他。其实，他俩完全是一个人，完全是被人为的政治，活生生地拆开的两个不同的样板。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   而让我感到欣慰的，是他俩谈话时的心情：自由自在地直抒胸臆﹑互相的理解与提升，全不受社会的政治、舆论、歧视与等级的影响，这，需要有多高的修养和心胸啊！需要有多么丰富的知识啊！他们只被对方的思想与人品所吸引，忘记了疲乏，忘记了应该去睡觉；他俩知道这一夜过去，将不知何时才会再相见，他们宁可一夜不睡；这，又需要多深的理解和爱！
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   遗憾的是：过去的几十年来，中国都没有什么节假日。谁出差路过哪个城市去看望亲友，顶多半天一天就得离开，不仅不可能为这向单位请假，更不可能为这失去不易得来的工作。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   丁灏大哥走后，便与哥哥经常通信，好像那一夜的聊天还远未聊够。每次的来信，哥哥都爱不释手，赞他有文才有思想，且一手潇洒漂亮的钢笔字，哥哥极为珍爱地保存着他的每一封信。往往，
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   他会情不自禁地夸赞：“看这首诗，他写得多好！”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   回信时，除了叙述生活里的事，他便也以诗回赠。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   不久，文革“破四旧”来临，
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   有一天，
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   哥哥把这事告诉了我，并看他亲手烧了信。当时我还不以为然，心想：“真有那么严重吗
  </span>
  ?
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

   那些都是心灵的真迹啊，烧了多可惜！”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   往后，就是我因日记被劳教，哥哥进了监狱被判死刑，俩弟弟去陕西插队……我们两家人，彼此无音信直到今天。一晃，就是半个世纪！
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

  <span lang="ZH-CN">
  </span>

 <p class="MsoNormal">

  .

 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

 <p class="MsoNormal">

  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   “文革期间，丁灏因受罗克兄的《出身论》的牵连，被关押六年，受尽摧残迫害，头部被打伤，留下了严重的后遗症－－癫痫……”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

   受《出身论》的牵连，这可能吗？
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   文革“破四旧”之前，他俩就把彼此的信全部烧掉了，两个极为理智的人，以后在那无法形容的骚乱里和惨无人道的打砸抢中，自然不会有任何联系。接着，在广州和北京就是油印的《出身论》，在北京是铅印的《出身论》和一期期的报纸，六期《中学文革报》的头版头条文章，以及其他的化名文章，都是哥哥白天劳累地上班，夜晚及周日为报纸赶写的，他怎么有空，怎么有可能去与丁灏大哥联系？！他怎能不深知避免牵连任何人，何况是自己的挚友？！
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   除非只有一个可能：《中学文革报》的读者来信需用麻袋装，罗文罗勉他们是用平板车，从邮局一车车地拉走。或许南京也有此报纸的流传与复印，丁灏大哥看了以后，赞不绝口，他并不知道“家庭出身问题研究小组”的笔名就是遇罗克，一定会按捺不住激动的心情，给报社写过信。哪怕他用了假名，但公安还是会侦破的。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   尤其是：哥哥把很有水平又深为感动他的来信地址及人名，抄在了一个本子上，共有三千多人。哥哥在工厂被捕时，公安人员来家搜查，也把它搜走了。哥哥的审讯材料为何有满满的八麻袋之多？因公安局认为他要搞党派，但哥哥是拒不承认的。审问他：“为何你把人名地址记在这个本子上？”哥哥甚至回答：“我也不知为什么。”为此他受了大罪，背铐整整半年；是否还有其他的人身惩罚，不得知。
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   既然他不想交代任何人，又不能胡编滥造，他也只能如此回答了。故在他的死刑判决书里有一句：“在押期间气焰嚣张。”
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

  </o:p>

 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">

  </o:p>

 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">

  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

  <span lang="ZH-CN" style="font-family:

  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">

  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   过后，有朋友对父母说：“您不提那个要求，照样也会给他们平反的，受害人自己都会提出的。您这么一来，就失去了一个条件。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   可以设想
  </span>
  :
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   文革时，即使丁灏大哥根本不知道“家庭出身问题研究小组”的笔名是遇罗克，即使他给报社写信时不写真名，但由于信封上的邮戳是南京来的，哥哥认识他的字迹，更由于信里内容的高水平，哥哥会立即知道是他写的。既然连哥哥都能很快知道，公安局那么多有水平的侦探，也绝对不是傻子。

 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   但可笑的是：
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   我家四个孩子竟然是两个出身；我和哥哥奉母之命填写的是出身“资本家”，因在“反右”前，母亲正在得意之时：她是北京市代表，全国妇女代表，北京市工商联副主任，副厂长，还在中南海与毛主席合影，还曾经想过入党，她不仅不为“资本家”这身份担忧，相反，她还很自豪得意呢。但到了俩弟弟上初中时，说什么也要填父亲是工程师的“职员”出身了。他俩说：“我们又不想入团，怎么会调查我们？同学中
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   隐瞒出身和政治情况的，绝对不会只有我们两个。”所以他俩从未受过班主任老师的刁难和歧视。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   而哥哥一定要入团，他就必须把家庭情况交代清楚。最上进和诚实的人，也就最倒霉，他的初中和高中都深受政治歧视。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   历次的政治运动，摧毁的是人与人之间的关系。过了半个世纪，当遇家与丁家的人再次相逢时，唯有许多不为人知的情义，令人痛彻心骨……
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   那些电影般的画面是永生难忘的：
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   每当丁大爷从南京来信，母亲就高兴地对姥姥说：“浦哉来信了。”也对我们孩子们说：“你丁大爷来信了。”她就高高兴兴地看信，简略地叙述着南京那家人的生活近况，总之，一切都挺好。父亲有时来看望我们，母亲回回不忘把丁大爷的信给他看。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   当时我在上二年级，瞥见到那几页信纸上的龙飞凤舞又自成一格的浪漫字迹，正如丁大爷叙述生活情景时的豪爽与开朗。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   不久，丁大爷来到我家看望我们。彼时父母尚未复婚。
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   家里住处宽敞，客人独居一间单独进出的屋子。高大魁梧又亲切的丁大爷，言谈话语，举手投足，活像母亲的亲哥哥。在客厅里，他俩隔着茶几，对面坐在小沙发里，与母亲聊得十分融洽投机。母亲两手飞快地织着我们的毛衣或毛裤
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   那是她每年乐此不疲的家务和“任务”
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，喝着茶，一面和丁大爷聊着与感叹着。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   初夏，无风的蔚蓝晴天，客厅的双扇木门大敞着。
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   院里四角花坛的花，正在盛开。我在院子里独自玩耍，闻闻这朵花，闻闻那朵花。
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   姥姥和保姆吕姨都在各自的屋里休息，哥哥和俩弟弟也不在眼前；离了婚的父亲自然在他那小家里正享受着甜蜜与幸福。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   只听丁大爷说：“秋琳，别织了，咱俩跳跳舞吧，我好久没跳了。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   母亲有点不自然：“跳舞？也没音乐呀。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “来，
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   我自己哼音乐。”丁大爷把母亲拉起来，亲切地把她揽在怀里，他比她高出一大块，显得那么魁梧，大度和体贴。就在那间并不很大的客厅空地上，两人慢慢地悠闲地起舞。丁大爷没有哼音乐，似乎，他只想用这种方式，给予母亲兄长般的爱；也只想用这种方式，让一直充当父亲的母亲体会到还有十分爱她的人。母亲变得少有的温顺和舒心，那舒心里含着感动，
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   她那表情是我一生中从未见过的；一切，只沉浸在友谊的爱与理解中……
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   六
  </span>
  .
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   丁平的信：
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   ……
  </span>
  1972
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年灏兄落实政策后，工资恢复并补发了隔离审查期间扣发的部分；由于他在隔离审查期间头部被打伤，留下了严重的后遗症－－癫痫，难以继续从事野外水文地质考察工作，调入南京大学图书馆工作。他不甘平凡，以其聪明才智和非凡的毅力，克服身体上的疾病，在地质、考古、书法、文博以及西方现代艺术等领域都有所建树。用“郁郁而终”形容他不符合事实。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   当我找出灏兄悼念罗克兄的诗稿时，虽然以前也读过这四首诗，但是睹物思人，依然被他们之间兄弟般的深情厚谊所震撼！我的泪水忍不住夺眶而出，心情久久不能平静……
  </span>
  <o:p>
  </o:p>
 </p>

  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   悼遇罗克
  </span>
  )
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   一
  </span>
  )
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   血荐乾坤头不轻，拼将玉碎默雷鸣。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   健儿含笑作豪鬼，千载龙泉夜有声。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  (1)
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   《出身论》正式出版后，罗克有诗赠我称“十年读典此剑成”又言献身之志曰：“鬼豪回首堪笑慰”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   二
  </span>
  )
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   相期风雨未绸缪，谁信斯人真断头？
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   讯报春回寒欲尽，情余梦灭泪不流。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   怜才笛咽古今赋，伤乱黍离班马忧。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   细看长征修远路，前驱血印在神州。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  (2)
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   罗克于六七年十月与我诀别，余南归，得其赠诗曰：“未有余情贻少年”，“羌笛一梦两地声”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   三
  </span>
  )
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   遇生赴国忧，慷慨一奇男。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   逆风奔骐骥，偷火沥心肝。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   强项立天地，横眉斥权奸。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   烈士贱头颅，岂薄细碎冤？
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

   人生本易老，急书胆剑篇。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

 </p>
 <p class="MsoNormal">

   四
  </span>
  )
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   遗业艰难赖众英，网罗冲决贵先行。
  </span>
  <o:p>
  </o:p>
 </p>
