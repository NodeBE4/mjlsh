   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    多名南方来的“孤儿”。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    不论在当时，还是之后的半个多世纪里，人们都不愿意用“弃儿”来称呼这些来自江南的孩子。他们曲折迂回地将孩子们统一称为“江南孤儿”或“国家的孩子”，似乎只有这样才能让各方都获得暂时的宁静。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family: 宋体;">
   <b style="">
    <font size="4">
     二
    </font>
   </b>
  </span>
  <font size="3">
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family: 宋体;">
   <b style="">
    <font size="4">
     <br/>
    </font>
   </b>
  </span>
 </p>
 <p class="MsoNormal">
  <font size="3">
   1960
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年春天起，从上海开出的列车，行驶在京沪和京包线上。孩子们的各色小衣裤和白色尿布，挂满每一扇打开的列车车窗。装着小乘客的车厢，穿过江南水乡，一路掠过平原、山川，经过长江、黄河，最终驶入一望无际的草原。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    当时尚无名字的宝宝和额尔登，是这特殊车厢里众多婴儿中的两个。不满周岁的他们，分别在
   </span>
   1960
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年和
   </span>
   1963
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年加入这次迁徙。没有人知道人生中的第一趟火车之旅，将把他们各自的命运带向何处。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    列车的起点是温暖、湿润、葳蕤的南方，而终点，是干燥、寒冷、人烟稀少的牧区和草原。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    面积
   </span>
   110
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    多万平方公里的内蒙古草原，直到上世纪
   </span>
   60
   <span lang="ZH-CN" style="font-family:
宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    年代，人口不过
   </span>
   1000
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    万，每平方公里不足
   </span>
   10
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    人。荒寒的气候，繁重的劳作，低下的医疗水平，逐草而居的艰难生活，让牧区妇女的生育率极低，不少人患有不孕症。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    而绵长的边境线和日益紧张的“中苏“关系，让此地愈来愈成为国防和充实人口的重点区域。南方来的饥饿的孩子们，就这样承担起双重的功用：既成为缺少子嗣的“牧民的孩子”，又成了能够充实边疆的“国家的孩子”。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    其实早在
   </span>
   1958
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    年
   </span>
   9
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    月，内蒙已经从安徽接运
   </span>
   500
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    多名孤儿安置在锡盟。那时安徽还是夏末的气候，孩子们刚到内蒙就赶上寒流。由于体质弱，大量孩子生病死亡，仅有
   </span>
   200
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    多名幸存下来。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    这次事故后，内蒙古要求各地接引孤儿前必须先设立保育院。孩子们到达后，先收入保育院抚养，待状况稳定后，才让牧民领养。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   1961
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年，在保育院生活了一年之后，包括宝宝在内的七八个孤儿，被分到达茂旗草原。宝宝是其中两个男孩之一。因为长期营养不良，当时已经两岁多的他不会坐也不会爬，脖子细得甚至不能撑起大脑袋。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    一到达茂旗，宝宝就被镇上粮食局的一个副局长领回了家，但不到
   </span>
   3
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    天就被“退回”了民政局，理由是“这孩子肯定养不活”。
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    粮食局一个没有孩子的普通工人听说了，赶紧跑去民政局，抱走了这个可怜的男孩。
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    这就是宝宝的养父。此前，这对工人夫妇曾育有一子，但因保姆照顾不慎而夭折，夫妇俩再也没能生育。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    为了照顾他，养母专门从纺织厂辞了职。他们疼爱这个几乎可能夭折的孩子，怀着无限的暖意给他取了一个小名：宝宝。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

    </b>
   </span>

 </p>
 <p class="MsoNormal">

   6
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    岁左右的童格勒嘎。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    他们每一个人都没有准确的生日，出生年份也只能模糊地界定在两个年份之间。就连究竟是在
   </span>
   1963
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年的什么季节来到达茂旗草原，也不大有人记得了。只有来时年纪最大的童格勒嘎，还依稀记得来时汽车窗外绿油油的草原。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <img alt="46.jpg" border="0" height="442" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3709/46.jpg" width="590"/>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <br/>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    萨仁琪琪格说，这张照片是
   </span>
   1960
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    年到
   </span>
   1964
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年间，某一年分到乌兰察布育婴院的“上海孤儿”。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    牧民们一听又有孩子来了。心急的，骑上马就赶来了。心细的，拿上牛奶和衣服，套上勒勒车，拼命往镇上赶。这一次，所有孩子都被领回了家。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    三
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    童格勒嘎如今依旧住在草原上。只是她的罗圈腿越来越严重了，连走路也一瘸一拐。
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    她已经
   </span>
   60
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    岁了，住在养父母留给她的院子里。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    她还记得，父亲骑马接她回来的那一天，这里还没有房子，只有两只孤零零的蒙古包。
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    草原上的牧民，是真疼这些白白嫩嫩的孩子。
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    苏布达和萨仁琪琪格来时不满周岁，她们是被养父用袍子紧紧裹在胸口，骑马带回蒙古包的。额尔登是坐着勒勒车回家的。来时年纪最大的童格勒嘎，被抱上了养父的枣红骏马，那时的她还是一口江南方言。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    取名，是养父母表达接纳和爱意的开始：“额尔登”在蒙语里就是“宝宝”的意思，“苏布达”是“珍珠”，“萨仁琪琪格”是“月亮花朵”，“童格勒嘎”是“澄澈无云的天空”。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <img alt="47.jpg" border="0" height="482" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3709/47.jpg" width="360"/>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <br/>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    苏布达很幸运，养父母带着不到两岁的她去包头留影，很少有“上海孤儿”留下这么小时候的照片。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    相较于其他人，来时已经五六岁的童格勒嘎，需要重新适应新的家园。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    她记得刚到蒙古包，养父就倒了一碗奶茶给她喝，算是一个小小的欢迎仪式。
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    之后开始学蒙语。“刚开始，学得累了，爸爸就端来一碗炒米，说只要会读一个字，就奖给我一把，有时候还会奖励我干奶酪条。”
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    那个年代，炒米和奶酪都是非常珍贵的食物。
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    除了童格勒嘎之外，其他的孩子都不存在重新适应的问题。等他们开口讲话时，说的就已经是蒙语了。
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    长到七八岁，他们都成为大人的帮手。额尔登记得自己从
   </span>
   5
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    岁起，就开始和姐姐一起放羊。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    再大一些，他可以一个人帮大队放
   </span>
   200
   <span lang="ZH-CN" style="font-family:
宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    只左右的羊群。冬天骑着骆驼，夏天骑着马，每天两次把羊群赶回家喝水。
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    他的记忆力本就很好，再加上与羊群朝夕相处，很快，羊群里每一只羊的样子，他都能辨认得出来。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    几十年后，他的手机里从来不存联系人的姓名，只凭记忆手机号就能知道对方是谁。朋友们惊讶于他的记忆力，知道自己身世的他，总是自嘲“毕竟我是南蛮子嘛，脑袋瓜子精明些”。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

    一个包头来的男子，给达茂旗的每个“上海孤儿”送了这张老照片。据说这是当时孩子们被送到达茂旗民政局时拍下的。苏布达他们很喜欢猜测照片里的孩子到底是他们中的哪一个。

 <p class="MsoNormal">

minor-fareast">
    放羊、骑马、挤奶、剪羊毛、照顾小羊羔，是这些南方来的孩子们从小就学的生存技能。
   </span>
   <span lang="ZH-CN">
   </span>
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    因为少女时期早早学会骑马，导致还未发育完全的骨骼受到影响，这让童格勒嘎和苏布达都有不同程度的罗圈腿。宝宝记得，
   </span>
   20
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    岁接养父的班在粮食供销社上班的时候，就注意到了她们。

   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
