<p class="MsoNormal">
<b>
<span lang="ZH-CN" style="font-family: 宋体;">
<font size="5">
     长沙知青大逃亡
    </font>
</span>
<font size="4">
<o:p>
</o:p>
</font>
</b>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
<b>
<font size="4">
     －－作者：陈秉安
    </font>
</b>
</span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  (
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   一
  </span>
  )
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   引语
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  1964
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年，湖南省组织了数万未被录取的城市学生和闲散青年奔赴人口少、耕地相对宽裕的湘南、湘西农村。仅零陵地区的江永县一个县就下放了
  </span>
  6000
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   多名长沙知青。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   这
  </span>
  6000
  <span lang="ZH-CN" style='font-family:宋体;
mso-ascii-font-family:"Times New Roman"'>
   多人绝大多数都出身于资本家、地主、右派、国民党军官……的家庭，其中颇有一些三湘名人，如两江总督陶澍、军机大臣左宗棠、湖南都督焦达峰、国民党将领廖耀湘……等的后人。有些公社
  </span>
  (
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   比如白水公社
  </span>
  )
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   ，“黑五类”子弟比例高达高达
  </span>
  95%
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   以上。以致于有些知青点要成立一个团支部，会“很困难很困难”。这为后来当地屠杀“地富反坏”风潮时，屠杀到长沙知青留下了恶根。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   但这批知识青年长期受到党的“革命理想”的灌输，老实、听话，一般都乐于“改造”自己，甚至有一部分知青抱有“赎罪”的心理。在最初的两年里，这些年轻人积极劳动，办夜校、组织农民学文化
  </span>
  ;
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   有知青为病重的农民献血，白天在烈日下劳动，晚上，还打着电筒在被窝里面写歌颂党和毛主席的诗歌。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   下面是一首我们将提到的一个人物王伯明的小诗《新农民之歌》：
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   扶着犁，扬起鞭
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

<p class="MsoNormal">

<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

</o:p>

<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

   多个知青要进城去。公社有一部解放牌，正好停在我们场里。我们就跑过去，把车门撬开了，
  </span>
  20
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   几个知青上了车，我点燃了火，开得“呼呼”的，往城里跑。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   大约是
  </span>
  3
  <span lang="ZH-CN" style='font-family:宋体;
mso-ascii-font-family:"Times New Roman"'>
   点多的样子，我们的车到达了县汽车站附近。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   当时的位置我记得很清楚，有一段较陡的下坡路，下坡路的最低处是个大“之”字拐弯。拐弯处的右边是零陵县东方红小学，学校外是一道沿马路而建的围墙。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<o:p>
<img alt="5.png" border="0" height="341" src="https://i.imgur.com/1m5HS9E.png" width="582"/>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   这时候我看到公路中间，怎么横着两株树呢？我还骂了一句：“缺德。”就把车停下，把头伸出去朝车上喊：“下来几个人啥，搬树啥！”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   当时从车上跳下来两个人，一个是我弟弟谢祥怡，另一个是余泽。两个人把树搬开了，正要上车，突然间，马路两边“砰砰砰”枪声大作，从学校的围墙和后面山上射出的子弹，把我的驾驶室玻璃“啪啪”打的粉碎。猛地我感到肩上一热，我马上意识到，是中弹了！
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   我还想坚持开车跑，但手已不听使唤，我又坚持开出了约
  </span>
  20
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   米，再坚持不住了，就猛地一个刹车，停住了车。我从驾驶室中翻身而出，滚下了马路边的一条水沟里躲避。这时又感到屁股上一热，可能是又中了一弹！
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   这时候，我只听得车上一片悲惨的喊叫声：“不要打，不要打－－”“我们是知识青年－－”但是枪声并不肯停止。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   知青姚海不顾一切地脱下身上的白背心，站起来使劲晃功：“莫打哒，我们是知识青年－－”。结果他也被打倒。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   接着唐立人被打得脑浆四溅，杨海洲被打中了，吴德胜也被打中了……
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   猛然间，“轰－－”地一声巨大的爆炸震耳欲聋，原来是迫击炮响了，他们竟然向汽车开炮！我们根本都失去反抗了，还用炮轰，这不是屠杀么？
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   这时，我身上已多处负伤。血流满身，晕过去了。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   下面的情况我是后来听车上的彭稚鳌说的：
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   彭稚鳌写道：“突然，枪声猛地停了一下，他们在喊：‘下车来排队－－’。何小明、段良农等人顺次地下车排队，突然，枪声又响了，他们竟然朝排队的何小明他们开枪！’
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   ‘枪声终于停歇，我跳下了车，就看到段良农右手扶腰，手上流满了血，腰上打了个对穿洞。我又朝左前方地上看去，看横躺在车子左边的何小明：一只黄色军挎包还斜背在他肩上，肚子被打一排弹孔，肠子已经流了出来。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   我伤心地恸哭，用他挎包内的白背心把他流出一半的肠子裹着塞进他肚内……’
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<o:p>
<img alt="6.png" border="0" height="376" src="https://i.imgur.com/e1leNg8.png" width="520"/>
</o:p>
</p>
<p class="MsoNormal">
  13
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   个知青在文化大革命中组成文艺宣传队，满怀热情宣传毛泽东思想，谁知何小明
  </span>
  (
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   中排右一
  </span>
  )
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   ，竟在“
  </span>
  8.18
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   ”惨案中无端被杀。
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   （照片由知青周光武保存）

</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   在这场屠杀中，一共打死了我们
  </span>
  9
  <span lang="ZH-CN" style='font-family:
宋体;mso-ascii-font-family:"Times New Roman"'>
   个知青：何小明、唐立人、杨海洲、吴德胜、陈文华、王成章、廖忠普、余泽、彭柏元。
  </span>

  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>

  </span>
  1
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>

  </span>
  15
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   人。”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   惨案大大震慑了下放到整个零陵地区上万长沙知青。数日内，几乎所有的长沙知青都逃离零陵，寻路返回长沙。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  (
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>

  </span>
  )
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   充满凶险的逃亡路
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   原来，长沙知青还存有最后的一点希望，盼望通过自己扎根农村的优异表现，获得“党和人民”的认可。但是，现实粉碎了他们的幻想。当屠刀就要架到脖子上来时，最后一丝留恋被割断了，只有一条路：逃！
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   但是，即使“逃”也很不容易，一方面农村民兵在各处设有路卡，日夜放哨，抓到逃跑的，视为逃跑的“黑五类”，一律由“贫下中农最高法院”“就地执刑”。另一方面。原来从江永回长沙的公路必经道县，其时道县正在大屠杀，所有的班车都停开。知识青年要回长沙，就必须绕道广西省，再进入广东省，从广东省沿京广线回长。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   但从江永县进入广西，许多公社必须翻越高一、二千米的都庞岭，十分艰难。其时，有些知青已在农村结婚，有的甚至有了孩子。所以，长长的攀越山岭的队伍中，常见父母背着、甚至用箩筐挑着孩子而行。
  </span>
<o:p>
</o:p>
</p>

<p class="MsoNormal">

<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   铜山岭农场知青：路遇
  </span>
  6951
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>

<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   由大逃亡而引出的长沙知青的大游行、大演出、大宣传……波澜壮阔、规模空前，不仅在湖南，而且在全国也产生了很大的影响。
  </span>
  1967
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   年的
  </span>
  10
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   月，由湖南、湖北、广东、四川、河南、重庆等省市自发组织的全国的知青座谈会在长沙召开，这次民间组织的全国性知青活动，标志着当时由长沙的知青主导的知青运动，逐步走向了全国。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
<img alt="12.png" border="0" height="359" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/2745/12.png" width="603"/>
<br/>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   谢几何：
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “我是那次座谈会的参加者。发起和主持人是知青刘锦长，是个女同学。参加者还有后来因此次座谈会而写了《长沙知青运动考察报告》和《中国向何处去》的著名学者杨曦光
  </span>
  (
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   后改名杨小凯
  </span>
  )
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   等。我们广泛探索了知识青年问题、文化大革命问题、以及中国向何处去等问题。会后，我们又组织了代表团，前往广州。住在沙面的农垦厅招待所，又在广州召开了座谈会。并协助广州的知青办报纸，从组稿、编辑，一直到出主意搞纸张印刷……代表团还直接到广东增城县等广州知青下放点去‘传经送宝’，扩大影响……。”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  (
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   七
  </span>
  )
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   长沙知青大逃亡最后的结局
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   在当时的历史背景下，长沙知青的反抗注定是要失败的。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   当长沙知青运动一波波高涨，全国各地的知青运动也在伺机起动时，有关部门高度紧张，特意下发了文件，要求回城了的知识青年立刻返回农村去。
  </span>
  1967
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   年的
  </span>
  10
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   月
  </span>
  8
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   日，毛泽东对该文件批示了“已阅，同意”
  </span>
  (
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   通常简称为“
  </span>
  10.8
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   通令”
  </span>
  )
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   ，刚刚充满了热情的知青们，一下子掉进了冰窖。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   就在知青们不知道何去何从时，长沙城发生了一件胆大妄为的“反革命”事件！
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   一夜之间，有人在几乎所有贴在大街小巷上的毛泽东批示后批上：“已阅，不同意！”，落款是“知青”。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   政府一方面严密封锁消息，追查冒犯天颜的“罪犯”，一方面公开抓捕逗留的知青，有的派出所一天竟抓了三四百人，然后用大卡车送回江永。江永方面答允：回乡者每人发
  </span>
  5
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   元钱、一担谷。政府另一面又对知青父母施加压力，逼迫知青回乡。政府还拿出了最厉害的一招，停发了发给知青的口粮。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   没有饭吃，逃亡回城的知青再无办法，只好被迫返回江永、零陵等地。部分坚持不走的，只能躲藏在长沙的河码头、建筑工地，打零工，买黑市粮票度日……
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  1967
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年年底，逃亡的长沙知青几乎全部回到了零陵地区，湖南知青的第一次大抗争宣告失败。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  1967
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年
  </span>
  11
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

  </span>
<o:p>
</o:p>
</p>

<span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>

  </span>
<o:p>
</o:p>
</p>

  </span>
