minor-fareast">

 <p class="MsoNormal">

 <p class="MsoNormal">

Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    人与人之间不再像原来那样有着密切的关系和交往，不再像过去那样每到过年时相互串门，集体上街玩等等。为死者守丧和送葬，在农村反而成了村里人团聚和交流的一个契机。这也是我在家乡看到的唯一能够让大家团聚的方式。
   </span>

 <p class="MsoNormal">

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

    （
   </span>
   1
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

    ）妻子
   </span>
   <o:p>
   </o:p>

 </p>
 <p class="MsoNormal">

 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">

 <p class="MsoNormal">

minor-fareast">

 <p class="MsoNormal">
  <o:p>

   </font>
  </o:p>
 </p>
 <p class="MsoNormal">

minor-fareast">
    曾听过一个搞量化统计的学者对农民工的调查报告，得出的结论之一是：农村孩子结婚越来越迟。
   </span>

   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    但我看到的情况恰恰相反：因为女孩子难找，男孩子到二十岁，父母就张罗着给儿子物色对象，物色好对象之后，既怕女孩子变心，又考虑要到城市讨生活的现实情况，就催着孩子赶快结婚。

   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    可以想象：在现代社会这种动荡不安的生活中，这样的婚姻会出现多少问题！事实上，农村离婚的情况，也是与日俱增的。

   </o:p>

    （
   </span>
   2
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

    ）房子
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    过年时，有的也会把父母接到县城过年，但父母住不惯，在县城过了大年，初一就赶回来了。在老家的生活是“老米酒，蔸子火，除了神仙就是我”，而在县城除了那套房，什么都没有。

 <p class="MsoNormal">

 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    老百姓不认识车子的牌子，不知道车子的价位，只知道这些车叫“小车”。不管什么小车，关键是要有！
   </span>

   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>

   </font>

 <p class="MsoNormal">

minor-fareast">

 <p class="MsoNormal">

   </font>
  </o:p>

 <p class="MsoNormal">

minor-fareast">

minor-fareast">

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">

   </span>

 </p>
 <p class="MsoNormal">

 <p class="MsoNormal">

minor-fareast">
    这十多年来，外界对于农村的关注主要集中于农民工身上。众所周知，他们在城市打工的日子很苦，而家里的老人和孩子往往无人照料。其中酸甜苦辣自不待言。
   </span>

 <p class="MsoNormal">

 </p>
 <p class="MsoNormal">

minor-fareast">

   </o:p>
  </font>
 </p>
 <p class="MsoNormal">

   <font size="3">

  </o:p>

 <p class="MsoNormal">

minor-fareast">
    倒是有两类家庭，他们处于最困难的境地，却往往被忽视。一类是孤寡老人。一类是举全家之力，把子女培养成大学生的家庭。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">

 </p>
 <p class="MsoNormal">
  <font size="3">

minor-fareast">

   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>

 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    “有钱人吃低保”，早已成为农村公认的一桩怪事。过年的时候，大家也不再像传统社会那样，家家户户给这些孤寡老人送点东西。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    这里所谓第二类家庭，主要是指有孩子在
   </span>
