   大概是觉得太没意思了，便停下笔来。《随笔》编者多次约稿，无以为应，只得找出这几篇东西寄上。附上这几句，说明这是二十五年前的旧稿，当时并无发表之意。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<img alt="10.jpg" border="0" height="266" src="https://i.imgur.com/tYRhCYU.jpg" width="190"/>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  1979
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，李锐平反复出后随康世恩任团长的中国能源代表团，出访时护照所用照片。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  1922
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年父亲去世的时候，我不满
  </span>
  5
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   岁，大姐
  </span>
  8
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   岁，二姐
  </span>
  7
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   岁，妹妹还不到一岁。父亲虽然做过不小的“京官”
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   民国初年的国会议员
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，但“为人清正”，因而“身后萧条”。由朋友们的帮助，好不容易父亲的丧事完了，跟着来的问题是：我的母亲－－
  </span>
  30
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   出头的寡妇，带着四个儿女如何生活下去？
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   我们家住在长沙，平江老家的祖母和父亲一辈好多房人，都是靠收地租吃饭的。我的大伯父，家中的长辈，这时给我母亲汇来二百元钱，信中写道：“望弟媳携子女扶柩返乡”。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   父亲同我母亲结婚之后，
  </span>
  1905
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   年便到日本留学去了。那个时候是张之洞的官费母亲比父亲小
  </span>
  7
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   、
  </span>
  8
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   岁，很不愿意呆在封建大家庭中做媳妇，便由父亲安排，到平江县城他的朋友新创办的一所女子师范读书。为此，她曾受到许多歧视和虐待，同威风凛凛的婆婆作过不少抗争。因为那个时候女人读书是很奇怪的。每年开学进城时，婆婆只给媳妇一吊钱的零花钱，连买纸笔墨都不够。还是父亲节省读书的官费，补贴妻子，母亲才在这个学校毕了业。父亲是最早的同盟会员，有民主思想，常给母亲写信，反对老家的子侄靠收租过日子，主张人要“自食其力”。对于那个乌烟瘴气的老家，对那些游手好闲、抽大烟、嫖赌逍遥的子弟，母亲是深恶痛绝的。因此，接到大伯父的这封信后，立即打定主意：决不回老家去，不管将来日怎样困难，要带着四个孩子在省城念书。她写了一封剀切陈辞的信给大伯父，并且将那笔钱退了回去。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<img alt="11.jpg" border="0" height="412" src="https://i.imgur.com/qW8MNdP.jpeg" width="273"/>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   作者的父亲李积芳在北京任国会议员时的照片。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   由于一时筹不到钱，父亲的灵柩没有运回老家去安葬，寄存在长沙南门外专门停放灵柩的处所，这种地方叫“厝屋”，里面很是简陋。走进大门，是一块空无所有的泥土地门厅，左右伸出很长的廊道，廊道还有拐弯抹角的分道。廊道两边整齐地排列着停放棺材的小房间，用木板隔开，没有顶棚，每间恰好放下一口棺材。一走进厝屋的大门，就有一股泥土夹着漆醋的特殊气味扑鼻而来。除开偶尔能碰到守屋的老头以外
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   这个老头也是从来不开口讲句什么话的
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，里面从来见不到一件活的东西，连猫狗也没有一条。射进来的阳光少，夏天也是凉气逼人；走进长廊，更是阴森森的。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<img alt="12.jpg" border="0" height="365" src="https://i.imgur.com/HGZtgsT.jpg" width="513"/>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  1916
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，李锐出生前一年，父亲李积芳与母亲李张淑在北京的合影。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   一间间的小房间都没有关严，里面的黑色棺材和香烛供物，好像故意要跟人打招呼似的。有两口用麻绳捆缠得很严密的棺材，就搁在门厅后面进廊道的入口处，好像守卫着长廊。母亲告诉过我，这是被赵恒惕杀害的两个人，领导罢工的－－黄爱与庞人铨的灵柩。小孩子喜欢听鬼怪的故事，又总是怕鬼和怕黑暗的。我每次随母亲到这里来，心绷得很紧，贴着母亲身边，半步也不敢挪开，差不多每隔一个月，母亲就要带我来一次，父亲去世不久，由于照顾不好，小妹妹随着也死了，这更增加了母亲的痛苦。母亲所有的委屈和苦难，都到这里来向父亲的灵柩倾诉，还没有走到那间小房子，便放声号啕大哭起来。打开那扇小木门，母亲紧抱着我跪在灵柩前，哭得更伤心了。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   母亲是并不迷信的，家中从不敬天地祖宗，也不用迷信物品。只是过年过节父亲的生日忌辰，摆几样菜在父亲的照片前，我们向着照片磕头。到厝屋来，她也从不带香烛之类的东西。长期摆放在灵柩前小桌子上的，是父亲的灵牌和随灵送去的一对“金童玉女”小纸人，母亲哭得太厉害的时候，我也跟着哭起来。我还记得，当年母亲头磕得太多了，前额正中曾长期留下一小块红印记。那个守厝屋的老头，对我们母子俩似乎也习惯了；我的记忆中，几乎没有遇见过来哭灵的人。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<img alt="13.jpg" border="0" height="346" src="https://i.imgur.com/jwrF4FI.jpg" width="576"/>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   湖南省平江县长寿街李锐故居，早已捐给当地政府建成学校。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   没有了父亲，我们的家突然变得狭小而黯淡无光，生活中种种不幸几乎都由此而来，这在幼小的孩子心灵中，是渐渐感受到了。每回母亲带我到南门外去，去“看看爷爷”，我很害怕，但又愿意去。走进那阴森森的长廊时，心里特别害怕，但和母亲一起跪坐在那所小房的门槛上，对着面前那口大黑棺材时，又好像亲近了“爷爷”似的。父亲留在我的记忆中，是一个严肃而和善的人。他带我上街买过小洋号；我先起床时，曾经把他的鞋子藏到厕所里去过；他死去的那天早晨，我在床头摸过他那双冰冷的脚……母亲在灵柩前哭完，也累了，总是坐在门槛上，把我紧紧地抱在怀中，用湿脸挨着我的脸。再安静地坐一阵，我们才回家去。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   这样哭灵的生活，继续了近两年，直到父亲的灵柩运回老家安葬。那时留在我印象中最深的，就是父亲灵前小桌上那一对“金童玉女”，对它们有着特别亲切的感情，似乎觉得它们真好，日日夜夜陪伴着如此寂寞的“爷爷”。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   秦城旧作《忆童年》长诗中的开头八句：就是写的哭灵之事。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   人世凄凉事，失怙未五龄。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   月月随阿母，南郊哭父灵。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   母亲泣且诉，厝屋阴森森。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   间间大黑柩，对对小纸人。
