 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   《论语·颜渊》里有句话
  </span>
  :
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   “以文会友，以友辅仁”，这句话成就了一所在中国影响很大的高等学府，那就是辅仁大学。辅仁大学是
  </span>
  1925
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年由天主教所创办，“辅仁”二字即取自这句话。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   早在
  </span>
  1912
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，中国天主教知名人士丹徒马相伯、北平英敛之两位先生，为振兴公教教育上书罗马教廷，“请派高才硕德之教士，来华创设公教大学，发展中国固有之文化，介绍世界科学新知识。”后来因第一次世界大战爆发，办学之事便搁置了。直到
  </span>
  1919
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年，罗马教廷才派人来华，考察中国教会教育情形。经过一番调查，“亦以公教高等教育机关尚属缺乏，因是，教廷确悉中国公教设立大学为刻不容缓之事。”
  </span>
  1920
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年，美国本笃会神父奥图尔博士来华，进一步调查教育。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1923
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年
  </span>

  </span>

 <p class="MsoNormal">

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;

 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   陈垣
  </span>
  (1880-1971
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   年
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，字援庵，广东新会人，是我国著名教育家、历史学家。年轻时，他在广州创办刊物，宣传反清思想。
  </span>
  1911
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">

  </span>
  1913
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年春，陈垣以革命报人身份当选国民政府议员，赴北京参加中华民国第一届国会，从此定居北京。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   陈垣与西城有着不解之缘。他从广州来京后，住在宣武门内国会街
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   今新华社所在地
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，在这儿一住就是
  </span>
  8
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年。后来搬到西安门大街住了
  </span>
  5
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年。随后，在诩教寺
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   今育教胡同
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   、丰盛胡同、米粮库、南官房、李广桥西街
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   今柳荫街
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   等处都住过。最后，辅仁大学腾出定阜街学生宿舍的前院，截出一个独立的四合院作为陈垣的居所，年届花甲的陈垣才免去频繁搬家之苦。这里就是兴化寺
  </span>
  5
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   号院
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   今兴华胡同
  </span>
  13
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   号
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，陈垣在这里居住了
  </span>
  30
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   多年，直到逝世。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   尽管陈垣数次搬家，但历次住所始终未出西城。他致力于教育事业的发展，
  </span>
  20
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   世纪
  </span>
  20
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年代以来，他创办平民中学、担任教育部次长，后来担任辅仁大学副校长、校长，他把大半生的精力献给了教育事业，在西城教育发展史上留下了辉煌的一页。尤其是辅仁大学，从创立之初到并校关闭，陈垣几乎一直主持学校全面工作，成为辅仁大学的一面旗帜和精神支柱。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  20
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   世纪三十年代，辅仁大学在陈垣的领导下不断发展。
  </span>
  1930
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年，涛贝勒府南院新校舍落成，建有实验室、图书馆、教室、办公室及宿舍等。于是就以新校舍为大学部，原址改设附属男子中学，另在西城太平仓购地筑舍，
  </span>
  1932
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年开办女子中学。
  </span>
  1935
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年，辅仁大学受罗马教廷委托，开办中国天主教司铎进修机构，购买位于校区东侧的恭王府旧址，计地百余亩，成立“司铎书院”。
  </span>
  1937
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年，增设预防斑疹伤寒血清及国文、历史等研究所。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   抗日战争爆发后，北平沦陷，国立高校纷纷迁移内地，最后只剩下辅仁大学在北平。八年抗战期间，陈垣作为辅仁大学的校长，在复杂的形势下与日伪当局周旋，艰难维持学校正常上课。辅仁大学于
  </span>
  1933
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年后由美、德两国圣言会接办。
  </span>
  1936
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年后，德国人雷冕神父担任辅仁大学校务长。由于德国是日本的盟国，所以辅仁大学得以继续开办。而且辅仁大学一直坚持“三不”做法，即
  </span>
  :
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   文理各科仍用原有教材，不改用日文教材
  </span>
  ;
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   不悬挂日本国旗
  </span>
  ;
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   日文不作为必修课。这在当时严酷的环境下实属不易，也与陈垣的声望是分不开的。当然，日伪当局是不会轻易放过陈垣的，总是企图拉拢他，委以各种官职，但都被陈垣严辞拒绝。陈垣对国破家亡的景象非常痛心。有一次，他和几位师生在学校三楼观看书画展，看到楼下日军趾高气扬走过。陈垣低声吟诵
  </span>
  :
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   “登临独恨非吾土，不为城关画角悲”，吟毕潜然泪下。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   陈垣在北平处境十分危险，他也曾考虑过离开北平。但师生们的期望、辅仁的前途命运，使他下不了决心。
  </span>
  1943
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年底，陈垣的学生柴德赓准备悄悄地助师南下，被校务长雷冕神父察觉。雷冕神父极力挽留，以致失声痛哭。他认为陈垣留在北平有利于辅仁，有利于师生。校长一走，辅仁将难以为继，很快就会关闭或被日伪当局接收，几千师生顿失依靠，将难以就职和求学。陈垣百感交集，反复思考，最后还是决定留在北平。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在沦陷区北平，辅仁大学是一座“孤岛”，为不甘心就职于日伪大学的知识分子保留了一块栖息之地。许多知名教授纷纷转入辅仁大学，一些爱国志士也以辅仁大学为基地，秘密从事地下工作。辅仁大学虽未停办，但师生被日伪当局逮捕却是常有的，如英千里、沈兼士、胡鲁士神父
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   荷兰人
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   、范文澜、赵光贤、李德伦、萧乾等。每一次陈垣都要积极营救。在那八年时间里，陈垣还尽可能地向一些生活困难的学人约稿，藉此补贴他们的生活。在课堂教学和著述中，陈垣大力宣传爱国思想，鼓励年轻学子努力学习，将来报效国家。在这一时期他写出了《明季滇黔佛教考》、《南宋初河北新道教考》、《中国佛教史籍概论》等著作。而最能体现他深明大义、保持民族气节的著作，则是一部研究胡三省思想的作品—《通鉴胡注表微》。胡三省是宋末元初人，人元后不仕，退隐作《通鉴》注，抒发爱国情怀。陈垣通过对胡三省的研究，表达自己的爱国之志。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   综观北平辅仁大学，自
  </span>
  1925
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   年之辅仁社起，至
  </span>
  1949
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年
