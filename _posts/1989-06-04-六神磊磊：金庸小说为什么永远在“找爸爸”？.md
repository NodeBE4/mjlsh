  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    后来有关方面纠正了错误。这件事距离今天已经六十多年了。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    这是金庸的一大伤心事。只因为他前些年自己公开写了这事，不然，我也感觉不忍心说。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    查树勋先生是不是地主呢？是地主。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    众所周知，他的家族，是浙江海宁当地的名门望族，祠堂上还挂着康熙皇帝亲笔写的对联：“唐宋以来巨族，江南有数人家。”
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    到金庸出生的时候，家里据说已经没落了，但还是有
   </span>
   3600
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    多亩土地，
   </span>
   100
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    多户佃农。金庸弟弟的回忆里说，家里的宅子是五进的大院，有
   </span>
   90
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    多间房子，有大花园，在镇上还有钱庄、米行和酱园店。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    二
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    但金庸的父亲还不是个普通的土地主，而是受过那个时代中国最好的高等教育。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    查树勋生于
   </span>
   1897
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    年，是当时著名的教会大学－－上海震旦大学的毕业生。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    因为现在没有这所学校了，大家可能不了解。震旦大学和后来的复旦大学渊源很深的。后来马相伯先生办复旦大学，意思就是要“复兴震旦”。（另一说是来自于《卿云歌》）
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    建国后，震旦大学被肢解了，院系被分散并入复旦、交大、同济等学校。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    其实，不但父亲是个高材生，金庸的两位伯父也都受到了很好的教育。他的二伯是当时北大国文系的学生，大伯是清朝的秀才。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    在当时的落后的中国，查树勋是典型的精英，中西结合的人物。据称他曾鼓励金庸说，你表哥徐志摩读剑桥，你以后也要上剑桥。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    有一年圣诞节，他送给小金庸的礼物是狄更斯的一本小说，叫做《圣诞颂歌》，内容是讲一个冷酷的守财奴，受到了精灵的启发，变成了一个善良的人。这本书对金庸的性情、人格影响很大。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    想想看吧，一个出生于
   </span>
   19
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    世纪的中国的父亲，给孩子过圣诞节，送礼居然能送出狄更斯的小说来，还不是《双城记》《雾都孤儿》这样大家耳熟能详的作品，而是到今天大家都不太熟悉的《圣诞颂歌》。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    你能看得出这个父亲的视野和文化品位。到今天，我们很多父亲都送不出这样的礼物来吧。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    说到这里，很惭愧，出版社送了我一套《圣诞颂歌》，到今天都没拆开来读。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    三
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    查树勋在乡里办了不少好事。在我翻找的有限资料里，没有发现恶行，而是有不少赡赈、办学之类的义举。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    比如他晚年花大力气办“义庄”，出力周济宗亲，兴办教育。他给金庸送书，让儿子不要做冷酷的守财奴，他自己似乎也在努力做些善事。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    根据《文史月刊》赖晨文章介绍，查家把
   </span>
   1000
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    亩水田充当本族“义田”，请地方官把这些田另立了户名，登记注册。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    这些义田是上好的良田，旱涝保收的。扣除赋税和损耗后，一般每年可收租谷
   </span>
   3500
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    石，变卖后按月周济族人。每到秋收，查树勋会视年成的好坏，给佃农减租或免租。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    查树勋热心办学校，从义庄拿出了一笔钱，兴办了龙头阁小学，孩子们均免费入学。让人感慨的是，他后来就是在这所小学的操场上被枪毙的。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    当时他们镇的金竺庄上有一个穷秀才，叫杨良丰，儿子杨德举考上了震旦大学，杨良丰太过激动，发病死了。由于家里太穷，只能用破箱子改了棺材，遗体停在房里。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    查树勋听说之后，带着管家前往，重新给杨秀才置办了棺木，从查氏的田地里划了一块作墓地，从义庄出钱安葬了杨秀才。之后又资助了杨德举的学费。杨德举毕业后曾在南京银行任职，后来又任四川凉州行署专员。
   </span>
   <span lang="ZH-CN">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    四
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    变故发生在上世纪
   </span>
   50
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    年代。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    当时“镇反”运动大规模开展起来，搞“杀、关、管”。镇反人员组织村里人揭发查树勋的罪行，但他一贯为人很好，周济乡里，受到尊敬，没有人揭发他。最后，邻村有一个残匪出来揭发他窝藏枪支。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    最终给他罗织的罪名是四个：抗粮、窝藏土匪、图谋杀害干部、造谣破坏，决定以不法地主罪，予以枪决。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    “抗粮”是怎么来的呢？目前看到最详细的解释，是
   </span>
   1948
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年解放前，当地受了灾，国民党政府又要抓粮，催征催收。当地的中共地下党、农协会组织“抗粮会”，不给地主交租。查树勋是地主，当然也成了“抗粮”的对象。
   </span>
   <o:p>
   </o:p>
