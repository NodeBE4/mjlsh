  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   全称是《无产阶级革命派大联合筹备委员会》－－所有群众组织筹备酝酿大联合的组织
  </span>
  )
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   ，尔后是《大联委》
  </span>
  (
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   全称是《革命大联合委员会》，受军代表领导。
  </span>
  )
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   领导清理阶级队伍、批极左抓坏人运动，成立《调查组》，调查跨单位的重大要案。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1968
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年中央指出，《五·一六》的组织者和操纵者，是一个搞阴谋的反革命集团，应予彻底揭露。中央成立清查《五·一六》专案领导小组，号召在全国开展清查《五·一六》运动。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   外交部成立以军代表马文波
  </span>
  (
  <span lang="ZH-CN" style='font-family:
宋体;mso-ascii-font-family:"Times New Roman"'>
   注
  </span>
  )
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   为首的清查《五·一六》领导小组和专案组，号召广大群众迅速掀起一个清查《五·一六》反革命集团》的高潮，打一场深挖《五·一六》反革命集团的人民战争。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1969
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年第一季度，外交部下放大批人员到“五七干校”，接受“再教育”。当年第四季度又下放一大批人员去干校。两千多名人员包括全部“有问题”的人和专案组人员都离开外交部，部内只留下三百多人从事日常工作。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   外交部的“清查运动”主要在湖南、湖北、江西和山西等四地的外交部《五七干校》进行。《清查组》在军代表领导下按中央关于清查《五·一六》的精神，采取以往审干中采用的一些做法，诸如“坦白运动”、“一般号召”、“个别突破”，把那些极左的造反派当作《五·一六》分子的嫌疑对象进行审查。运动中没有极左表现，但也没有审查阅历和清查经验的“五七干校”学员充当专案人员。他们成为清查、审讯《五·一六》的主力。有些专案人员受派性的影响，混淆了敌我界限，引出一些假口供。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   外交部开展“十查”：
  </span>
  (1)
  <span lang="ZH-CN" style='font-family:
宋体;mso-ascii-font-family:"Times New Roman"'>
   查反革命黑手的滔天罪行；
  </span>
  (2)
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   查走资派的幕后策划；
  </span>
  (3)
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   查地富反坏右的罪恶活动；
  </span>
  (4)
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   查叛徒、特务、《五·一六》分子的阴谋诡计；
  </span>
  (5)
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   查
  </span>
  108
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   反革命案件在外交部的线索；
  </span>
  (6)
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   查失密、泄密、窃密事件；
  </span>
  (7)
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   查社会上反动组织或坏人在外交部的崌点或联系；
  </span>
  (8)
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   查野心家、政治投机商、政治扒手的煽风点火；
  </span>
  (9)
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   查对无产阶级司令部的立场、感情；
  </span>
  (10)
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   查妨碍揪坏人的“私”字。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   周总理很关心“清查运动”。在一次接见外交部军代表马文波、韩立业和“大联委”负责人时，周总理指示他们要作好清查工作，并多次提醒外交部防止清查《五·一六》扩大化。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1970
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年
  </span>
  3
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   月
  </span>
  23
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   日晚的一次外事活动后，冀朝铸
  </span>
  (
  <span lang="ZH-CN" style='font-family:

  </span>

  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>

  </span>
  ?
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   时间又这么短，极左思潮被《五·一六》利用了。但不等于有极左思潮就是《五·一六》。《五·一六》分子一定要搞反革命阴谋，推翻无产阶级司令部的，这样才能算是《五·一六》分子”。周总理叮嘱姬鹏飞要马文波军代表给各基地
  </span>
  (
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   指外交部的《五七干校》
  </span>
  )
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   打电报，提醒他们注意。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1972
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年，外交部上呈周恩来总理的报告称，外交部有
  </span>
  20
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>

  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   外交部清查《五·一六》的运动有乱咬、乱供的现象，也有逼、供、信，弄虚作假，以假乱真，真假难辨。有的嫌疑对象怕受煎熬，就乱咬、乱供
  </span>
  ;
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   有的却别有用心地为《清查组》专案人员设陷阱，胡诌瞎编。打击面宽了。另外，文革初期挨造反派批而后又重新掌权的当权派自觉或不自觉地对造反派伺机打击报复，彰显秋后算帐的情绪。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>

  </span>
  380
  <span lang="ZH-CN" style='font-family:

  </span>

  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>

  </span>
  <o:p>

 </p>
 <p class="MsoNormal">

"Times New Roman"'>

  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

  </span>
  1982

 <p class="MsoNormal">

  </span>

 </p>
 <p class="MsoNormal">

"Times New Roman"'>
   第一，“打倒陈毅”、“封闭部党委夺外交大权”、“砸政治部”等重大事件，是少数狂热分子煽动起来极左的、过火的行动。没有证据说明是反革命阴谋活动。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

 </p>
 <p class="MsoNormal">

  </span>

 <p class="MsoNormal">
