  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   她笑着说，也算是让儿子去完成他曾祖父对他祖父的希望和寄托。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   一百余年前的上海，孙道临的父亲孙文耀从震旦预科毕业，考取官费留学，赴比利时攻读土木专业。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙文耀希望子女能学理工，实业救国，孙道临没有听；父亲让他别搞文艺，他也没听。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

   孙道临出生在嘉善书香之家，这个孙家的第五个孩子，后来成为中国电影百年标志性人物，亦是中国电影史上代表男演员的丰碑之一。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   诗人西川说，在孙道临的身上，集合了一个时代的道德、修养、热情、才华。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   【印象】
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “父亲是一个很执著、很独立的人”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  1940
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，深秋，不到

  20
  <span lang="ZH-CN" style="font-family:宋体;

宋体;mso-fareast-theme-font:minor-fareast">
   岁的孙道临满怀心事从北平回到故乡浙江嘉善。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   孙道临后来回忆：
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

   “那是我首次离开了家，独自住在故乡，一个荒凉的小城里面，最初是一点故昔之情静润着我，使我的心情很平稳，城里面的乡音和街道，和缓缓行过物，都给与我安静的心情。”
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

  </span>
  )
  <o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">

  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  1938
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

  </span>

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;

<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">

  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

   孙道临
  </span>

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">

  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

</o:p>
</p>

</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

</o:p>
</p>

<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">

  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

  </span>
  19
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   岁时在《碧玉簪》中就是头肩花旦。
  </span>
  1962
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年
  </span>
  7
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   月
  </span>
  2
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   日，两人成婚。唯一的女儿孙庆原生于
  </span>
  1964
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   【影响】
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “父亲最注重的是我的精神食粮”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙道临给女儿取名叫庆原，是为了纪念中国原子弹爆炸成功。按祖传的家谱，女儿应该是“经”字辈。孙庆原的名字是家族中
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   她这代
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   唯一没有按家谱取名的，孙道临为此写信给自己的兄弟姐妹，陈述理由，恳请理解。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   对于自己富有时代特色的名字，孙庆原说：“上世纪
  </span>
  60
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年代的时候，中国有能力去造原子弹，这是让父亲这辈很感自豪的事。”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙庆原的童年、少年都是在武康大楼的家中度过的。武康大楼在上海淮海中路与淮海西路交接处，楼身侧看很像一艘大船，原来的名字叫诺曼底公寓，由旅居上海的匈牙利设计师邬达克设计，解放后上海的一些文艺界名流均入住此间。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  1965
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，孙道临一家住进武康大楼，那时孙庆原才牙牙学语。从孙庆原有记忆开始，家里很挤。“我们一家子好几个人住三间房间，我的舅舅、舅妈和外婆跟我们住在一起。”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   女儿是孙道临人生中重要的一部分，但孙道临继承家风，对女儿非常严格，以至于她小时候怕自己的父亲。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙庆原清楚地记得这样一件事：“父亲前一天跟我说好今天把琴弹好，他晚上回家带我去看电影，但是我没弹。晚上他回来的时候说，电影票已经买好了，你把琴弹一遍给我听。但是我弹不出来，他当场就把票撕掉了。”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   父亲教育她：“你的承诺，你要兑现，要做一个可信任的人，说了话就要算数。”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   除了学习，孙道临非常注重对女儿生活能力的培养。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙庆原小时身体不好，父亲只要在家，每天早晨不管多冷，都带她一起跑步，六点半准时把她从被窝里拉起来。“从幼儿园一直到小学三四年级，这个习惯坚持很多年，我身体慢慢好起来。”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  8
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   岁那年，孙道临开始教女儿骑自行车，那时的生活条件不及现在，她只能用父亲的“老坦克”学骑车。黑色的“老坦克”又高又沉，她几乎是站在上面骑，很辛苦。在安静的余庆路上，父亲来来回回扶着车跑。“因为他觉得骑自行车是一门本领，在上海你一定要学会骑自行车，现在想来，那时的爸爸已经年过半百了。”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙道临的“老坦克”是
  </span>
  1964
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   年买的，凤凰牌，黑色车身，
  </span>
  28
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   英寸。文革的时候，为了这辆自行车不被抢了去，他的母亲遭遇人生第一次耳光。这辆“老坦克”现在被收藏于电影博物馆。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   作为艺术家庭的孩子，孙庆原的童年是孤独的，因为父母常年外出演出不在家，她和外婆住在一起。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   小时候别的孩子都不愿意去幼儿园，但她却盼望去幼儿园，反复问：“爸爸，你什么时候带我去啊？”因为到幼儿园就可以跟小朋友一起玩了。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙庆原毕业于上海外国语学院，读的是英文文学专业。大学毕业后她工作了一年，跟着当时学工科的男友一起去了德国留学，选择的是自己感兴趣的企业管理。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   她在德国汉诺威大学念了两年，后找到了工作。“我们当初不像现在的留学生，生活都由父母资助。那个时候一方面受生活条件限制；另外我觉得学的是企业管理，需要实践经验，我可以在实践中学习。”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在德国的
  </span>
  9
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，孙道临知道女儿喜欢看《新民晚报》，每个月都给她寄厚厚的一叠《新民晚报》，“父亲最注重的是我的精神食粮，母亲更偏重我的衣食冷暖，所以他们两个人给我寄的东西不一样，妈妈会寄些衣物零食和南北货。”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   父亲的意思，是希望女儿人在国外，也要关心国内的事情，不要和国内有脱节。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   受父亲的影响，孙庆原的性格比较像父亲，做事认真，自律，比较有计划性。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   她从
  </span>
  2000
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年起在美国通用电气公司工作。她的丈夫朱解鸣在一家德国公司做管理工作，他们有一子一女。儿子今年高中毕业去了美国留学，女儿则在读高二，“女儿很会写文章，《孙道临自传》一书中其中一篇序言就是我女儿
  </span>
  9
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   岁左右写的。”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙道临不仅是艺坛上的常青树，他还做过不少影视剧的导演。
  </span>
  1996
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年，这对夫妻还合作了一把－－由他导演，王文娟主演越剧电视剧《孟丽君》。晚年，他把很多精力放在播音和朗诵上。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙道临曾对女儿说，要把对一些问题的思考，用“留言”的形式写下来。可是
  </span>
  2005
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年，他因带状疱疹住了很长时间的医院，记忆力很受影响。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  2005
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，上海节目主持人曹可凡在医院采访了他，准备做一档《可凡倾听》节目，他让女儿到家里找来烫好的衣服，可是他不满意衣服的色调。王文娟给他换成米色西装，白衬衫，绛红领带。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙道临一生都喜欢白色。“他的衣服，他的文章跟资料，不许人家碰。”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙庆原说：“他要求你出去见人要穿得整齐，这是对别人的尊重。”在孙庆原看来，父亲很会从别人的角度看问题。“从小他就跟我说：你这样，你想想另外的人会是什么感觉？”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙道临对故乡嘉善的感情很深，他不太喝酒，但推崇充满乡情的嘉善黄酒，对于家乡提出的一些要求，他很少回绝过。每次回去，都会去父母的墓跪拜。“包括建孙道临电影艺术馆，他也去了很多次，那时候他身体已经非常不好了。”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   嘉善魏塘城中的下塘街孙宅，曾是孙道临的父亲出生的地方。孙庆原说，好几年前她曾和姑姑去找过，看到的是一地碎片，“后门一条河都平掉了。”她说，父亲的兄姐和他们的后代，基本都在海外和北京。据嘉善县政协文史委的人介绍，留在嘉善老家的，有孙道临的堂弟孙以冕。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  2007
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年春，孙道临电影艺术馆落成，这是新中国第一座以个人名字命名的电影艺术馆。落成之日，孙道临应邀回故乡，这是他最后一次回故乡。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   当日，当参观者潮水般涌进艺术馆展厅时，他顽强地离开轮椅，在亲人的搀扶下，缓缓走向长廊。当走到母亲的照片前，一路安静的他停了下来，拍了拍女儿的手说：“这是你祖母－－我的母亲。”
  </span>
  1966
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年，他被打入“牛棚”，他的母亲快死了，他得不到回家许可。等到他回家后，母亲的身体已经冰冷了。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
  2007
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

   年
  </span>
  12
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

  </span>
  28
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   日，这位从燕京大学哲学系走出的艺术大师离开了人世。他去世后穿的衣服，也是他生前配好的，那是他为下一场演出做的准备。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   【对话】
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “对社会有贡献，并不妨碍我去做个自由的人”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   记者
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   下简称为记
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   ：黄宗江说你父亲是一首诗，是一首舒伯特和林黛玉合写的诗，你怎么理解这句话？
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙庆原
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   下简称为孙
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   ：父亲和母亲成长在完全不同的家庭背景里，虽然有如此多不同，但他们合起来就像一首诗一样，那么和谐。这是我对宗江伯伯这番话的理解。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   因为爷爷是个留学生，他的生活方式比较西化，包括饮食，父亲会做很多西餐……妈妈小时候生活在浙江农村，喜欢吃的是浙江菜，腌的咸的酱的，妈妈的饮食是非常中国化的。虽然生活背景不一样，但他们有一个共同的追求，就是对艺术的热爱，对生活永远有一种积极的态度，所以他们又是非常相像的。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   记：你觉得父亲一生受谁的影响最深？
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙：父亲的善良以及做事态度都受到了家庭的影响，但他的思想非常独立。他认准的理，就会坚持下去。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   记：艺术家的父亲在生活中是什么样的？
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙：父亲非常幽默，跟他不熟的人可能会觉得他很严肃，会让人害怕；但在跟他接触以后，你会发现他是个非常有意思的人。跟他在一起你会一直笑，但他不笑。他生病的后期，我去看他，我说，你要运动，你老坐着不动，肚子会越来越大。他说：“唉，你不懂，我肚子大是因为我满腹经纶。”
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   记：你后来从事的是企业管理，父母对你的选择是什么态度？
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙：父亲觉得没问题，他非常尊重我的选择。只有一次，我在高中学文科还是理科的时候，比较关键的时刻，他帮我做了决定，让我去学文。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   我见的几乎每一个人都会问我同样一个问题，你为什么没有走你父母的那条路？当然，如果你做的事和父母的专业是一样的，可能更容易成功；但你做了不同的专业，虽然不能达到父母那么高的高度，但更能证明自己。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   记：你是不是不那么喜欢文艺？
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙：我觉得从事文艺的人，很少自由，我不是很有欲望去做文艺。其实做演员，很不容易，一直生活在别人的关注中是很累的，他们要比普通人承受更大的外来压力，包括他们的小孩也挺辛苦的。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   自由做人，做你想做的事情，我觉得这点很重要，也很珍贵。当然我对人生也有自己的看法，要可信，要自律，要对社会有贡献，但这并不妨碍我去做个自由的人。我的人生观，简单说，我不要在光环下，我要自由，我要老百姓的自由。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   记：现在有种说法，说从事父母的职业，可以享受“祖荫”。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   孙：享受祖荫并不需要从事父母的职业，我平时接触人更容易得到别人的信赖，这就是我享受到的祖荫。
  </span>
