<br/>
</p>
<p class="p2">
<span class="s1">

  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   政治运动
  </span>
<span class="s2">

  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   群众专政
  </span>
<span class="s2">

  </span>
<span class="s1">
   方法的普遍采用和校园暴力的大规模发展；另一方面是迫害的对象进一步扩大，不但在人数上增加，而且被迫害的群体种类也越来越多。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   从受难者的角度切入反右和文革历史，不但出于对人的生命和权利的尊重的基本价值，也因为只有了解受难者才能认识这两个大事件，因为对人施行迫害是反右和文革过程的主要组成部分。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>

  </span>
<span class="s1">

  </span>

  </span>
<span class="s2">

  </span>
<span class="s1">

  </span>
</p>

<span class="s1">

</p>

<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   各大学受迫害人数呈现明显的均匀现象，显然是因为反右和文革都是在中央的统一领导下进行的。而北大因其
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   最高学府
  </span>
<span class="s2">

  </span>
<span class="s1">

  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s2">
   2
  </span>
<span class="s1">
   、反右和文革是大规模群体性迫害
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   在以上描述中可以看到，反右和文革，都造成了一个数字巨大的受难者群体。也就是说，二者都是大规模的群体性的迫害。在笔者看来，这是二者主要的也是共同的特性。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">

  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   右派分子
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   后来和别的
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   敌人
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">

  </span>
<span class="s2">

  </span>
<span class="s1">
   地富反坏右
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，又称
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   五类分子

<span class="s2">
   ”
  </span>
<span class="s1">
   。这是一个
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   类
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，而不是个别人。这个
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   类
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   人数众多，可以百万计数，到了文革，在已有的
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   五类分子
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   之外，又加出了多个种类。例如，
  </span>
<span class="s2">
   1969
  </span>
<span class="s1">
   年的北大一份题为《关于清理和改造阶级敌人的情况报告》说，该年
  </span>
<span class="s2">
   7
  </span>
<span class="s1">
   月上旬到
  </span>
<span class="s2">
   9
  </span>
<span class="s1">
   月
  </span>
<span class="s2">
   2
  </span>
<span class="s1">
   日，在北大的教职员工中，清理出了
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   叛徒
  </span>
<span class="s2">
   ”“
  </span>
<span class="s1">
   特务
  </span>
<span class="s2">
   ”“
  </span>
<span class="s1">
   历史反革命
  </span>
<span class="s2">
   ”“
  </span>
<span class="s1">
   现行反革命
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   和
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   地富坏分子
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   共
  </span>
<span class="s2">
   102
  </span>
<span class="s1">
   人。这份报告由毛泽东批示
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   照办
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   后发给全国学习，编号为
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   中发〔
  </span>
<span class="s2">
   1970
  </span>
<span class="s1">
   〕
  </span>
<span class="s2">
   17
  </span>
<span class="s1">
   号。也就是说，在全国各地都要抓出这些
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   类
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   的
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   阶级敌人
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   来。在北大，三个月
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   清理
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   出的
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   阶级敌人
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   数字占教职员工的百分之二还多，在全国显然会制造出一个极大的绝对数字。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   除了立
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   类
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   之外，还制定百分比，以确保每一类都能填够配额。北大的
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   反右
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，先是按照百分之五的比例划的。
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   上面
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   嫌百分之五不够，北大就进行
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   补课
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，增加到了百分之七。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   在苏联档案中存有一些关于杀人和关人的命令。
  </span>
<span class="s2">
   1937
  </span>
<span class="s1">
   年
  </span>
<span class="s2">
   7
  </span>
<span class="s1">
   月
  </span>
<span class="s2">
   30
  </span>
<span class="s1">
   日的一份，列出各个加盟共和国要杀的人数从
  </span>
<span class="s2">
   100
  </span>
<span class="s1">
   到
  </span>
<span class="s2">
   2
  </span>
<span class="s1">
   ，
  </span>
<span class="s2">
   000
  </span>
<span class="s1">
   ，同时还另外列出了数倍于此的要逮捕入狱的人数。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   这样的命令非常恐怖：不但在于数字大，还在于其随意选定
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   敌人
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   的数字。同样，毛泽东以百分比来划
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   右派分子
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，也是同样恐怖。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s2">
   3
  </span>
<span class="s1">
   、文革是反右迫害的升级与扩大
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   上文已经描述，
  </span>
<span class="s2">
   1957
  </span>
<span class="s1">
   年反右的受害者，在文革的火坑中被推到更深处，在
  </span>
<span class="s2">
   1957
  </span>
<span class="s1">
   年还未成被
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   戴帽子
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   的人，文革也被关进了
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   牛棚
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，甚至连反右的积极分子也被置于死地而丧生。一方面，反右和文革都是大规模的群体性迫害，另一方面，后者比前者更为严重。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   不仅事实表明如此，从两次
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   运动
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   的发动者和领导者毛泽东的理论中上，也可以明显看出。
  </span>
<span class="s2">
   1957
  </span>
<span class="s1">
   年，他创立了所谓
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   两类不同性质的矛盾
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   理论。他把公民分为两种，
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   人民
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   和
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   敌人
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，至于谁是
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   敌人
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   一类，则是由他规定的，比如
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   右派分子
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   。按照当时的宪法，公民享有
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   言论自由
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，但是毛泽东说他们是
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   敌我矛盾
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，他们就成了
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   敌我矛盾
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   。文革中知识分子被指控为
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   资产阶级知识分子
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，也是毛泽东一个人规定的。在文革中，毛泽东进而提出了
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   继续革命
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   的理论，也就是说，要把原来的革命更推进一层。在不断叠加的革命中，受难者越来越多越来越苦。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   本文第三节叙述了反右积极分子在文革中遭到迫害甚至北迫害致死。这是一个明显的事实，但是对此事实的解释，却并不一致。有的人以此证明文革的
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   荒唐
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，非逻辑，打击了不该打击的人。这一说法似是而非，首先因为这多少等于肯定了反右。另外，看施害一方，他们的革命就是迫害，他们在逻辑上和行动上相当整合，在基本方面并不自相矛盾。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   从理论到实践，文革领导人的改变意向是清楚的。江隆基和陆平这些领导了
  </span>
<span class="s2">
   1957
  </span>
<span class="s1">
   反右的人，即文革前的大学的领导人们，在文革中全数都被
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   打倒
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   。如笔者在《文革受难者》书中写到
  </span>
<span class="s2">
   20
  </span>
<span class="s1">
   名大学领导人之死时指出，这些人从未担任大学教员，由他们担任大学最高领导，已经改变了历来由教授学者中产生校长的传统。但是江隆基和陆平自己上过大学。文革开始，毛泽东就指控学校系统被
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   资产阶级知识分子统治
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   ，接着他派送了他的警卫部队中的迟群、谢静宜等没有受完中学教育的人担任清华和北大的负责人，直到他死亡。在这种暴力迫害的同时，毛泽东一步一步改变了教育体制。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   还有一种解释离开事实更远，即文革前受迫害的人在文革中
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   造反
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，反对迫害过他们的人。实际上，文革领导人对镇压右派分子颁布了一系列明确规定的有一系列明令规定。从北大案例可以清楚看出，文革中
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   右派分子
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   是受苦最多的群体之一。至于反右积极分子被
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   斗争
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，决非出自
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   右派分子
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   的报复，而是来自最高领导人的指挥。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s2">
   4
  </span>
<span class="s1">
   、这是一种什么犯罪？
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   杀一个人，就是犯罪。杀害了数字如此巨大的人，侵犯了如此多的人的自由、人权，在没有法律依据的情况下残酷惩罚了那么多人，当然更是犯罪。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   但是对于同一历史事实，记述和判断却有很大不同。在反右和文革的发动者和领导者那一边，他们称之为
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   伟大的革命
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   。直至今天，在反右和文革的同情者眼中，注意的是当时作为驱动力量一方的
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   理想主义
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   和高昂欢乐的情绪，他们回避提及受害者一方的苦难，更不承认这是一桩罪恶。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   在毛泽东死亡两年以后，邓小平和胡耀邦为反右和文革的受害者
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   平反
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，这是人类历史上最大规模的群体性
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   平反
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，在人道方面的功劳应该被记入史册，当然也是因为发生过的迫害的规模是前所未有的。他们为这个庞大的受害者群体的悲惨生存状况作出了一些实际性的改善。但是，在说法上，他们仅仅称此行动为
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   平反冤假错案
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，似乎这些都是个别性的案例。这个说法显然不符合事实，而且他们自己也并不真如此认为，否则不会给百分之九十九以上的反右和文革的受害者一起都平了反，而不是只给其中的一部分人甄别。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   对于反右和文革的反对者来说，有一个问题是，应该把它们称为什么样的罪恶？显然这是一种巨大的罪恶，然而既有的关于罪恶的名称，谋杀，盗窃，投毒，诽谤等等，却难以被用来指称这样的罪恶。反右和文革罪恶包含传统意义上的这些罪恶，但是这些名称只有很小程度上的适合，因为这些罪恶杀害和迫害的人数，都不可能与反右和文革相比。普通的罪犯，即使是惯犯或系列杀手，也不可能危及如此巨大数目的受害者。只有战争会危害如此多的人口，但是战争主要由军队持武器进行，而且战争是两军对垒的冲突，相当不同。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   在现实中，没有名称的罪恶就可能不被认为是罪恶。反右和文革的的罪行不但需要得到记录，而且需要命名。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   在抵抗和清算希特勒纳粹的漫长过程中，一位名叫
  </span>
<span class="s2">
   Raphael
  </span>
<span class="s1">
</span>
<span class="s2">
   Lemkin
  </span>
<span class="s1">
   的学者，用其毕生精力创造并推广了一个关于罪恶的新词
  </span>
<span class="s2">
   genocide
  </span>
<span class="s1">
   。在
  </span>
<span class="s2">
   Lemkin
  </span>
<span class="s1">
   和其它人的长期努力之后，
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   群体灭绝罪
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   成为在国际上被接受的罪名。
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   群体灭绝罪
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   是指蓄意全部或者部分消灭某一群体，包括杀害该群体的成员，造成该群体成员在身体上和精神上遭受严重伤害，故意使该群体处于某种生活状况下，毁灭其全部或部分的生命，强迫这一群体的儿童离开他们等等。这种群体，是基于政治、种族、民族、文化、宗教、等理由划分的。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   希特勒杀害犹太人的行动，就属于典型的群体灭绝。在那里，被迫害、监禁、罚作苦役以至被杀害，最重要的
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   理由
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   （加上引号，因为这仅是作恶者一方如此认为的）是该人属于
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   犹太人
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   这个按照民族划分的群体。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   这个模式是否适合反右和文革？在人数上，显然是的，在群体性方面，显然是的。不同的是用来划分这些群体的
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   理由
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   和名称。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s2">
   5
  </span>
<span class="s1">
   、历史的审判
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   在反右
  </span>
<span class="s2">
   50
  </span>
<span class="s1">
   年后和文革
  </span>
<span class="s2">
   40
  </span>
<span class="s1">
   年后，在书写和传播技术如此便利的电脑网络时代，已经有条件来做历史的审判，也就是通过历史写作来审判罪恶。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   这种历史的审判是有意义的吗？
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   文革后期，按照毛泽东的旨意在北大校园建立了一个
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   梁效
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   写作班子，发表了大批文章，主要内容是批判孔子和鼓吹秦始皇。例如，其中一篇文章题为
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   略论秦始皇暴力
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   （《人民日报》
  </span>
<span class="s2">
   1974
  </span>
<span class="s1">
   年
  </span>
<span class="s2">
   1
  </span>
<span class="s1">
   月
  </span>
<span class="s2">
   21
  </span>
<span class="s1">
   日），对在中国历史上一直受到谴责的秦始皇大加赞美，并把秦始皇
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   焚书坑儒
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   和文革
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   镇压反革命
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   直接等同，也就是说，要从历史的高度来肯定和合理化文革的群体性迫害与杀戮。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   现在回头看这个曾经不可一世影响很大的
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   梁效
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   ，也让人思考另一个问题：在中国历史上，能和秦始皇一样大规模
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   焚书坑儒
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   的统治者，毕竟秦始皇以后的两千多年里也只出现过一个。是什么力量起了阻挡作用呢？应该说，在秦始皇后两千年来人们把
  </span>
<span class="s2">
   “
  </span>
<span class="s1">
   焚书坑儒
  </span>
<span class="s2">
   ”
  </span>
<span class="s1">
   当作罪恶来记载和谴责，这些历史评判尽管以现代标准来说还不够深刻，却多多少少起了作用。
  </span>
</p>
<p class="p1">
<span class="s1">
</span>
<br/>
</p>
<p class="p2">
<span class="s1">
   这个历史的纵向观察告诉我们，通过历史写作对罪恶做出审判，在文字上、学术上对历史事实进行描述和分析，虽然不能像在司法系统中审判罪恶一样有力，但是并不完全无用。所以，对反右和文革做出历史的审判，仍然是重要的和有意义的。
  </span>
</p>
