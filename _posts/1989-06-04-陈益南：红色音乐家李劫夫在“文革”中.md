</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    由于年代，也许，你已不知道他是谁。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    但，你应该听过《我们走在大路上》这首歌，或许还知道《革命人永远是年轻》那支歌；
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    并且，几乎可以肯定，你在戴红领巾的那阵，必然唱过“牛儿还在山坡吃草，放牛的却不知道哪儿去了……”
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    而，这首被传唱了大半个世纪、并至今不衰的哀婉动人的《歌唱二小放牛郎》，其作曲者，就是劫夫－－李劫夫。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    至于在五、六十年代的中国，大批脍灸人口流唱甚广的歌曲，如《我们走在大路上》、《革命人永远是年轻》、《一代一代往下传》、《蝶恋花》、《沁园春·雪》、《哈瓦拉的孩子》等，则也都是李劫夫的杰作。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    劫夫是一个非常勤奋的音乐创作家。从他在抗日根据地开始创作歌曲时起，一生中，他一个人竟就写了二千多首歌；其中，有相当部分，被人传唱不衰。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    文革中，劫夫的创作，更是进入一个特别的时代：因那个时代的需要，他为毛泽东公开发表的所有诗词，都谱了曲，让毛泽东的诗词，成为了可歌可唱的流行作品；他还为那些散不成词、律不成韵的《毛泽东语录》，写下了大量的所谓“语录歌”，并被广为传唱。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    无疑，共产党员李劫夫的音乐创作，是非常的政治化，而且可以说，基本就是为政治服务。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    如何看待劫夫写的那些歌，自然可以仁者智者，各持己见。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    但是，劫夫则的确是诚心诚意地以艺术的规律去为政治服务，以音乐的美感来传导时代的旋律，而并不是生硬地将政治塞进歌曲。所以，他创作的歌曲，甚至连那些《语录歌》，都能让人传唱，流行多年。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    这是一个已经时间长期打磨了的事实。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    有人总主张艺术要远离政治脱离政治，认为有政治就会没有艺术。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    其实，这是一种偏见。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    我们之所以不喜欢政治介入艺术，是因为往往这种“介入”，不是生硬直白地塞入，就是以强暴艺术、压制艺术为前提，活活地扼杀了艺术。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    我们平日多见的所谓艺术介入政治的作品，如《
   </span>
   X
   <span lang="ZH-CN" style="font-family:
宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    啊，母亲！》之类，其实只是体现了宣传。
   </span>
</font>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    而宣传仅是政治需要本身，仅是目的，却还不是艺术。很遗憾，对此，许多的宣传人员与仁者智者，都没能看到想明。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    其实，具有政治意义的艺术，常常最能释放人性的情怀，展现人性美的高度，最能使人努力奋进，让人始终能常生激情，永葆青春。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    抗日战争时的《松花江上》，好不好听？
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    对越反击战时的《血染的风采》，动不动人？
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">

   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    当然，真正要让艺术体现政冶，并非容易的事，并非简单的事。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
<font size="3">

   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    风花雪月，固然也是艺术需要表现的范围，因为她们都是人性的组成部分。只是，政治往往显现了人性的刚，而风花雪月则往往展露了人性的柔。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
<font size="3">

</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

<font size="3">

   </font>
</span>

<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

   </span>
   1972
   <span lang="ZH-CN" style="font-family:宋体;

</span>

<p class="MsoNormal">

   3
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

    月，河北邢台大地震，劫夫奉命前往参加救灾活动。其时，遇到了周恩来总理。周对劫夫说：“劫夫，我最佩服你的‘大路上’（《我们走在大路上》），你的四段词我都会唱。”说罢，周总理还真唱了几句。
   </span>
</font>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

<font size="3">
    受此鼓舞，劫夫在灾区便创作了后来风行一时的《爹亲娘亲不如毛主席亲》一歌。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    劫夫一生中，担任的最后一个官职，是沈阳音乐学院革命委员会主任（
   </span>
   1968
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;

    年

   8
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

    月）。
   </span>
</font>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

    他的盖棺政治结论，辽宁省委在
   </span>
   1979
   <span lang="ZH-CN" style="font-family:

   </span>
   11
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;

    月对他下达的决定，则是：“犯有严重政治错误”。
   </span>
</font>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    而他最后欲拟创作的一首歌、并亲笔写下了的歌题，竟是《紧跟林主席向前进》！
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    －－这个“林主席”是谁？
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    是“林副统帅”、林彪啊！
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    为何会这样？下文将一一说明。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    上述这三件事，从劫夫一生的经历中看，其实都不是他要追求的东西。对当官，他的确历来无什么兴趣；对政治，他也从来没有过真正的认识；他想做的，就只是专业醉心于音乐创作。顶多，只想用音乐，来为党的事业服务。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    然而，鬼使神差，阴差阳错，劫夫却偏偏被那政治黑洞给吸了进去，且英华之年，便断了他的创作，也断了他的生命。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    打倒“四人帮”后两个月的一天，即
   </span>
   1976
   <span lang="ZH-CN" style="font-family:
宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    年
   </span>
   12
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    月
   </span>
   17
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    日，李劫夫因心脏病发作，猝逝于尚在办他的那个专案“学习班”中，终年
   </span>
   63
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;
mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    岁。
   </span>
</font>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    天妒英才，收他而去。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    （三）
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    李劫夫的“政治错误”，帽子很大，其罪名是“投靠林彪反革命集团”。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    一个音乐人，一个作曲家，怎么会陷到那个吓人的枪杆子集团中去的呢？
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    用一句政治术语说，是“事出有因”，但绝对“查无实据”。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    文革开始后，劫夫其实就很害怕。因为，他是个音乐天才，在音乐方面，能无师自通，但对其他方面，则常显幼稚。尤其，他的确不懂政治，不知道政治游戏中的潜规则，而他从延安时期起，却又看到过很多个人政治的悲剧。因此，对政治运动，他常常害怕，害怕会无端整到自己头上来。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">
   1957
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年反右派运动时，李劫夫就已名列省文化局右派“侯选”的黑名单中。在那运动中，他就害怕得精神有些失常，对家人，对老朋友说：“我完蛋了，就等着抓我的右派了！”
   </span>
</font>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    幸亏，当时的省委宣传部长是劫夫在抗日战争时的老上级，非常了解他，而负责抓右派的省委书记，平日也清楚劫夫的诗化性格，故不仅不抓他的右派，反而来劝他放心，不要作急。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    不过，
   </span>
   1959
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年进行党内“反右倾”，作为沈阳音乐学院院长的劫夫，却没能逃脱。因为，他那业务第一、政治第二的作风，几乎众所周知。因此，沈阳市委给他定了一个“犯有右倾机会主义错误”的结论，不过，还算幸运，没载正式的右倾帽子，只需作捡讨。
   </span>
</font>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    文革开始后，象成千上万的那些领导干部一样，劫夫自然逃脱不了被音乐学院的红卫兵批斗、抄家的厄运。对此，劫夫只好逆来顺受，等着挨批斗，等着被抄家。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    事情到此，劫夫与绝大多数其他领导干部们一样，若熬过了此关，以后文革及文革后的日子，无非就是捡讨捡讨所谓“走资派”之类的错误，然后，便能随“众神归位”的大流，可重新回到音乐学院院长之类的地位。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    然而，此时发生的一件好事，却可能在之后对他发生的灭顶之灾，预伏了危机。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    沈阳音乐学院的红卫兵，分为两派：一派为“红色造反团（红造团）”，是要打倒李劫夫的；另一派叫“井冈山”，却是想保劫夫的。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    “红造团”是激进造反派，势力不小，而“井冈山”虽是拥军派（拥护军区），但估计力量斗不过“红造团”。于是，他们为了不让“红造团”劫走李劫夫院长，
   </span>
   1967
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年元月
   </span>
   15
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    日，便派专人将劫夫送到北京，藏了起来，同时，向中央文革与周恩来总理送信反映。
   </span>
</font>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    结果，元月
   </span>
   31
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    日，中央文革的戚本禹真的派他的秘书刘汉，来李劫夫的住处了解，并将李劫夫转移安排住到了北京航空学院。因当时，北航已由“北航红旗”造反派红卫兵掌了权，没有中央文革的话，谁也不可能从北航将劫夫弄走的。
   </span>
</font>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">
   2
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    月
   </span>
   9
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    日下午，戚本禹出面接见了劫夫与沈阳音乐学院各派红卫兵代表，并公开表示，中央文革江青、康生与他戚本禹，都是同意“保”劫夫的，因为，他们认为，劫夫所作的歌曲《我们走在大路上》等，他们都熟悉，也都认为是符合毛泽东文艺路线的。
   </span>
</font>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    这一来，劫夫总算脱了一次难，回沈阳安然无恙了。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    当然，这次保劫夫，虽是戚本禹出的面，还打了中央文革的旗号，但实际是当时的中央高层都知道的事，而并非只中央文革的人同意。中共“九大”后进入了中央政治局的沈阳军区司令陈锡联，对此事就作过证：“这件事中央是清楚的。”所以，当戚本禹不久就倒台时，劫夫便没有因这一接见而受到整肃。
   </font>
</span>
</p>
<p class="MsoNormal">

<font size="3">

<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

   </span>

minor-fareast">
<font size="3">

</span>

<p class="MsoNormal">
<o:p>

<p class="MsoNormal">

minor-fareast">
<font size="3">

   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

   </span>
   1943
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

</font>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">
   1962
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年，有次李劫夫去广州开会，遇到了多年没见的老上级黄永胜（时任广州军区司令）。当黄永胜的夫人项辉芳得知劫夫是沈阳音乐学院的院长时，便让劫夫将黄永胜的三儿子黄春跃，招到沈阳音乐学院去学音乐。结果，
   </span>
   12
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    岁的黄春跃到“沈音”后，劫夫让他住到了自己家中，以从生活上能照顾这个小孩子。后来，项辉芳还让黄春跃做了李劫夫的干儿子。
   </span>
</font>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
<font size="3">

   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    文革开始后，黄春跃离开了学校，当兵去了广州部队海上文化工作队。
   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

   </span>
   1967
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    年的北京期间，便同当时也常住北京的黄永胜一家，有了较多的往来。劫夫的夫人张洛与项辉芳的往来，则更是越来越密切，二人甚至以姐妹相称。
   </span>
</font>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年
   </span>
   3
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

<o:p>

</o:p>
</p>
<p class="MsoNormal">

minor-fareast">
<font size="3">

   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">

</span>

<p class="MsoNormal">
<o:p>

</font>
</o:p>
</p>
<p class="MsoNormal">
<font size="3">

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">

   </span>
</font>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
<font size="3">

<o:p>

</p>

<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
<font size="3">

   </font>
</span>
</p>
<p class="MsoNormal">
<o:p>
<font size="3">
</font>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
<font size="3">
    虽说事由“做媒”起，但当时受到“林副统帅”接见，毕竟是大事。回家后，劫夫与张洛夫妇，便连忙给林彪写了一封感谢信，在信中，表示要“永远忠于毛主席，永远忠于林副主席”等等。

</span>

<o:p>

</o:p>

minor-fareast">

<font size="3">
