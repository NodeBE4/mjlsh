   年我参加“出国培训”考试时很爽快地给我签章放行的某县委组织部长。我向
  </span>
  W
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   书记和
  </span>
  L
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   部长写了一封信，诉说我自评先进教师和机构改革民意测验以来受到
  </span>
  G
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   校长的打压的情况。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1984
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年
  </span>
  4
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   月
  </span>
  9
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   日，州人事安排领导小组派人来我校调查，教职工群众的反映跟机构改革民意测验时是差不多的。他们还特地跟我进行了交谈，我诉说了在评先进教师和机构改革民意测验中谬承教职工群众的热情举荐，却屡屡遭到
  </span>
  G
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   校长的打压抹黑，竟成了罪人似的……
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1984
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年
  </span>
  5
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   月下旬，我校将调来一位新校长的消息已传开，而且，据说这位新校长正是某县农机科的
  </span>
  B
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   副科长！我们是相识的，当时我们都是技术员，同病相怜，还很谈得来，关系不错。他是七十年代末兵团合并到地方时从兵团某团场调到县上的农机技术员，党员；
  </span>
  1980
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   年提为农机科副科长；
  </span>
  1981
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   年在全州评定农机工程师技术职称时，因为他大专毕业的
  </span>
  XX
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   农学院已无从查考，还很费了一翻周折。他曾亲自对我讲过，他家在农村，十岁才上学，初中毕业考入一所农业学校，
  </span>
  1958
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   年大跃进也套了个农学院，算大专毕业的。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   很巧，在出席全州首次科学技术大会吃午饭时，恰好跟李主任同桌，我问起我校将调来新校长时，他竟躲躲闪闪支吾其词：一会儿说我可以找
  </span>
  L
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   部长，一会儿又说我如果要调离州技校他可以帮着说话。－－态度大变，令人费解。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   （又过了些年月，我才知道，当年新来的
  </span>
  B
  <span lang="ZH-CN" style='font-family:
宋体;mso-ascii-font-family:"Times New Roman"'>
   校长竟是李主任的近老乡和远亲。）
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1984
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年
  </span>
  5
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   月底，州党委组织部
  </span>
  L
  <span lang="ZH-CN" style='font-family:宋体;
mso-ascii-font-family:"Times New Roman"'>
   部长亲自找我谈话。他还像过去那样穿一身深灰色的制服，脸方，肤黑，浓眉细眼，面带微笑。他态度和蔼地说，你的信我看到了。你在
  </span>
  W
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   县工作这个阶段我了解情况，有人说你文革中如何如何，我可以负责任地说，你是属于受冲击的，已作过结论，没有尾巴，更谈不上什么“三种人”
  </span>
  [3]
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   。至于你来州上以后的情况我就不太了解了，可是据说有些反映。……你可以归于愿意搞好业务工作的知识分子。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   我接着郑重声明：那些所谓“有些反映”，不过是我对学校存在的一些不正之风提出批评之后遭到的报复。全体教师选举我为出席自治区先进教师代表大会的代表、全校大多数教职工在机构改革的民意测验中推举我，－－这些情况上级应当了解吧－－难道不比“有些反映”更能说明我这几年的作为吗？一场民意测验下来，我反而成了罪人－－被抹黑、被中伤、被歧视！
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   上级应当就民意测验的问题对州技校的教职工有一个说法。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  L
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   部长说，我可以把你的意见反映一下。这责任在州人事安排领导小组，不在他本人。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   （后来才知道，
  </span>
  L
  <span lang="ZH-CN" style='font-family:宋体;
mso-ascii-font-family:"Times New Roman"'>
   部长和新来的
  </span>
  B
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   校长两家关系非同一般，不久就结为亲家了。）
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  L
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   部长的谈话算是给这场严肃的民意测验画的句号吗？倘若我不给州党委写信反映
  </span>
  G
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   校长对我的打压抹黑，
  </span>
  L
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   部长会接见我吗？这场严肃的民意测验就这样虎头蛇尾不了了之了吗？为什么不给热情参加民意测验的群众一个说法？
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  G
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   校长对我的所谓“有些反映”，为什么不敢公开呢？
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   这场严肃的民意测验的结果，它的板子本应打在群众认为不称职的
  </span>
  G
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   校长身上，结果却打在了群众举荐的我的身上，也给热情的群众泼了凉水，公平吗？
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   许多年后我反思这件事：你一个大学生工程师，在一所技校受到大多数教职工们的举荐又能怎样？那些著名学者又有几个被选为名牌大学校长的？
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  8
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   、又空降了一位校长
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1984
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年
  </span>
  6
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   月
  </span>
  12
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   日，州劳动人事局的
  </span>
  S
  <span lang="ZH-CN" style='font-family:宋体;
mso-ascii-font-family:"Times New Roman"'>
   局长、
  </span>
  E
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   副局长来校宣布新校长的任命。
  </span>
  G
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   校长主持会议。当
  </span>
  S
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   局长宣读了
  </span>
  B
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   副校长的任命书之后，并宣布今后州技校由
  </span>
  B
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   副校长主持工作。
  </span>
  G
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   校长随即宣布散会。竟未给
  </span>
  B
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   副校长等表态留时间。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   于是，州技校又空降了一位新校长。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   之前不久，
  </span>
  G
  <span lang="ZH-CN" style='font-family:宋体;
mso-ascii-font-family:"Times New Roman"'>
   校长已被免去副校长，任命为副县级调研员；阿校长则仍调回州党校任副校长。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  G
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   校长退居二线、当了“调研员”之后，就不再来上班了。他办公室门口挂着的“校长室”牌子换做“调研员室”之后，门上上了锁就再没打开过。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  G
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   校长并不甘寂寞，他也凑热闹在街上摆了个地摊，卖袜子鞋垫针头线脑之类的小商品。后来竟然摆到了市党委、市政府大门口。他穿一身旧干部服，戴一顶旧解放帽，颠着大腹便便的肚子，认认真真地跟顾客讨价还价，没有一点官架子了。这时候他话也多了，不像当校长时常是寡言少语的。不知为何，当时社会上流行的一些顺口溜竟然常常先出自他的口：
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “现在是知识分子坐大堂，工农干部站两旁。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “这次机构改革年龄是个宝，文凭少不了，水平作参考，关系最重要。”
  </span>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “宁要四人帮，不要胡耀邦。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “胡耀邦胡说，胡乔木胡写－－二胡独奏。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “八十岁的老人召集一群六七十岁的老人开会，叫我们这些五十来岁的人退下来。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   ……
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   据说，市党委的人实在看不过，他这样一位副县级的老干部竟然摆地摊摆到了市党委的门口－－成何体统！市党委派人找他谈了话，他又心脏病复发差点丧命，在家人的一再劝阻下，他才不再上街摆地摊了。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   后来，
  </span>
  G
  <span lang="ZH-CN" style='font-family:宋体;
mso-ascii-font-family:"Times New Roman"'>
   校长又开始放羊，每天清早赶着一群羊到河的南面戈壁滩上去放牧，天黑才赶着羊群回来。他还是穿一身旧干部服，头戴一顶灰色的圆形布凉帽，挎一只小布袋，带着水和干粮，提着鞭子，吆喝着羊群。他常常笑着对人说：“我小时候放羊，老了又放羊。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  G
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   校长放了几年羊，先是十多只，后来到了四十多只。他原来胖得有些臃肿的身体变得结实了；原来有的心脏病、“三高”病也没有了，精神也开朗了。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">

"Times New Roman"'>
   （根据本人的日记如实整理。）
  </span>
