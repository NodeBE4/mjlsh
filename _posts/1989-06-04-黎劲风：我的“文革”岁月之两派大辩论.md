 <p class="MsoNormal">
  <b>
   <span lang="ZH-CN" style="font-family: 宋体;">
    <font size="5">
     我的“文革”岁月之两派大辩论
    </font>
   </span>
   <font size="4">
    <o:p>
    </o:p>
   </font>
  </b>
 </p>
 <p class="MsoNormal">
  <b>
   <font size="4">
    <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
     －－作者：黎劲风
    </span>
    <o:p>
    </o:p>
   </font>
  </b>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “文革”时期的大辩论，通常是两派之间的大辩论，是
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   “文革”初期一种很普遍的社会现象，是各阶层民众的亲身经历。作为参与者，已过去将近半世纪的大辩论经历依然留在我的记忆深处。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “文革”时期，大辩论与大鸣、大放、大字报，被列为“四大”，被曲解为“大民主”。“四大”排列顺序是大鸣、大放、大辩论、大字报。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   大辩论的内容千奇百异，在各个时期、各个地方也不尽相同。在
  </span>
  850
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   平方公里、几十万人口的广东吴川县
  </span>
  (
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   今吴川市
  </span>
  )
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   ，“好得很”与“糟得很”之辩却贯穿于两派大辩论的始终。生长在吴川的四零后、五零后，绝大多数对
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “好得很”与“糟得很”
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   之辩印象深刻并记忆犹新。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “糟得很”与“好得很”之说，曾出自毛泽东著作《湖南农民运动考察报告》。在论及湖南农民运动的“打土豪分田地”时，毛泽东写道：“这是四十年乃至几千年未曾成就过的奇勋。这是好得很。完全没有什么‘糟
  </span>
  '
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   ，完全不是什么‘糟得很’。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “文革”初起的
  </span>
  1966
  <span lang="ZH-CN" style='font-family:宋体;
mso-ascii-font-family:"Times New Roman"'>
   年
  </span>
  8
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   月
  </span>
  23

  </o:p>

 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>

  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>

  </span>
  1967
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   年
  </span>
  1
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>

  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>

  </span>
  <o:p>
  </o:p>
 </p>
