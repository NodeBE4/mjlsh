   有官不做，学霸扮学渣
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生，
  </span>
  1885
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   年出生在一个书香世家。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   曾祖父黄凤楼是道光年间进士，做过知县、同知。三个叔伯父均为举人，父亲黄儒藻也是秀才出身。母亲姚氏，也颇通诗书。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   因良好的家庭熏陶，黄远生自幼便广涉经史子集，父母还特地聘请了一位外籍女教师教他学习英语。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   十八岁，黄远生参加县试，考中秀才；同年秋中举人，为江西省第七名。第二年，参加了清朝组织的最后一次会试，高中进士。当时还不满
  </span>
  20
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   岁，是年纪最小的进士。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   短短两年，连中秀才、举人、进士三榜。在当时，黄远生是一个名副其实的学霸。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   按照惯例，会试之后，还有一次由皇帝亲自出题的殿试。表现好的，可以入翰林院编修。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生却不愿在翰林院任职，就学霸扮学渣，故意“考砸”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   尽管这样，因他此前已中进士，获得了“知县即用”的资格。黄远生还是拒绝了河南知县的官职，东渡日本，入中央大学攻读法律。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  6
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年后，学成回国。正值清廷大倡新政、推行法制，满怀抱负的黄远生感觉看到了希望，便接受任命，历任清政府邮传部员外郎、参议厅行走、编译局纂修、法政讲习所讲员等职。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   进士记者，报界之奇才
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   辛亥革命爆发之后，对清廷大失所望的黄远生退出官场，创办了《少年中国周刊》，正式开始了记者生涯，成为中国历史上第一位进士记者。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生被称为“报界之奇才”，是因为，他在短短几年内，迸发出卓越的新闻天赋，产生了极大的影响力。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   除了《少年中国周刊》外，黄远生还主编过梁启超创办的《庸言》月刊，担任过上海《申报》、《时报》、《东方日报》的特约记者和北京《亚细亚报》的撰述，同时还经常为《东方杂志》、《论衡》和《国民公报》等报刊撰稿。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="30.jpg" border="0" height="548" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3540/30.jpg" width="400"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在从业的四五年里，黄远生写下了大量的新闻通讯和时评政论。《远生遗著》里收录了
  </span>
  200
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   多篇文章，几乎全部直接或间接与时事有关。其工作量之大、创造力之盛，令人惊讶。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   时人赞誉黄远生的文章“洞朗轩辟，幽隐毕达”。他的文章犀利泼辣、观点鲜明、见解独到，能从常事看到常人所看不到的深度，影响力极大。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   独立、客观、公正，中国第一个现代意义上的记者
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生的头衔很多，“报界之奇才”、“新闻界出色人物”、“报界之骄龙”、“报业三杰”之一……
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   真正有分量的，还是“中国第一个真正现代意义上的记者”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   什么是现代意义上的记者呢？
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   独立、客观、公正。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生所处的时代，报馆或为大总统、或为政府、或为官僚、或为党会、或为豪强雄杰奸商私人占有。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生曾说：“人当独立自尊……而余自为学生，以迄今日，对此四字，乃无丝毫做到，宁不愧死。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="31.jpg" border="0" height="479" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3540/31.jpg" width="590"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生遗墨
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   他正是将“独立自尊”这一人生原则贯彻到职业生涯中，挺直了一个自由主义记者的脊梁与风骨。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生还主张，报人应与党派划开界限，成为“超然不党之人”，“无所偏倚”；新闻应严格尊重事实，客观公正，避免舆论成为私利的工具。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   独立、客观、公正，黄远生以超越时代的远见卓识与亲身实践，为黯淡沉闷的民国报业吹进了一股现代之风。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   此外，黄远生身上的现代性，还在于对报纸新闻性的坚守与创新。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在黄远生之前，王韬、梁启超、章太炎等人，也以办报著称，但其身份都是政论家，所写文章也以政论为主，新闻反而居于次要位置。报纸不能成为真正的新闻纸，报人亦不能称之为真正的记者。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   而黄远生，一开先河，以新闻采访和写作驰名于世。他开创了通讯这一新闻文体，还开设一个自媒体品牌，叫“远生通讯”，堪称民国初年新闻界的第一品牌。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   “远生通讯”对诸如宋教仁被刺、袁内阁两次倒台、二十一条、唐绍仪被迫下野等重大问题都进行了及时而深入的报道。他总能抓住读者的心，将真相大白于天下。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="32.jpg" border="0" height="500" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3540/32.jpg" width="500"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   启蒙世人，新文化运动的先驱者
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生不仅对新闻界和舆论界有着巨大的影响，还被誉为新文化运动的先驱者。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生所处的时代，正值辛亥革命前后，是中国文化觉醒前最黑暗的时刻，旧的欲去未去，新的将生未生。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="33.jpg" border="0" height="225" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3540/33.jpg" width="450"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生所处的时代
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在这古与今、旧与新、中与西转折的焦点时刻，黄远生以其奔放的激情与深沉的忧患意识，填补了时代的空白。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在《忏悔录》一文中，黄远生写道，“今日无论何等方面，自以改革为第一要义”，要改革国家，则必须改造社会，而欲改造社会，最终必须“改造个人”。这成为后来“五四”新文化运动提倡“民主与科学”的先声。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   新文化运动的旗手胡适在《五十年来之文学》一书中，将黄远生称作新文学的“先声”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   五四运动的鼓吹者蓝公武更明确指出，《新青年》所提出的“文学革命”和“思想革命”正是黄远生未竟的事业。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="34.jpg" border="0" height="545" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3540/34.jpg" width="380"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   《新青年》书影
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   实际上，五四时期的新文化先驱者几乎都受到黄远生思想的启发与影响。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生死后，其影响仍遍及整个思想文化领域。仅就当时思想界影响最大的《新青年》和《新潮》而言，提到他名字或涉及他言论的文章就有
  </span>
  30
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   多篇。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   骂袁世凯，骂革命党，骂同行，真正的无冕之王
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生曾说，新闻“第一义在大胆，第二义在诚实不欺”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   上至国家总统，下至各级官僚，黄远生都敢指斥揭露，批评袁世凯，批评革命党，批评同行……可谓“黄大胆”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生批评最多的，便是袁世凯。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="35.jpg" border="0" height="461" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3540/35.jpg" width="450"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   袁世凯
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   他曾在文章中一针见血地指出，“《临时约法》对袁世凯来说完全就是一纸空文，毫无约束力。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   对于当时国家出现的各种病状，黄远生也认为皆由袁世凯而起，“政局之日趋于险恶者，非他人为之，乃袁总统自为之也。吾国运命可以二言定之：盖瓦解于前清，而鱼烂于袁总统而已。”袁世凯“终将为亡国之罪魁。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   对袁世凯称帝的企图，黄远生也作过警示，在
  </span>
  1912
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年
  </span>
  6
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   月的《政界小风潮零记》，就分析了“恐怕大总统要做皇帝”的动向。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   除了袁世凯，黄远生对孙中山等革命党人的活动也冷嘲热讽，揭露和讥讽
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   “孙大炮”出卖国家利益、借助日本财阀的资金图谋执政等事。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   他对辛亥革命后的政坛和舆论环境也给予了批评。黄远生把辛亥革命后的政坛斥之为“循私弄权，无异于前清”，新闻自由亦“不及前清远甚。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   对于甘当袁世凯“家奴或走狗”的同行，黄远生自曝家丑地进行了批判，认为要改变这种情况，就必须“屏绝因缘，脱离偏依”，对最大权势袁世凯的态度应是“以公明之正义督责之”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   黄远生极力揭露和抨击各种内幕和丑恶，锋芒之尖锐，时人不及，义愤之高涨，前所未见。但正是“以故名益盛，而仇者忌者日益滋”，为后来的命运埋下了伏笔。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   中国之大，容不下一支自由的笔
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   袁世凯称帝前夕，极力收买社会各界知名人士，为之造势。名震海内外、拥有强大舆论号召力的黄远生，便进入了他的视线。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="36.jpg" border="0" height="301" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3540/36.jpg" width="590"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   袁世凯称帝
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   袁世凯强邀黄远生担任御用的上海《亚细亚报》总撰述，黄远生心虽不愿，却也不敢一口回拒。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   随后，袁世凯又对黄远生表示：如能撰写赞成帝制的文章，则将酬谢十万元，并奉送一个部长的席位。袁大总统出手真是大方，可谓一字千金。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   为袁世凯称帝背书，这个就突破黄远生的底线了。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   想拒绝，又经不起袁氏的再三催逼，便写了一篇“似是而非”的文章敷衍了事。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   袁世凯“龙颜”大怒，不满意他的搪塞，又派人逼他重写颂文。黄远生这才如梦方醒，深知再也不能与魔鬼周旋下去。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1915
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年
  </span>
  9
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   月
  </span>
  3
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   日，他逃离北京，南下上海。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在途中，他就在上海各大报刊刊出了《黄远生反对帝制并辞去袁系报纸聘约启事》，明确表示反对帝制，与袁世凯决裂。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   抵沪后，又连续
  </span>
  9
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   天在《申报》和《时事新报》上刊登启事，宣布自己与曾担任的《申报》驻京通讯员及上海《亚细亚报》的撰述，一概脱离关系。但《亚细亚报》在出版广告中仍把他的名字列为总撰述，他只得又连续
  </span>
  7
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   天报纸上声明与《亚细亚报》划清界限。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="37.jpg" border="0" height="411" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3540/37.jpg" width="500"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   《亚细亚日报》
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   为了彻底躲避袁党的追逼，
  </span>
  1915
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   年
  </span>
  10
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   月，黄远生乘坐轮船离开上海，来到了美国旧金山。
  </span>
  <span lang="ZH-CN">
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

   名记之死，谋杀还是误杀
  </span>
  <o:p>

 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">

 <p class="MsoNormal">
  1915
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

   年的旧金山唐人街
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

   黄远生一到旧金山，当地的报纸便报道中国名记者
  </span>
  Yuan
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;

  </span>
  Yung
Huang
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;
mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  Yuan
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

   －
  </span>
  Yung Huang
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:

 <p class="MsoNormal">

 </p>
 <p class="MsoNormal">

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

   年
  </span>

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

 <p class="MsoNormal">

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

  </span>

 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
