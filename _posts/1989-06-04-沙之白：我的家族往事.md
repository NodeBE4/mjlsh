 <p class="MsoNormal">
  <b>
   <span lang="ZH-CN" style="font-family: 宋体;">
    <font size="5">
     我的家族往事
    </font>
   </span>
   <font size="4">
    <o:p>
    </o:p>
   </font>
  </b>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   <b>
    <font size="4">
     －－作者：沙之白
    </font>
   </b>
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
  <span style="font-family: 宋体;">
   一转眼，父亲已经离世七年，母亲也走了三年多了。他们一走，我再也不能像过去那样，向他们打听那些老年间的真实经历；哪怕一个看起来很简单的实例，也由于缺少了目击者或当事人的证实而显得苍白无力。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   近几年，我正在撰写关于家族的历史。常常是，写着写着，想就某一件事要向父母核对一下的时候，张开嘴才醒过来，他们已经离我远去。这时候，那种撕心裂肺的感觉，那种再也没有依靠的感觉，就格外强烈。心里空落落无所凭借的滋味，实在是笔墨难以形容。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   这种感觉，在祖父去世的时候就有，但表达不出来。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   因为，那时候我还没跨进
  </span>
  13
  <span lang="ZH-CN" style="font-family: 宋体;">
   岁的门槛。按照旧例，过了阴历年的除夕才能涨一岁。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   那时，是
  </span>
  1961
  <span lang="ZH-CN" style="font-family: 宋体;">
   年初。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   那天，是农历庚子年的腊月初十。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   祖父初七逝世，我和父亲初九把他送出城去。那时尚未普及火葬，我们护送他老人家的棺木到沈阳市郊区靠近浑河岸边的一处公墓。硬硬的雪粒儿和冰凉的雪花裹在一起，在辽沈大地上肆虐，我坐在“大解放”汽车的驾驶楼里，缩着脖子仍感觉很冷。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">

  </span>
  <span style="font-family: 宋体;">

  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">

  </span>
  ?
  <span lang="ZH-CN" style="font-family: 宋体;">

  </span>

  <span lang="ZH-CN" style="font-family: 宋体;">

  <span lang="ZH-CN" style="font-family: 宋体;">

  </span>
 </p>
 <p class="MsoNormal">

  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

  </span>
  <span style="font-family: 宋体;">

  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   祖父属羊，
  </span>
  1883
  <span lang="ZH-CN" style="font-family: 宋体;">

  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">

  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   我作为一个后辈，不恭敬地说，那也该算是一种愚昧吧？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

  </span>

 <p class="MsoNormal">

  </span>

  </span>
  <span style="font-family: 宋体;">

 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   从此以后，父亲心灰意冷，滋生了离开这个学校的念头。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  1952
  <span lang="ZH-CN" style="font-family: 宋体;">
   年秋天，
  </span>
  25
  <span lang="ZH-CN" style="font-family: 宋体;">
   岁的父亲带着全家老小：祖父、母亲、我以及刚刚出生不久的大妹，离开了生他养他的小山村，到一个未知的世界去闯荡。那也就是我离开老家的日子。直到六十多年后，我才重新回到那个让我几十年来梦牵魂绕的地方，我的祖居地。我家的祖坟就在那个村子的南山坡上，我的胞衣就埋藏在那个村子里某一间房子的门槛下。为什么相隔了六十多年才回到我的出生之地
  </span>
  ?
  <span lang="ZH-CN" style="font-family: 宋体;">
   跪在祖坟的前边，面对着那些见过面的、没见过面的前辈的坟墓，我热泪如涌。人世间的波谲云诡，红尘里的恩恩怨怨，都在那一瞬间涌上心头。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   我们家搬到了沈阳市，在郊区的艳粉屯住下，租住一个外号叫“李大胖子”的房子。好在五十年代的工作相对好找，父亲在化工学校
  </span>
  (
  <span lang="ZH-CN" style="font-family: 宋体;">
   ？
  </span>
  )
  <span lang="ZH-CN" style="font-family: 宋体;">
   找到一个烧锅炉的工作，后来单位知道他识字就让他去夜校当老师，再后来又去市废旧物公司做营业员。一开始，工资微薄且不稳定，生活拮据，时常被房东撵着要房租。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   一直到
  </span>
  1956
  <span lang="ZH-CN" style="font-family: 宋体;">
   年，我们搬到沈河区的惠工街，才算结束了租私房的历史。父亲调到了新单位，工作部门好像是在工会？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   如果，就这样发展下去，也还不错。一个老老实实的国企职员，退休时挣点退休金，也还不错。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   可是，还是那句话，现实不会那么发展的，它总会有一些波折。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   我小学三年级的时候，又搬家了。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   从沈河区搬到和平区的民族电影院对过
  </span>
  (
  <span lang="ZH-CN" style="font-family: 宋体;">
   如今，那里是市检察院的高楼大厦
  </span>
  )
  <span lang="ZH-CN" style="font-family: 宋体;">
   。那里有废旧物公司的一座日式小楼。就是在这一段时间里，父亲许诺我，要送我到鲁迅美术学院去念书。可他不过一个平头百姓，哪里会知道世事的变化？别说送我去念书，就连他自己，他又何尝知道会有一些什么样的遭遇？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   祖父已经七十多岁，可他闲不住，总要出去赚点零花钱。在艳粉屯住的时候他卖糖葫芦，有时我们吃他的糖葫芦吃多了，他就有点儿心疼，说：“我好好好，妈了个巴子的，把爷爷的本钱都吃没了。”后来被管理市场的人抓住，好说歹说，父亲去把他要出来。他又今天卖点耗子药，明天倒腾点“六六粉”，在北市场的小胡同里转悠。我也用书包装过几袋“六六粉”偷偷摸摸地去卖，被人抓住好一顿训斥，再也不敢去了。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   三四年级的时候，正是挨饿的时候，也就是所谓“三年自然灾害”的时候。那时的“盲流”遍地，到处都是乞讨的人。得浮肿病的人也特多。在北市场南口，我亲眼见过，一个“乞丐”把唾沫吐在人家刚买的煎饼上，遭到毒打，满脸是血，倒在地上，还在大口吃着已经沾上了血迹、唾沫和泥土的煎饼。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   供应的口粮根本不够吃，记得好像是粮本规定的数量按比例缩减，所以几乎是家家户户都在想办法搞吃的。父亲单位有个农场，他去农场弄些做苹果罐头剩余的边角料，苹果皮、苹果核，聊以补充饥肠。母亲这时候在街道收废品，天天游走在街上，劳动量很大，她自己就不够吃，可她还要节省出来给我们吃。一顿饭“二两”烤糕两个，可那根本不够；而且，“二两”还不够数。这时候家里已经七口人，都在张嘴等吃的！
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   一次间操时间，母亲路过我们学校
  </span>
  (
  <span lang="ZH-CN" style="font-family: 宋体;">
   我这时已经转学到“南京街第九小学”
  </span>
  )
  <span lang="ZH-CN" style="font-family: 宋体;">
   。她招呼我和妹妹过去，隔着栅栏把两个烤糕分给我和妹妹。天啊，我那时候怎么就不知道心疼母亲？怎么就不知道她也在挨饿？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   可怜天下父母心。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   有时候，祖父领着我们兄妹几个推着载着三弟的婴儿车，去公共汽车车站等候母亲下班归来。那时候也真怪，会议特别多。动不动就是开会，一开就开到很晚。母亲后来回忆，看到我们在车站等她，像个小鸟似的等吃的。可她哪有什么吃的呢
  </span>
  ?
  <span lang="ZH-CN" style="font-family: 宋体;">
   在公共汽车上她看见我们，急忙扭过头去，暗自垂泪。但面对我们，她还要装成一个没事人似的。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   祖父熬不过去了，终于在
  </span>
  1961
  <span lang="ZH-CN" style="font-family: 宋体;">
   年
  </span>
  1
  <span lang="ZH-CN" style="font-family: 宋体;">
   月
  </span>
  23
  <span lang="ZH-CN" style="font-family: 宋体;">
   日去世，享年
  </span>
  78
  <span lang="ZH-CN" style="font-family: 宋体;">
   岁。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   用那时候的话说，那年头“
  </span>
  (
  <span lang="ZH-CN" style="font-family: 宋体;">
   阴间
  </span>
  )
  <span lang="ZH-CN" style="font-family: 宋体;">
   收老人”。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   说祖父是饿死，恐怕有点过分，主要还是营养不良吧！
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   祖父去世不久，外祖父来到沈阳。他说辽南乡下的日子也不好过，也挨饿，他得出去想办法弄点粮食。外祖父的家，那几年也有大变故：大舅原在某重点学校当校长，
  </span>
  1957
  <span lang="ZH-CN" style="font-family: 宋体;">
   年成为“右派”，开除公职，遣送回农村原籍；二舅有历史问题，成为“历史反革命”，同年被捕入狱；三舅原为老师，也成了“右派”。看到这些，外祖父心如刀割，如果再不能为孙男娣女们弄点粮食，他们怎么活
  </span>
  ?
  <span lang="ZH-CN" style="font-family: 宋体;">
   听说“边外”
  </span>
  (
  <span lang="ZH-CN" style="font-family: 宋体;">
   老一辈人把内蒙古等地叫“边外”
  </span>
  )
  <span lang="ZH-CN" style="font-family: 宋体;">
   乡下生活好混，能有一口吃的，他决定去碰碰运气。我的父母不能给他老人家什么帮助，也只好任他去了。日子不长，“边外”亲戚发来电报，说外祖父已经去世。那年，他才
  </span>
  67
  <span lang="ZH-CN" style="font-family: 宋体;">
   岁。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   祖父与外祖父亲家俩，都是属羊的，祖父比外祖父正好大一轮。民间传说，属羊的命苦。真的么？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   这两个老人在我的印象中，极为深刻。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   祖父是直性子，嗓门儿大，如果他会唱歌，一定是个男高音歌手。他那么和善，慈爱，会打骂欺压穷人么？我听说的，土改之前那会儿，人家都愿意到我们家干活，原因就是吃得好工钱多。这与书本上的大地主大坏蛋好像不大一致呢！
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   外祖父呢，慢性子，说话柔声细气，好像从未高声过。我很小的时候他就教我认字。他抱着我，走在县城的大街上，手指烧饼铺幌子上的两个大字，一字一顿地让我念：“火，勺，火，勺……”－－我老家的土话，把烧饼叫作“火勺”。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   那一年，果然是“收老人”的年头？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   两个老人，同年去世。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   说一句晚辈不该说的话，幸亏外祖父逝世于
  </span>
  1961
  <span lang="ZH-CN" style="font-family: 宋体;">
   年。假如晚几年赶上“文革”，遇到红卫兵们烧古旧书的时代，他又该多么伤心
  </span>
  ?
  <span lang="ZH-CN" style="font-family: 宋体;">
   这种情况，恐怕比看他的孙子们挨饿还要痛苦。近几年我去表姐家串门，不免谈起姥爷，她们说到“破四旧”的时候，家里的好多线装书都被拿出来烧掉，一些古旧字画也都在被焚之列……
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   心疼之余，也想到，彼时彼地，此情此景，一个旧式老读书人的外祖父该如何面对？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   真的不敢多想。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   接到外祖父离世的消息，父亲急匆匆向单位告假，去料理外祖父的后事。在内蒙古的短短几天，他的感觉竟然是，这里的农村比沈阳好，起码，粮食够吃。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   从内蒙古回来之后，他就积极向单位申请离职。正好赶上“城市人口精减”，单位动员还怕不起作用呢，当然批准。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  1961
  <span lang="ZH-CN" style="font-family: 宋体;">
   年秋，我们全家来到内蒙古农村。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   这一年，父亲
  </span>
  34
  <span lang="ZH-CN" style="font-family: 宋体;">
   岁，母亲
  </span>
  37
  <span lang="ZH-CN" style="font-family: 宋体;">
   岁，我
  </span>
  13
  <span lang="ZH-CN" style="font-family: 宋体;">
   岁，大妹
  </span>
  10
  <span lang="ZH-CN" style="font-family: 宋体;">
   岁，二弟
  </span>
  7
  <span lang="ZH-CN" style="font-family: 宋体;">
   岁，最小的三弟
  </span>
  5
  <span lang="ZH-CN" style="font-family: 宋体;">
   岁。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   此后几十年，父母在谈到来内蒙古的理由时，总是说“挨饿”。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   但我也有我的想法。挨饿确实是不能回避的事实，但也并非扛不过去
  </span>
  ;
  <span lang="ZH-CN" style="font-family: 宋体;">
   其实，在他的内心，应该还有更重要的理由。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   我听母亲说过另一件事：
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   父亲曾在财务上差了单位
  </span>
  80
  <span lang="ZH-CN" style="font-family: 宋体;">
   块钱，被说成是“贪污”，被迫用自己的工资顶上了这笔账。虽然还了钱，但贪污的名声还是挂在身上，一些人不免就用另一种眼光看他，让他不舒服，所以才心生离意。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   究竟是不是这样
  </span>
  ?
  <span lang="ZH-CN" style="font-family: 宋体;">
   人已离世，无从考证。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   而且，他在世的时候并不善于表达自己，他对自己的经历似乎也没有做过怎么认真地咀嚼。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   我是从他第一次离开家乡时候的心理，来推测他之所以做出迁居内蒙古这个决定时的思想根据。从这里，我觉得我发现了父亲的一个弱点，这就是不能承担。有一点压力，就受不了，就想躲开，就想逃避。我这样估计自己的父亲，也许不够准确。谁愿意顶着压力过活呢？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   顺境尚可过，逆境不能担。话说人生，又怎么可能只有顺境而没有逆境？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   全家搬到内蒙古，吃饭的问题好像得到了解决。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   但其他问题呢？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   他从青少年时期开始，就没做过苦力，没哈下腰真正地干点体力活。现在，现实却要他出大力流大汗了。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   不会割地，学；不会铲地，学；不会种地，当然还是学。种铲趟割藏，哪样不会都得学。他学说自己第一次割黄豆，豆荚上都是硬毛毛，豆荚的尖儿扎手，黄豆的秧棵又硬又矮，一刀下去，弄不好刀刃都崩了，豆棵却没割下来。一根垄割毕，腰酸腿疼，手指肚儿冒血筋儿。白色线手套换了好几副，那玩意是好看不中用。但也得干啊，不干没有工分，一家人吃什么呢？来内蒙古几十年，也当过极短时间的会计，但好景不长，启用他的队长被人说成是“阶级路线不清”，再也不敢使用他，他也就只好规规矩矩地干活儿。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   他再也不说要送儿子去鲁迅美术学院念书的事儿。他连自己也顾不得了。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   很快，他变成了庄稼里手，扬场簸簸箕，扶犁点种，编筐窝篓，养蚕漏粉，没有他不会的。他这样做，一方面固然是为了生活，另一方面也是为了摆脱那个“富农成分”的黑色印记。他拼命地干活，向“农业劳动者”的方向努力－－“四清”期间，针对家庭出身地富的子女，有一项改变成分的政策：经过审议，可以由“地富”成分改为“农业劳动者”。谁愿意被人歧视
  </span>
  ?
  <span lang="ZH-CN" style="font-family: 宋体;">
   谁愿意被人视为另类
  </span>
  ?
  <span lang="ZH-CN" style="font-family: 宋体;">
   为了摆脱另类的名声，父亲和那些与他同类的人，付出了多少代价？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   “四清”结束，他真的被评为“农业劳动者”了，后来又由旗人民政府颁发了改为“中农成分”的通知书。可是，当他欣喜地看着这张来之不易的通知书的时候，轰轰烈烈的“无产阶级文化大革命”开始了。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   一切都产生了新的变化，一切都不算数了，连“四清成果”也不算数了。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   不知父亲的心里，此时是否升起了一股失望与悲凉？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   艰苦的农业劳动累坏了他，再加上营养不良，他肝病复发。在沈阳工作的
  </span>
  50
  <span lang="ZH-CN" style="font-family: 宋体;">
   年代，他因患病在小河沿
  </span>
  (
  <span lang="ZH-CN" style="font-family: 宋体;">
   也许是南湖？记不清了
  </span>
  )
  <span lang="ZH-CN" style="font-family: 宋体;">
   的职工疗养院住了一段时间，病愈。现在，他再也不能去疗养了。他是个纯粹的农民，而农民是没有疗养院的。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   “文革”后紧接着的运动是，“三忠于四无限”，读“红宝书”，跳“忠字舞”，“挖肃”，“一打三反”，“三两斗争”，“儒法斗争”，“批林批孔”，“评水浒”，“反击右倾翻案风”……运动一场接一场，无尽无休。翻过来掉过去，反正就是不能让你稳稳当当地过日子。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   我在这期间浑浑噩噩地结婚生子，分家时是一无所有。当时我有点自嘲地想，我才是真正的无产阶级呢！房屋一间地无一垄，苦人儿两头，还有两只豁牙的饭碗，两双长短不齐的筷子。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   在各种运动的忽悠中，父亲失去正确的判断力了。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   他只知道“今朝有酒今朝醉”。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   在混沌的农民生涯中，他习惯了听命由天，更习惯了求神问佛，固执地相信算命瞎子提供给他的各种奇异的结论。他们也坚定地相信，如果眼睛有了“翳子”，那一定是祖宗牌位前边挂了什么不吉祥的东西；如果小孩子哭闹，烧几张用过的邮票“收收魂”也一定会止住哭闹；他们甚至还相信，在某个土庙门前跪下求一求，神就会降下“灵药”来免去他们的所有病痛。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   “文革”结束，梦靥的日子终于结束。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   外祖父家，大舅的“右派”被平反，恢复公职，并且成为人大代表
  </span>
  (
  <span lang="ZH-CN" style="font-family: 宋体;">
   驻会
  </span>
  )
  <span lang="ZH-CN" style="font-family: 宋体;">
   ；二舅的案子虽不能得知详情，但此时也已获得保外就医，回家与亲人团聚；三舅的右派帽子摘下了，也恢复了工作。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   我于
  </span>
  1977
  <span lang="ZH-CN" style="font-family: 宋体;">
   年有资格参加高考，在填写“家庭成分”一栏的时候，终于能够正儿八经地写上“农民”两个字。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  1985
  <span lang="ZH-CN" style="font-family: 宋体;">
   年，当我从乡下调到城市的时候，不免感慨万千：
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   父亲
  </span>
  30
  <span lang="ZH-CN" style="font-family: 宋体;">
   多岁的时候把全家从城里弄到乡下，我
  </span>
  30
  <span lang="ZH-CN" style="font-family: 宋体;">
   多岁的时候又费劲巴力地把全家从乡下弄到城里。老天爷究竟在开什么玩笑
  </span>
  ?
  <span lang="ZH-CN" style="font-family: 宋体;">
   它在捉弄我们吗？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   梦靥过去，但过去的岁月给父母心理造成的伤痕并不能跟着消失。甚至连我，我们兄弟姐妹的心里，也都有着磨灭不掉的伤痕。每当回家走上那个熟悉的小山包，就担心：山包后面的家不会有什么意外变故吧？明知道不会有什么事，但那种想法总会不由自主地冒出来。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   上个世纪的八十年代，父亲听说辽宁省有关于城市职工精简的新政策－－当然，也只是听说，偏僻农村的农民只能是听说而不可能看到文件，父亲曾几次去原单位，以求得落实政策。偏偏地，在多次往返的途中，那些能够说明他经历的退职证明等一系列文件，都丢失了。最后，单位也只能是出于同情，给了一点可怜的生活补助。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span lang="ZH-CN" style="font-family: 宋体;">
   父亲于
  </span>
  2008
  <span lang="ZH-CN" style="font-family: 宋体;">
   年去世，三年之后母亲去世。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   我捏着父亲的脉搏，感受着他的心跳，一点一点微弱，微弱，最后停止。他走了。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   “离休”、“老教员”这些称号，竟与父亲无缘。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   他们都静静地走了。带着生前未实现的愿望，也带着在生命最后日子里的满足，走了。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   遥望祖辈父辈的背影，我能说些什么？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   一晃之间，几十年的时间匆匆而过，再也不能回头。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   我不希望彼此争斗的族群分裂重新来过，我也不希望不能让老百姓稳稳当当过日子的岁月再次到来。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   我希望我的儿孙们记住，还是那话，任何时候都要凭自己的劳动吃饭，出大力流大汗，不自卑，不自傲，也就行了。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <span style="font-family: 宋体;">
   愚昧就愚昧吧，老实人常在。
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span style="font-family: 宋体;">
   莫非，这也是一种宿命？
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
