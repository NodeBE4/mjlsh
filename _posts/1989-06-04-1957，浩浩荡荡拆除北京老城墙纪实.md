    捐献出四十多万块旧城砖
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    原本，城墙和城楼都是由孔庆普他们这个专业队伍来拆的，从龙潭湖往南
   </span>
   200
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    米长的那段城墙开始，情况发生了变化。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   1956
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年，东城区房管局给北京市政府打了一个报告，申请用一些城墙砖盖房子。市政府就让建设局给他们“选地方”。最后确定了龙潭湖向南
   </span>
   200
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    米长的城墙，由东城区房管局负责拆除。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    让东城房管局大为失望的是，这部分外城的城墙已经破损得没有几块整砖了，于是拆了一半，就扔在那儿。其他区房管部门就有人过来继续拆剩下的墙砖，当然是没有任何报批手续的，后来被北京市房管局给制止了。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    晚上
   </font>
  </span>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    ，大批的市民也来了，还找来了不少当时农业生产合作社的农民。虽然北京市政府马上就下令禁止他们拆城墙，不过又同时做出了另外一个决定：城墙以后都归各区政府管，他们组织人拆，孔庆普他们只负责拆城楼和箭楼。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    到
   </span>
   1958
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年底，北京外城的城墙和城楼，基本上都拆完了。孔庆普说，真正将内城也全部拆完，是在
   </span>
   60
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年代末。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   1966
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年，中央计划在北京修环城地铁，最方便的做法就是利用内城城墙的地基来修。孔庆普他们马上接到了规划任务书，当时的计划是，除了西直门和安定门以外，内城的城楼和城墙全拆掉。周恩来又特别指示，正阳门和古观象台不能拆。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    但是，还没动工，“文革”就开始了。建设局的领导基本都成了“当权派”，孔庆普成了“臭老九”。结果西直门和安定门后来被红卫兵们三两下给拆了，内城的城墙也被老百姓基本拆完。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    崇文门后面那段现在已经成为“遗址公园”的断墙，东半截正好挨着北京火车站，铁路部门的家属宿舍，都是倚着这面墙盖的；西边的一截，当时是建设局材料库、电业局路灯队的办公室。地铁在北京站拐了弯，这段城墙不碍事，加上依墙而建的那些房子做了“掩护”，这段墙留了下来，隐藏在层层叠叠的房子后面。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    到了
   </span>
   1996
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年，当那些依墙而建的房子终于被推倒，这段城墙露了出来。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    很多过去居住在城墙边上的老居民，把家里收存的旧城砖献出来，用做“城墙遗址公园”的修缮，总共有
   </span>
   40
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    多万块。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    他们中的大多数人，不仅见证，而且亲身参与了当年绵延十多年的“拆除城墙行动”。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    一位捐赠者说：“就像是小时候不懂事，轻易撕毁了一张价值连城的‘龙票’，时间越长，越让人觉得懊恼。”
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    今天路过那段老城墙，你会看到，有人在草坪上休息，恋人在城墙脚下私语，成群的学生，围坐在城墙的影子里，唱着流行歌曲。
   </font>
  </span>
 </p>
 <p class="MsoNormal">
