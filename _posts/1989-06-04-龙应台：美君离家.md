minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    已经是兵荒马乱的时候，美君仓促上路，临别前对母亲也就是平常地说一句：“很快回来啦。”跨出家门，头都不曾回过一次，虽然知道那瘦弱的母亲，裹着小脚，就站在那老屋门边看着她走。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    美君也没有对淳安城多看两眼。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <img alt="139.jpg" border="0" height="333" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3785/139.jpg" width="500"/>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <br/>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    余年春老人手绘的淳安城，这就是那座美君以为很快就能回来的城。美君没能很快回来，在她离家后
   </span>
   6
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年，城也沉入水底。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    庭院深深的老宅，马蹄达达的石街，还有老宅后边那一弯清净见底的新安江水，对美君而言，都和月亮星星一样是永恒不变、理所当然的东西，时代再乱，你也没必要和月亮星星作别吧？人会死，家会散，朝代会覆灭，但是一个城，总不会消失吧？更何况这淳安城，已经有一千五百年的历史。美君向来不是个多愁善感的人，她聪明、果决、坚强。城里的人都知道，应家这个女儿厉害，十七岁就会独自押着一条船的货，从淳安沿水路送到杭州城里去做买卖。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    有一回，买卖做完，回程上，一个家族长辈装了满船的盐，从杭城运回淳安；半路上突然出现缉私队的士兵，拦下船准备检查。船上的人紧张得就想跳水，长辈脸色发青，美君才知道，这一船的盐，大部分是私盐。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <img alt="140.jpg" border="0" height="357" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3785/140.jpg" width="559"/>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <br/>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    据《淳安县志》等历史资料的记载，贺城和狮城两座历史悠久的浙西古城因
   </span>
   1955
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年建设新安江水电站，而悄然“沉入”了千岛湖底。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    她看长辈完全乱了方寸，揣度了一下形势，便作主指挥，说：“速度放慢。”
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <br/>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    她要工人立即把两袋合法的官盐拖到船板的最前端，然后要工人那年轻丰满的媳妇，坐到存放私盐的船舱入口的门坎上，脱掉外衣，只留身上的小胸兜。美君像导演一样告诉她坐在哪里，怎么坐，然后盯着她看看，又说：“把簪子拿掉，头发放下来。”
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    船缓缓停下，缉私船靠近来，抱着枪的士兵一跃而上。美君先请他们检查船板上的两袋官盐。士兵打开袋子，检查标签，抓一把盐在手心里闻闻看看，然后转身要进舱房，可是一转身，就看见那年轻的江南女子坐在船舱入口，好像正要穿衣服，她大半个牛奶色、光滑的背，是裸的，士兵登时吓了一跳，美君就说“对不起对不起，嫂子刚刚在给孩子喂奶。”
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    缉私队长忙不迭地说：“那就不要打扰了。你们快开船吧。”
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    淳安的长辈们在对我叙述这故事时，美君就坐在旁边咯咯地笑。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   </span>
  </font>
 </p>
 <img alt="141.jpg" border="0" height="401" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3785/141.jpg" width="550"/>
 <p class="MsoNormal">
  <font size="3">
   <o:p>
   </o:p>

 </p>
 <p class="MsoNormal">

   <br/>

 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    水下古城中节孝牌坊静静地立在水下，遥远地回应着一缕天光。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    最后一次离开淳安时，后来美君跟我说，她确实回头看了一眼那城门两边的石狮子，一边一只，已经在那里好多、好多朝代。她走的那一天，石狮子就蹲在那里，不让你有任何的怀疑或动摇，他们会在那里天长地久。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    淳安，是三国时吴国的大将贺齐所开垦设置，当时的淳安人被称为“山越”，在土地上刀耕火种，逐渐发展成吴国的文明小城，明朝著名的清官海瑞，在这里做县令，淳安人为他建了个“海公祠”，是美君小时候每天经过的地方。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    美君会描述她家里的家具：柏树做的八仙桌，有一种扑鼻的清香味；母亲的床，木头上全是雕花；天井里头的黑陶大水缸，一大缸一大缸养着高高挺挺的粉红色风荷。家的大堂正中挂着三代的祖宗画像，谁是谁她不知道，但是她很骄傲地说：“最下面那一排穿着清朝的官服，是高祖，他是同治年间乡试的武举，后来还是衢州府的留守呢，官很大的。”
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    <br/>
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    我问她：“留守”是什么官？她歪着头想想，说：“不知道，大概是，嗯，警察局长吧？”
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
