 <p class="MsoNormal">

  108
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   线花滩公社三大湾段发生了一桩惨案－－支援三线建设的北京车队驾驶员郭卓康碾死了正在路边锤石头的右派分子、反革命分子张学炜。张时年三十五岁。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   张学炜，是被六合公社革命委员会强制派往路边锤石头的。处理安葬张学炜及如何抚养其子女的会议，由六合公社革命委员会主任孙大富主持，县公安局李光平、人保组、军管会的邹正如、李才巨、北京交通部第一汽车总公司革命委员会副主任秦树杰、张崇海等官员参加了会议。雅安监理所沈华昆作证。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   会议一致认为：张学炜之被汽车撞死，完全驾驶员郭卓康的负责。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   张学炜之妻明元春、弟弟张学康要求用棺木安葬，但孙大富等官员认为张学炜是右派加反革命份子，他在三线建设中被撞死，而三线建设的意义又特别重大，是毛主席号召的，被撞死的张学炜是无产阶级专政的对象，没什么了不起。最后决定给一个短小的“火匣子”，将尸体硬塞入其中，命令几个“五类份子”（地、富、反、坏、右）抬去草草掩埋了事。
  </span>
  <o:p>
  </o:p>
 </p>

 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   大儿张仕清，三岁；
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   小儿张仕桃，尚在母腹中，几天后出生。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   在如何抚养这三个娃娃的问题上，北京车队的官员秦树杰、张崇海表态：“张学炜的三个孩子是新中国的儿童，在抚养的问题上应与其父张学炜的身份区别开来，死亡事故是车队驾驶员造成的，抚养责任应由车队负责，按国家有关规定抚养。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   六合公社革命委员会主任孙大富发言：“要体现人民公社一大二公的优越性，减轻国家负担，张学炜的子女由六合公社抚养。”
  </span>
  <span lang="ZH-CN">
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   最后，北京车队只得给了一笔款子了事。公社从中给了明元春
  </span>
  200
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   元作为医药费开支，并责成此款由生产队掌握控制，明元春领用。之后，在
  </span>
  1970
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   ——
  </span>
  1972
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   的三年中，六合公社革命委员会强行将张学炜之弟张学康在生产队应得的红利钱
  </span>
  400
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   元左右，作为张学炜的三个子女的口粮开支。实际上把抚养的责任转嫁给了张学康。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   张学炜之妻明元春拖着三个娃娃，严守妇道，不招不嫁，不移母志，在难以言说的政治、经济、人格等多方面的压力下痛苦挣扎，直到十一届三中全会之后，才得见天日。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   二．学炜其人
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   张学炜，六合乡水池上队人。
  </span>
  1948
  <span lang="ZH-CN" style='font-family:
宋体;mso-ascii-font-family:"Times New Roman"'>
   年毕业于荥经中学，
  </span>
  1951
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   年参加教育工作，执教于荥经县城关第二完全小学。他工作积极，热爱生活，喜欢文学创作，曾对人言“用笔尖改造世界，改造自我……”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1957
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年
  </span>
  12
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   月寒假中，荥经县委集中全县小学教师在“二完小”开展整风运动，由于成功地贯彻了“引蛇出洞”的方针，
  </span>
  30
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   多个右派份子相继落网。既已批深批透，批倒批臭，运动也接近尾声。然而，据说“二完小”没有完成右派指标，最后，只好将张学炜抛出凑数，并集中最后火力进行批斗。当时批判他的专栏称为“作家专栏”，以表示对他的奚落和挖苦。在批斗中，张学炜对强加给他的莫须有罪名据理反驳，惹怒了打手们。打手们要他跪地认罪，他坚决不跪。一个穿黄军服的人扯掉了他一绺头发，也没把他弄跪下。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   最后，整风工作组还是给张学炜戴上右派份子的帽子，但他始终不承认自己反党、反社会主义。他的一位同行情侣表示仍然爱着他，但他坚决以前途利害相劝，三年之后，那女子才断了念想，离他而去。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1958
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年春开学，只有三四个右派份子有资格回到工作岗位，在暗中监督下工作，多数集中起来劳动改造。七月，在县大礼堂召开大会，宣布对右派份子的处理结论。张学炜被开除共青团籍，连降三级工资，由
  </span>
  43.5
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   元降到
  </span>
  29.5
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   元，下放到荥经石滓乡大山深处的云板岩石棉矿监督劳动。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   在高山野岭之中，道路崎岖，行走艰难；气候异常，一日数变；张学炜举目无亲，寝室难安；前途无望，命运难卜！后因石棉矿停办，又被发配到县纸厂继续接受监督劳动。一直改造到被精简回家，从事农业生产。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   张学炜回到农村，其合法住房已于
  </span>
  1958
  <span lang="ZH-CN" style='font-family:
宋体;mso-ascii-font-family:"Times New Roman"'>
   年平调（霸占）给花滩区供销社做了酒厂。要回住房本属情理中事。
  </span>
  1964
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   年“四清运动”中称其行为是“反攻倒算”，又加上“写小说，用文字形式进行反革命宣传，期望反革命复辟……”等罪名，被戴上“反革命份子”帽子。由荥经县人民法院
  </span>
  [64]
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   刑审字
  </span>
  35
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   号判处管制三年。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   三、昭雪
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1978
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年中国共产党第十一届三中全会终于拨乱反正，平反冤假错案。
  </span>
  6
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   月
  </span>
  5
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   日，六合乡党委执行县委通知，行文摘掉了张学炜右派份子的帽子。
  </span>
  12
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   月
  </span>
  28
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   日县委、政府、公安、法院等有关单位在县招待所为幸存的右派份子们落实政策，恢复工作。可惜张学炜君已是九年冤魂，不能享受阳间俸禄了。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1979
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年
  </span>
  3
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   月
  </span>
  5
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   日，中共荥经县委荥发
  </span>
  [1979]
  <span lang="ZH-CN" style='font-family:
宋体;mso-ascii-font-family:"Times New Roman"'>
   第
  </span>
  31
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   号文件《关于改正张学炜同志被错划为右派份子问题的通知》下达，张学炜一案从官方角度看，算是彻底平反了结。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1982
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年张学炜之妻及未满
  </span>
  16
  <span lang="ZH-CN" style='font-family:宋体;
mso-ascii-font-family:"Times New Roman"'>
   周岁的小儿子张仕桃按月领取生活补助费。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1988
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年
  </span>
  12
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   月，六合乡政府批准明元春建房所需的宅基地
  </span>
  60
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   平方米，乡党委书记×××认为明元春孤儿寡母创业艰难，深表同情，照顾明元春木材
  </span>
  1.7
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   立方米作建房之用。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   呜呼！张君学炜，君子正人。热爱祖国，敬业人民。热爱生活，创作诗文。引蛇出洞，周纳深文。强权作祟，更加车轮。时年三五，永成冤魂。孤儿寡母，依靠何人？天愁地暗，草木噤声。幸有邓公，狂澜力镇。三中全会，拨乱反正。冤假错案，一律肃整。政通人和，国家兴盛。呜呼哀哉，伏维尚飨！
  </span>
  <o:p>
  </o:p>
