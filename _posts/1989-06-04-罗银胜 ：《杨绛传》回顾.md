   </o:p>

   </span>

   </o:p>
  </font>
 </p>
 <p class="MsoNormal">

  </o:p>
 </p>
 <p class="MsoNormal">

   </span>
   <o:p>
   </o:p>
  </font>
 </p>

   </font>
  </o:p>

 <p class="MsoNormal">

   </o:p>
  </font>
 </p>
 <p class="MsoNormal">

 </p>
 <p class="MsoNormal">

 <p class="MsoNormal">

  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">

   </span>

   </o:p>
  </font>
 </p>
 <p class="MsoNormal">

 </p>
 <p class="MsoNormal">
  <font size="3">

   </o:p>
  </font>
 </p>
 <p class="MsoNormal">

   <font size="3">

   </span>

 </p>
 <p class="MsoNormal">

 </p>
 <p class="MsoNormal">

 </p>

 <p class="MsoNormal">

   </o:p>

 </p>

 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">

 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">

   </o:p>
  </font>
 </p>
 <p class="MsoNormal">

   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">

   <o:p>
   </o:p>
  </font>

   </o:p>

 <p class="MsoNormal">

   </o:p>

   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    为了不误课，只有一个办法：早起早走。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    一次，她因夜间工作得晚了，匆匆梳洗后出门，一路疾走，赶到车站，盼车，挤车……
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    总算到了学校松了口气，可上教学楼的台阶时，一低头，发现自己穿的布鞋，真是忙中出错！
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    乍看起来，这只是一件令人发笑的小事，可也说明钱瑗精神紧张的程度，她像一架上紧发条的机器，已经停不下来了。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    有人问她近况如何，钱瑗回答：“心力交瘁。”
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    “为何不赶紧‘勒马’呢？”
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    “我是骑在虎背上的……”
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    人非钢铁，而钢铁“过度疲劳”也会发生断裂。长期负荷工作，使钱瑗这块钢铁出现了裂纹。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    先是咳嗽，继而腰疼，经过医院的专家会诊，确诊为肺癌晚期，已经病入膏肓了。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    这一消息，对杨绛来说无疑是雪上加霜，她的丈夫钱钟书已经重病在身，在医院卧榻不起。这时女儿钱瑗又病倒了，杨绛非常着急，分身照顾两个病人。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    钱瑗的病情发展很快，缠绵病榻无几，便告病危。三月四日下午钱瑗的心脏停止了跳动。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    白发人送黑发人，这是何等沉重的打击啊，但是，杨绛坚强地挺住了。她打起精神，全身心地照顾丈夫。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    人们没有想到，钱瑗去世不久后，在九八年十二月十九日的清晨，钱钟书也匆匆而去。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    钱钟书弥留之际没有经受痛苦，杨绛始终陪伴在他身旁，不停地用家乡无锡话在他耳边轻轻地祝福着。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    钱钟书的呼吸停止了，杨绛亲了亲他的额头，久久地贴着他的脸颊。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    杨绛非常坚强，从始至终没有落下一滴泪，她说：“钟书不喜欢人家哭他。”
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    这是多么深爱丈夫的一个女子啊，但悲痛与劳累还是使得杨绛显得尤为疲惫。惟一的女儿和一生的伴侣相继离去，杨绛晚年之情景非常人所能体会。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    天上人间，阴阳隔断，却难断亲情、挚情。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    在人生伴侣离去四年后，九十二岁高龄的杨绛独伴青灯，用心记述他们这个特殊家庭的风风雨雨、点点滴滴，结成回忆录《我们仨》。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    在这部书里，她向彼岸的亲人倾诉心声：
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    一家三人就此失散了。就这么轻易地失散了。“世间好物不坚牢，彩云易散玻璃脆”。现在，只剩下了我一人。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    我清醒地看到以前当作“我们家”的寓所，只是旅途上的客栈而已。家在哪里，我不知道，我还在寻觅归途。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    杨绛的确以“我们仨”自豪：
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    “我们仨是不同的遇合”，“我们仨都没有虚度此生，因为是我们仨。”
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    这样的话绝不是寻常关系的人能够说出的。这样的话寻常生命态度的人也无法说出。因而使得“我们仨”最后的失散，令人痛心至极。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    杨绛在余下的晚年生活中，一如既往，杜门谢客，潜心读书，直至生命的终点。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    二零一六年五月二十五日，这注定是个不平凡的一天，杨绛与世长辞了，去寻觅与女儿钱瑗和丈夫钱钟书相聚的归途了。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    这位一生奉献于文学事业的作家，翻译家杨绛，走了。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    她走后，世间再无女子可称先生。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
