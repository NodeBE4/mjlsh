   在现代作家中，老舍应该算一个厚道人，老舍出身贫困，他的这种出身使他总有些忧郁。总有些悲观，总有些疾恶如仇，总有些冷眼看世界，使他自觉地和穷人站在一起，同情他们，怜悯他们，为他们鸣不平，成为他们的代言入。老舍先生笔下的平民世界为中国现代文学贡献了一份珍贵的遗产，成为能够跻身于世界文学之林的作品，享有巨大声誉。车夫祥子，月牙儿姑娘，倒霉一辈子的老巡警不仅成为中国现代文学中的典型人物而家喻户晓，老舍先生的人文思想和人道主义思想同样感人至深。同样的道理，他比较顺理成章地欢迎和接受革命带来的巨变，也是缘于他出身穷人。可就这么一个厚道人，经过解放后的改造，也变成一个在历次文艺斗争中无役不与的

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   必须端正态度，学会忠诚老实
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   。老舍平时对丁玲的
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   优越感
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   是看不惯的，至少要煞煞她
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   狂傲
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   的锐气。
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   丁玲一向看不起我们，今天依然看不起我们。她的优越感使她在交代自己的罪过的时候，还想向我们示威，叫我们看看她怎么心细如发，会做文章。她若是不能忘了她的狂傲，忘了自己的面子，就不会忠诚老实。
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   他还特别提醒：
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   各反党小集团的男女老少，必须老老实实地把心灵中的垃圾倾倒净尽，重新做个干干净净的人！
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="115.jpg" border="0" height="412" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3653/115.jpg" width="550"/>

 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   老舍对那些落难文人的深文周纳，有时到了匪夷所思的地步。譬如，当时被打成
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   右派
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   的青年作家从维熙就曾回忆：他当年在《长春》月刊发表的小说《并不愉快的故事》，
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   不过是揭露农村生活的真实一隅，却首先被长春市文学界声讨，视为一株
  </span>
  '
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   大毒草
  </span>
  '
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   。此篇声讨文章被《北京日报》文化生活版转载，我尊敬的前辈作家老舍在《北京文艺》上著文时竟说：
  </span>
  '
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   从维熙写《并不愉快的故事》，意在煽动农民造反。
  </span>
  '
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   （我非常尊重老舍先生在
  </span>
  '
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   文革
  </span>
  '
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   中的太平湖之举，老人为后代文人作出不甘屈辱的示范。但为了尊重
  </span>

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

   年历史真迹，请原谅我写上了这一笔。）
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   老舍的原文是这样的：
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   我们看一看这些人的创作实践，不是就很清楚吗：刘绍棠的《田野落霞》，把农民、党员、干部写得无可再丑；从维熙的《并不愉快的故事》，竟煽动农民闹事，反对农业合作化。难道能说他们脑子里没有什么思想支配吗？我看这就是资产阶级右派思想作怪的结果。……《田野落霞》和《并不愉快的故事》能给人们什么教育呢？只能教育人们去反对共产党、反对社会主义。这不是很清楚的事吗！
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   过来人都知道，在那个年代，如此
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

   上纲上线
  </span>
  "
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，足以将丛维熙送进地狱

minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   这岂是一个厚道人所应做的？
