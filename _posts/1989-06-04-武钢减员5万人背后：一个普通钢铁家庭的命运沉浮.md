   作为工业化最基础的行业，武钢半个世纪的兴衰变迁，以及对这个农村家庭的影响，昭示的却是一个百年工业文明未解的核心命题：工业化和城市化是否是一个不可逆转、无法平衡的社会现象？当工业化兴起时，我们圈地造城，驱赶农民进城变成各类产业工人；但有一天当工业化过剩，城市经济衰落，这些失去工作，也没有更多谋生技能的失地农民或农民工后裔，该向何处安置？
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

 </p>
 <p class="MsoNormal">

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

  <o:p>
  </o:p>

 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

  </o:p>

 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">

  </span>

  <span lang="ZH-CN" style="font-family:宋体;

 <p class="MsoNormal">

 <p class="MsoNormal">

 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   失去土地后，万咸超和刚刚初中毕业的子女万劲松、万秀丽都进入武钢实业公司，在这个
  </span>
  1979
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年成立的钢铁服务配套企业里，三人的职位分别是司机、钳工学徒和包装工。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   到
  </span>
  2012
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，这个用于安置职工家属的配套企业，就拥有正式员工
  </span>
  2
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   万余人。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   当时，学习最好的万志宏正在念高中，他面临两个人生选择：通过高考进入大学殿堂，或者考取武钢职业技术学校，进入武钢。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   武钢职业技工学校始建于
  </span>
  1973
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   年，为武钢定向培养技术工人，尤其是轧钢技术工人。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在后来的回忆中，万志宏觉得“如果当时没有其它选择，我肯定就铆劲考大学了。”但最终，他选择了武钢职业技术学校，并于
  </span>
  1990
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年毕业后进入了曾经向往的武钢，成为第二热轧厂的一名轧钢工。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   每月工资一百多元，和初中毕业后当了三年学徒的弟弟万劲松差别不大。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="24.jpg" border="0" height="335" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3403/24.jpg" width="461"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   少年时期的万志宏和万劲松
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   唯一的不同是，万志宏的身份是“全民”、万劲松则是“大集体”。前者属于全民所有制的央企武钢核心企业，是共和国真正的工人阶级，后者则是武钢下属的集体所有制企业员工。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   上世纪
  </span>
  90
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年代后期的国企改革后，他们都成为法律意义上的合同工。但事实上，整个青山，所有人依然沿袭着红色年代以来的身份称谓：“大集体”还是“全民”，因为直到如今，两者之间，身份待遇依然泾渭分明，近几年工厂效益下滑，还曾爆发过同工不同酬的敏感事件。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="25.jpg" border="0" height="255" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3403/25.jpg" width="440"/>

  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   万志宏曾经工作使用的操作台
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   进入武钢后，万志宏和绝大多数的工友一样，三班倒，每天工作
  </span>
  8
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;

  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

 </p>

 </p>
 <p class="MsoNormal">

minor-fareast">
   白玉山街道的武钢菜场
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   直到这时，万志宏才发现，武钢这个他曾经仰视着，难以抵达的世界，是一个更加封闭的体系。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">

 </p>
 <p class="MsoNormal">

 </p>
 <p class="MsoNormal">

 </p>
 <p class="MsoNormal">

 <p class="MsoNormal">

minor-fareast">
   但整天只能对着流水线上的钢铁发呆的万志宏却觉得无聊。他开始写诗，把作品寄往全国各地，并动了离开武钢的心思。他对同窗好友夸下豪言壮语“我一定会离开武钢的。”结果被讥讽，“等你退休了自然就离开了。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">

 <p class="MsoNormal">

minor-fareast">

   分钟的公交车，从白玉山去青山拘留所外面拉客户，碰到“看起来像管事的家属”从所里出来，就走上去，“我是律师，有什么可以帮你的吗？”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   和他一起在拘留所门外竞争的，都是
  </span>
  20
  <span lang="ZH-CN" style="font-family:
宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   岁出头的小伙子。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   有面熟的人，他还走上去打招呼，热情地问候“又来了啊？”人家气汹汹地啐问：“你想干嘛！”尴尬了一下，万志宏闪到一边，无所谓地耸耸肩，等着下一个人走出来。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   从武钢这个封闭的环境里走出来，万志宏其实并不擅长找关系、拉资源；和任何一个刚刚起步的普通实习律师一样，业务并不稳定，生活依旧艰难；但万志宏说，关于这一点，他想得很清楚，心态也放得很低，他相信随着时间的推移，一切会改变。重要的是，他终于走出了武钢，有了自己可以主宰的命运。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   有身边的工友向他请教转型经验，他毫不留情地回一句，“晚了！”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   他猜，很多以前拒绝和他相亲的姑娘，如今应该和他的父母家人一样，开始后悔了。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="49.jpg" border="0" height="407" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3403/49.jpg" width="574"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   万咸超夫妇退休后居住在这个家庭的第一套还建房里
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   但其实整个万家，除了父亲万咸超，其他人对他的近况一无所知，他妹妹万秀丽是通过光谷客才知道，万志宏已经辞职将近一年了。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="50.jpg" border="0" height="397" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3403/50.jpg" width="590"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   被裁员后，万劲松依然穿着精诚公司的制服
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   万劲松同样不知道哥哥考上律师辞职的事，他正在找律师想打官司。从
  </span>
  2005
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年通过武钢下属的劳务中介精诚公司重新进入武钢开始，长达
  </span>
  9
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年时间，万劲松都没有社保，甚至连劳务合同都没见过。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   直到
  </span>
  2014
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，万劲松的劳务关系被转入另一家劳务中介公司，才终于有了
  </span>
  700
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   元的社保。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在这家公司，万劲松的合同期是
  </span>
  2
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   年，到
  </span>
  2016
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年
  </span>
  4
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   月
  </span>
  1
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   日。这一纸来之不易的劳务合同给了他信心，在
  </span>
  2015
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年流言最盛的时候，他也没有作任何其它打算，“就这么几个月，总会熬过去吧。”但
  </span>
  2015
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年最后一天，接到一个毫无征兆的电话后他就被要求上交工牌，工作结束了。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="51.jpg" border="0" height="398" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3403/51.jpg" width="555"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   从红钢城通往白玉山街道的
  </span>
  21
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   号公路两旁布满了武钢的烟囱
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   一群工友正抱团准备和劳务公司打官司，但万劲松对此并不报希望，“现在的武钢，打赢了官司也拿不到钱。”他每个月要为自己和妻子缴纳
  </span>
  1600
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   元的社保，为今之计，是急需找到一份工作。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   他在街道办做了失业登记。不久前，接到街道办通知，光谷有一家企业招聘保安，问他是否愿意。万劲松犹豫之后，拒绝了，“离家太远了”，他还是想找一份家附近的工作，哪怕再回武钢做临时工。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">

 <p class="MsoNormal">

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
