 <p class="MsoNormal">
  <b>
   <font size="4">
    <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
     －－作者：张宁生
    </span>
    <o:p>
    </o:p>
   </font>
  </b>
 </p>

 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   时间退回到
  </span>
  1951
  <span lang="ZH-CN" style='font-family:宋体;
mso-ascii-font-family:"Times New Roman"'>
   年，初二代数课，因为缺教师，停了几节了。我个子小，座位在临近教室门的第二列的第一排。上课铃响后，教导处曹主任推开门，同学们静下来，曹主任站在讲台边说：上面给咱们派来了朱老师，今后就由朱老师代你们班的代数。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   全班同学的目光转向门口，才发现还有一位小老太太。这就是曹主任介绍的朱芝英老师。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   此前我们已经撵走了三位代数老师。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   第一位“井”老师。三十左右的男老师（我在这里首先要为自己的轻浮致歉，老师的姓早已忘记），极有耐性，只是不自信，口齿又不利索。讲了一段，总有学生举
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   手说：“听不懂”。老师说：“不懂？听我给你‘井’（讲）”。于是学生在私下都称“井”老师。现在想来，老师原籍大概是晋北内蒙一带。讲普通话“
  </span>
  n
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   ”、
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “
  </span>
  ng
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   ”不分。每节如此，终至讲不下去，老师被调走学习去了。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   随后，从社会招聘代课老师，这一位要年轻些，情况更糟，“井”老师讲课是学生不清楚，他自己清楚。这位是学生清楚了，他还不清楚。常常挂在黑板上，推导不
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   下来了。哪儿出了错，硬是检查不出来。急得脸红、冷汗直流。学生在下面七嘴八舌，有的大同学干脆走上讲台，说：“我来推导吧”。三下五除二，把问题解决
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   掉。老师不好下台，连连说：“好好，以后我们就提倡这种讨论式的教学方法”。引出哄堂大笑。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   第三位是年轻的女老师，据说还读过上海某私立大学，只上了一节课。就“不高兴”教我们，不辞而别。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   这样的刺儿头班，搁谁都头疼。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   这小老太太衣着极普通，个头没我高，年纪靠五十了吧。短发留得是当时小学女生样式，后面只齐到耳廓中间。脸上有害天花留下的痕迹。戴着深度近视镜，透过一圈圈的镜片，只能隐约看见眯缝的眼睛。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   曹主任退出去，朱老师登台开讲：“同学们，今天我们来学……”。高亢的声音三五句就把全班镇得鸦雀无声。这满怀激情的鼓舞人心的又条理分明的演说，你没法不注意听。听了想说不懂也难。谁敢在全班同学面前无理取闹？我们高兴还来不及呢。终于盼来了一位好老师。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   朱芝英老师是老北师大数学系毕业生，教了一辈子书，还拿不下我们这帮小初中生？
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   跟朱老师是我学数学的一个转折点，不但成绩猛涨，而且有了爱好，初三朱老师给我们讲几何，打下了坚实的逻辑推理基础。课余我买了许莼舫编的《几何定理和证
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   题》自学，独立解决了其中二百多道练习题。几何成为我功课中的强项。中考发榜（竖排从右至左）时我从初中榜的左下角跳到右下角，正数第八考上了本校高中。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   高中阶段，朱老师给我们上三角课。同学们来自全市各校，有的早就听说过老师的名声。说是：这位朱老师可不简单，原来担任过耀华中学的教导主任。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   耀华中学是一所老牌的私立中学，民间素来有“阔耀华”的称谓。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “阔”是说学校办在“阔人”聚居区，学校有最好的设施，高薪聘请最好的教师，当然，学费也是一流的。（到五十多年后的现如今，学校早已归公，挂的校牌是“天津市实验中学”。）
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   知情者还介绍：朱老师一生未婚，在本市无亲戚，只有一位保姆照顾她的日常生活。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   听着叫人真不是滋味。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   朱老师还是那样慷慨激昂地讲课。但学生的崇拜程度却日益下降。高中阶段，铃铛阁的名师纷纷登台亮相，教学风格、学识素养、教学艺术让同学们眼界大开。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   我当时是三角课代表，有同学提意见说：“朱老师作业全批全改，不如细改几份，发回来我们自己对照”。这意见是嫌作业改得不够细致。可我对老师不能直说，那
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   太不礼貌。我和学习委员商量后，换一个说法：“老师负担过重，不如重点改几份作为范例，同学们下去自己对照着改就行。”朱老师却说：“有别的老师帮我改，
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   能做到全批全改。”这老太太直来直去惯了，不会猜度话中有话。我只能暗自苦笑。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   同学们对朱老师还是尊重的。一次下小雨，上课铃响后，朱老师还没来。同学们就催我：去看看，别让朱老师滑倒了！我跑出教室，沿着操场东面的走廊前行。教研
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   室在最北边，中间隔着整个操场。远远望见朱老师正穿过操场快步走来，我跳下走廊的高台迎上去说：同学们让我来接您，怕您滑倒！。朱老师说：我没事！依旧急
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   速前行。她光着脚穿着凉鞋，匆匆行走的样子就定格在我的记忆中，至今不能忘记。这位朱老师比我母亲还年长几岁啊！
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   高中毕业，和朱老师再没见面。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1958
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年春节，部分同学在天津重聚。说起：朱老师在反右中被划为右派，已经逮捕！
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   设想，朱老师如果象讲堂上那样慷慨激昂地鸣放，是一定会戴上右派帽子的，我们高中同班同学也有两位因出言不慎栽了的。但又何至于被“严惩法办”呢？
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   知情者说：“朱老师
  </span>
  1948
  <span lang="ZH-CN" style='font-family:
宋体;mso-ascii-font-family:"Times New Roman"'>
   年当过国大代表，
  </span>
  50
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   年被管制，是解除管制后才调到铃铛阁的。历史反革命又有右派言论的，新账老帐一起算，从严处理。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   百年校庆时，我们这一群老头聚到一起，有人又悄悄打听朱老师的结局。知情者说：“进去没几天就死在里面了。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
