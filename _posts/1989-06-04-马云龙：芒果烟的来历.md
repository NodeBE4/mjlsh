 <p class="MsoNormal">
  <b>
   <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
    <font size="5">
     芒果烟的来历
    </font>
   </span>
   <o:p>
   </o:p>
  </b>
 </p>
 <p class="MsoNormal">
  <b>
   <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
    <font size="4">
     －－作者：马云龙
    </font>
   </span>
   <o:p>
   </o:p>
  </b>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   一个年轻人正在吸烟，“芒果”牌的。绿色的烟盒，上面画着个盘子，盘子中央摆着一颗金光闪闪的芒果。一个问题突然在我脑海里闪现。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   我问他：“这烟是哪里产的？”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   他回答：“河南新郑。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “河南出产芒果吗？”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   回答：“不出，听说是一种南方水果。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “河南不产芒果，为什么河南烟用芒果来命名？”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   他被难住了，一脸茫然，说：“还真没想过这问题，到底是为什么呢？”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   唉，历史真的这么容易就被忘却吗？当年那个曾让亿万中国人如醉如狂的“盛典”，如今，在人们的集体记忆中居然连一点影子也没有留下。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   我给他讲了芒果的故事，他瞪着眼睛，像听天书一样，将信将疑。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   时间过去了四十多年，不算太久，但已经隔了两代人。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   那是
  </span>
  1968
  <span lang="ZH-CN" style='font-family:宋体;
mso-ascii-font-family:"Times New Roman"'>
   年的夏天。北京红卫兵的“天派”和“地派”之间正在武斗，各大学里到处是战地工事，用长矛、盔甲武装起来的大学生们正打得津津有味，超强的高音喇叭不分昼夜地互相谩骂着，震耳欲聋，大家都声嘶力竭地证明着自己才是正宗纯粹的“毛主席的红卫兵”。有的学校（如清华大学）还不时传出激烈的枪声。此时全国各地也到处是战火硝烟，江青最早提出的口号“文攻武卫”取代了早期的“要文斗不要武斗”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “大联合”的口号几乎没有人真心响应，谁也不甘心解散自己的组织，放下手中的武器，因为“枪杆子里面出政权”的道理已经深入人心。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   于是，最高统帅走了一步大棋：抛出了“工人阶级领导一切”的口号，派
  </span>
  8341
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   部队和“工人宣传队”一夜之间接管了学校，用铁扫帚把红卫兵扫下历史舞台。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   为了显示自己对军宣队和工宣队的器重和支持，在那年的
  </span>
  8
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   月
  </span>
  5
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   日，毛泽东亲自批示，把巴基斯坦总统赠给他的一些热带芒果，转送给了北京的工人代表。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   这个消息在电台和报纸上一公布，北京城像是爆炸了，上百万的工人连夜涌上大街，敲锣打鼓，狂呼万岁。紧接着，北京各大工厂都举行了盛大的迎接芒果大会，成千上万人对着一颗摆在盘子里的金黄水果顶礼膜拜，高呼万寿无疆。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   很快地，这股狂潮便席卷了全国各地。一批蜡制的仿真芒果被紧急制作出来，用火车和飞机运往各地，各省市都举行了隆重的仪式，人们在山呼万岁后，又排起长队，争先恐后地瞻仰这颗来自中南海的“圣果”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   世界上的水果怕有千百种，其中得到过如此无尚殊荣的，大概只有芒果了。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   如今说起“乒乓外交”，人们还津津乐道，其实在那之前，还有个更精彩的“芒果内交”，只是因为后来彻底否定了文化大革命，这事就没人再提起了，如今的年轻人听了像听神话一样，无法理解，也无从想象。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   我是北方人，从小熟悉的水果是苹果、桃、李、杏、梨和大枣，在那之前，从来就没听说过芒果，也没见过芒果（只见过蜡制的模型），更没尝过芒果是什么滋味。在河南农村插队劳动时，精明的新郑卷烟厂赶了个时髦，抢先拿芒果当了自己的牌子，生产了“芒果”牌香烟。这烟属中档货，两毛多一盒，不算贵，比那些一毛来钱的低档烟好抽，所以我经常买来吸。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   记得有一次在田间休息时，和几个农民一起吸烟，有个老头看着芒果烟的烟盒，感慨地说：“芒果这东西到底是个啥滋味？咱这辈子能尝尝就好了。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   当过兵的民兵连长说：“肯定好吃，要不外国人会送给毛主席？”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   好抬杠的生产队长说：“不一定。说不定是毛主席尝了觉得不好吃，才让送给工人吃了。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   接着就抬起杠来，直争得脸红脖子粗也没个结果。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   还有一次，信息灵通的小学民办教师张黑子问我：“你们北京学生犯了什么错误，才被下放到农村来改造？”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   我不愿谈这个话题，就说：“一言难尽，说你也不明白。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   他诡秘地一笑，说：“你不说我也知道。听说是蒯大富把毛主席给工宣队的芒果给偷吃了，毛主席一恼，就让你们都下放了。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   我哭笑不得，无言以对。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
