   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   1937
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年卢沟桥事变，北大被迫南迁，先生南游至江苏省南通师范，受教于章太炎先生。听太炎先生教诲后深受启发，先生萌生教育救国的理想，于是返回甘肃，任省立天水一中教师（
   </span>
   1938
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    －
   </span>
   1943
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    ）。期间，先生结识了许多文化界名流，与晋察冀边区军政人物亦多书信往来，为以后创建礼县中学（现礼县一中）打下基础。先生在天水执教期间培养出一大批有志青年，其中有毛作模（后任礼县中学副校长）、赵廷举（礼县名医）、赵志伟等。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   1942
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年，时任礼县县长刘剑白三顾秦州，先生出任礼县中学校长，开始对礼县中学进行具有深远意义的学术革新和学科重组工作。礼县中学成立于
   </span>
   1940
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年，是在“国立第五中学初中部”基础上建立的，是礼县一中的前身。先生的到来，开始了礼县中学的创建与革新工作。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <img alt="112.jpg" border="0" height="491" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3995/112.jpg" width="360"/>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    当时的礼县中学徒存其名，学科不全、师资贫乏。先生到来后大力革新，利用自己的社会关系聘请了大量优秀的教师（多为昔日的同窗和门生），其中不乏名流，如张尔如（南开大学毕业，周恩来同学）、周燕生等。又建立开设了代数、物理、英语、美术、体育等许多新学科，为一中后来的发展奠定了坚实的基础。同时以礼县中学为阵地，创办《漾波》学报，宣传民主进步思想，带动了一大批礼县的知识分子，营造了自由纷繁的学术氛围。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    先生在礼县中学七年（
   </span>
   1942
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    －
   </span>
   1948
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    ）时间里，治校有方，治学严谨，教学上倡导“熟读精思，严格磨砺”。在他的影响下，学校重视培养学生对学科基础知识的掌握，采取严格考试措施，奖励优异学生。期间，学校秩序井然，教学质量稳步提高，为礼县培养了大批有用人才，其中赵升甲（曾任礼县中学校长，后任天水地委副秘书长、教委主任、宣传部长）、赵崇德（著名学者，民盟天水地区主任，先后在礼县中学、礼县师范执教）、王穆（曾任天水教委主任）、张秉泽（曾任渭南师范校长）、丁维祯（曾任礼县中学教导主任）、蔺文举（甘肃省著名书法家）等为其门生中的佼佼者。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <img alt="113.jpg" border="0" height="496" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3995/113.jpg" width="360"/>

 <p class="MsoNormal">
  <o:p>

   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    先生平易近人，在生活中对学生很是关怀友爱，常常以自己的工资接济贫困的师生。当时天水陇南一带征兵严重，年青人被迫逃离家乡，先生利用礼县中学和自己的社会影响，庇护收留了许多青年，使他们免于战火，有一些更是成长为社会有用之才。其中就有甘肃书法家蔺文举。蔺为逃离征兵，从家乡西和县流落至礼县，先生遇之收留家中半年之久，后带入礼县中学，悉心教导。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    先生家境殷实，在解放前夕货币贬值、教师薪俸不给的艰难日子里，他曾多次捐助广大教工，赢得教工尊敬和爱戴。在礼县中学创建的困难时期，先生更是将家中许多财产奉献给了学校，而对公物却是丝毫不动。有次为学校购买了一件钟表，家人见制作精巧，欲留以家用，没想到先生勃然大怒，便马上把钟表交还给学校。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   1948
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年，先生离开礼县中学校长职务，去省城兰州师范学院执教，但仍不忘故乡教育，时刻关注礼县中学的建设和发展。
   </span>
   <o:p>
   </o:p>

 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    解放后，先生于
   </span>
   1952
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    年参加革命工作，
   </span>
   1957
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年被错划为“右派”下放劳动，
   </span>
   1961
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    在甘肃省省委委员、中共礼县委书记何承华（后任中共牡丹江市委书记、陕西省人大常委会副主任）的关切下重返礼县中学。十一届三中全会后落实政策，先生被平反。
   </span>
   1984
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年
   </span>
   9
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    月
   </span>
   19
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    日，先生病逝家中，其事迹收入《甘肃省教育名人大辞典》。先生病逝后，甘肃省教育厅、天水地区教委、陇南地区教委、民盟甘肃省委、先生门人弟子都沉痛哀悼一代宗师离世。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    先生一生倾注于教育事业，是礼县中等教育的奠基人，为礼县一中的创立做出了卓越贡献。在校长任上，他礼聘贤师，开设现代科学学科，使礼县一中成为真正意义上的中学。可以说，王荩先生是礼县教育筚路蓝缕、以启山林的一代宗师，是礼县一中奋勇前行、不断攀登的永恒旗帜！礼县一中七十多年的发展历程中，淬炼精神、萃取精华，现在“修身、包容、广博、科学”的校训，就是先生一生所有教育实践的具体体现。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    先生在治学之余又搜录了礼县知府王彭龄以及前清多位举人、贡生的诗词文稿，为礼县乡邦文献的保存整理做出很大贡献。他一生著述颇丰。精于古文，长于庄老，并长于古篆书法。遗作有《淡斋近稿汇存》、《读庄随笔》、《郡邑近人杂作录存》、《荀子叙拔录要》、《篆书周易—文言说卦序卦杂卦》、《先秦诸子鼎脔尝试集》、《汉宋清三代巨人赞颂备要》、《红楼梦薛林贾三人诗词举要》等，部分作品收藏于礼县博物馆。
   </span>
