<font size="3">

minor-fareast">
    年我在大后方重庆中央大学就读二年级，一天早晨上完两节课之后我在校园内走动，突然在学校告示牌上看到一张布告号召学生支援参加随军英语译员的工作。
   </span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

   </span>
   1941
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

   </span>
   12
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    月珍珠港事件后，中美同盟抗日，美国罗斯福总统派遣大批成千美军官兵来华助战（不是战斗部队），而是担任各兵种的武器补给训练，参加野战部队的参谋、作战联系等，战地急需大量英语译员服务。但当时大后方和沿海不同，不可能找到足够的英语人才—尤其是能在战地奔跑的青年—所以动脑筋转到学生头上来了。
   </span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
<font size="3">

   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:

<font size="3">

<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

   </span>
   1937
   <span lang="ZH-CN" style="font-family:宋体;

   </span>
</font>

<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

   </span>
   1940
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">

minor-fareast">
    年夏我中学毕业，我很自然走上了这条道路。
   </span>
</font>

<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

<font size="3">
    那时从华北去大后方较便捷是走陆路，沿津浦路坐火车南下在安徽附近下车取道河南界首，通过“阴阳交界”进入国统区。这条线上两方对峙一般相安无事，并无战事。通常有少数商人跑单帮的往来，维持两区一些经济活动。事实上，还是当权者招财进宝，发财的门路。但是人员来往必须化装。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

    年开始，以昆明西南联大方面动作较早，由梅贻琦校长动员，学生踊跃应召。所以早期在战地工作的译员都是西南联大的。重庆方面当时并未认真动员宣传，没有大动作。
   </span>
</font>
</p>

<font size="3">
    译员面试
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

<font size="3">

Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    我记得第一批志愿报名的不过十几、二十人左右。并且当时社会上认识不清，还心有疑虑。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    在大后方住过一段的人都知道，当时特务机关为了扩充其恶势力，时常假借各种名目欺骗青年人参加特务组织。什么“军委会报务人员训练班”等就是常用的招牌。因此一些家长因此常告戒青年人不要卤莽行事。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    我报名后并未立即报到，因为那时我方才大病初愈（疟疾，即打摆子），体力不佳，所以请假休息了一段时间，实际服役已经进入
   </span>
   1944
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年。
   </span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
<br/>
</span>
</font>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    这时形势又有了改变。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    因为战局的发展对译员的需要更为迫切，仅靠大学生志愿人员也不能满足需要了。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    所以
   </span>
   1944
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年初国民政府颁布法令，征调当年全部应届毕业大学生一律去部队服役。将最后一学期课程取消，提前毕业。规定服役到抗战胜利为止。届时颁发文凭，此举因为关系民族大义，所以大学生一般从思想上接受，主动配合。
   </span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<br/>
</span>
</font>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    但也听到一些小插曲，有些达官贵人的子弟，大概觉得“随军”两字还很可怕，所以有人通过后门关系在什么后方军事单位补个名字，就算是应征服役，实际不去前线的。这时国民政府教育部次长顾毓秀曾到中央大学讲演、鼓励大家。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    征调令下达后，当即就在沙坪坝中央大学内办起了一个训练班，我也在此时奉令参加受训。这个训练班并无固定课程，也无时间限制仅由校内教师和美国军官讲一些应急的英语口语知识之类。因为大家水平参差不齐，并且前方急等用人也实在紧迫，所以看你差不多了就先走。我因是沿海来的，程度不错，只学了十几天就作为第一批人员（共十几人）提前服役。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    那时译员的去向有三个：
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    一个是去驻印军（
   </span>
   X
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    —军）；第二个是滇西的远征军（
   </span>
   Y
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    —军）；另一去向是广西（
   </span>
   Z
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    —部），还没有什么重大军事行动，只在做些准备工作。
   </span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
<br/>
</span>
</font>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    沙坪坝这个训练班，后来被命名为“沙（沙坪坝）磁（磁器口）译员训练班”，我们这批人被称为“第一届学员”。但我们这些当事人并不知晓，那都是我们离开以后的事了。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    我本来想去印度，后来考虑热带气候恐不适应没敢报，就去了云南，那里也是需人最多的地方；那时怒江战役已在紧锣密鼓，即将开始，我们这一批人正好及时参加了战役全部过程。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    译员的身份
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    译员隶属军事委员会外事局管辖。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    我们十几个人于
   </span>
   1944
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    年三月初被送到重庆城内外事局报到（局长商震）。简单办了手续，每人发了一身黄军装和一顶军帽，不是士兵的军装，但也没有领章等。
   </span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
<br/>
</span>
</font>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    我们的正式身份是军中文职人员，一般是三级译员，拿“同上尉”工资（在通货膨胀时期根本没有几文），并且因是单身一人，连食物米贴也不需要发给。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    在外事局只住了一夜，第二天就乘美军用飞机，直送昆明美军总部。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    关于“同上尉”待遇有必要做一些解释，这本是军队中文职人员的称呼，严格讲，并不是军官，一般常指为军队服务的医务人员、专门技术人员和教师等，后来一些文工团等也列在其中。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    凡是有些辩证法思维的人员大概都可以承认：既曰“同上尉”，那就是和上尉不同的另一个概念，身份、待遇等完全不同，相同的只有一点，即每月支付的法币工资数目和上尉相同，同理有少数二级译员是“同少校”待遇，拿得工资又多一些而已。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    有不少有过译员经历的人员在文革中吃了不少苦头，并且还出了一些悲剧。因为有些人武断的说，“同上尉”就是上尉，而国民党的带兵连长也不过是一个上尉，当时的“公安六条”明文规定这就是典型的反革命。实在是有理说不清，这话不必多提了。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    二、怒江战役背景及态势
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    我国云南省怒江（萨尔文江
   </span>
   Salween River
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    ）西岸直到中缅边界广大国土。自

<p class="MsoNormal">

</font>

<p class="MsoNormal">

<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
<font size="3">
    两星期很快过去。第二天清晨吉普车等在门前，我们几个人一路疾驶西去。沿着滇缅公路，经楚雄，祥云等地日夜兼程。夜晚公路上车灯点点，车辚辚，马萧萧，军运车不断。

<p class="MsoNormal">

<p class="MsoNormal">

<font size="3">
    这是工兵的重头戏，安排的战术是橡皮艇操纲渡，操纲渡就是准备一条粗缆绳，一头栓牢这边岸上，由少数战士使用美国供应橡皮艇划过对岸，将另一端系牢固定，这样汹涌的江面上就会横跨一条缆绳，后继的船只只要人力摸着缆绳就可顺利渡到对岸。
   </font>

<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

<font size="3">
    渡江

<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

    时间已经进入五月份，战役即将打响，我们要渡江了。那时译员多数人是在基层单位中的“联络小组”中工作，小组由美军官、军士和译员混合组成，一般只

   </span>

   <span lang="ZH-CN" style="font-family:宋体;

   </span>
   11
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    日开始，实际情况和预期很不同，原来日寇兵力不足，无力在江边阻击抵抗。改在西岸高黎贡山山脊处设防。我原定在沿江十几处渡口同时强渡，现在用不上了。

<font size="3">

<p class="MsoNormal">

<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

    这一阶段前线向前推进已接近中缅国境线，一路上，我们住的房舍一般就是敌占时日军占用的，因为当地本也就是不多那几所民房。我们入住时常看到日军遗留各房间上挂的小木牌，写着“食堂”，“便所”（厕所）等，还有“慰安所”，这些以侵略杀戮为乐，以荼毒无辜平民为乐的皇军“勇士”是要靠这种方式维持士气的。这时战役已近尾声。时间已是

    抗战胜利，大批美军联络组不再需要，译员也无事可做了，“遣散”在即，很快做了安排。九月份我们奉命经柳州空运回昆明。那以后是否还有少数译员留下来，我们就不清楚了。回到昆明后，我们回到外事局昆明办事处办了手续，领到了遣散证明，每人还发了拾万元法币，叫做“胜利奖金”，于是还我自由之身，又是学生身份了。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    但随之而来的是再也无人照顾我们的生活食宿，记得还是通过私人关系在西南联大学生宿舍中，找个地方，凑合过的这段时间的。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    遣散费看来数目不少，但在通胀严重的那时，只够供我这个穷学生维持了一二个学期的零花开销（伙食有“贷金”，不算），因为只有把它拱手送到银行，领取一点可怜的利息，眼看它像冰水一样化为乌有。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    应征服役的同学，因抗日胜利，有言在先，只等领取毕业文凭就业。我们志愿应召的，一般是二三年级，欠课很多，还需回学校复学。这时摆在我面前的还有个难题，因为我要回重庆，当时交通很困难，并且那时又已是无人过问负责的学生。然而终究还是回了重庆，我于
   </span>
   1945
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年十月回到沙坪坝复学。
   </span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
<br/>
</span>
</font>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    就是这几天本来安排有一次追悼会，悼念阵亡译员，我当然要参加的，记不清楚临时不知怎么的耽误了，现在想来，可能和中央军解决龙云的军事行动有关。蒋介石收拾龙云就选在这几天动手。他选择这几天调虎离山，把滇军调去越南“受降”，这边中央军就动手“接防”了，打了一二天就解决了问题。龙主席被架上飞机到重庆去做“院长”去了。所说的追悼会可能根本没开成。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    记得很清楚的是牺牲译员共九人，驻印军方面较多，他们所在那边战斗看来更激烈些。国内远征军方面约二三人。实际情况应以外事局的档案记录为准。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    十九个月的译员生活过去了。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    黔桂大溃败
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    这里想顺便谈一下当年的黔桂大溃败。在
   </span>
   1944
   <span lang="ZH-CN" style="font-family:
宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    年后半年日寇沿粤汉路南下经湖南直指广西。
   </span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:
宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
<br/>
</span>
</font>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    那时日本帝国主义有一个狂妄计划，要进广西。打到安南（越南）修一条纵贯亚洲大陆从我国东北直到新加坡的铁路。这是他们的“南进”。也是日寇在抗日战争中最后一次反扑。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    日寇推进到桂林、柳州后突然向北一转，闯入黔南，进入独山、都匀一线。这时从东边逃来的大量难民和败兵糜集在新修的黔桂铁道上。无数的人拼命往火车上扒，乱成一团。“陪都”震动，传说又要“迁都”了，但迁到哪里去呢？据说只能迁西昌。刹时间大后方也到了“放不下一张书桌”的局面。大学生纷纷参军，所谓“十万青年军”运动。学生大量参加到空军和各种军事机构中，都在这个时候。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    那时我们已在怒江前线，没有亲身感受这一混乱局面，并且还陶醉在前进胜利的喜悦中。但是后方不断有消息传来，后来回渝也有许多传闻入耳。
   </span>
   1945
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年我们进入广西境内。那时社会秩序混乱，难民很多，多少还能体察一些前一年的混乱和恐怖。
   </span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
<br/>
</span>
</font>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    六、译员与盟军
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    译员的待遇和生活
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    首先谈一下译员的待遇和生活。我们应由外事局支给法币工资，在通胀严重的当时，本已是很微薄的。这笔钱每月由重庆先拨到外事局驻昆明办事处，而译员分散远在滇西（或印度）的大山丛林中，巡回流动，那里去找？不过也不需着急，慢腾腾往下拨转就是。这样贪官污吏手中就经常多了一大笔钱可以挪用，做生意的资金了。我做译员一年半以上，清楚记得只领到过一次转来的工资；有一段时间，在小乡镇理发的钱都付不出，很狼狈。这种情况已影响到译员的工作，所以到了
   </span>
   1944
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年后半年，不得不安排改善。美军总部和重庆协议，由它们代发，将来在“租借法案”范围内结算。其实问题很简单，美国人效率奇高，每两个月来一个美军会计，我们本人签个字，钱就到手了。
   </span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
<br/>
</span>
</font>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    伙食没有问题，跟着他们吃就是。联络组随身有法币，只要有集市都可采购，并且还有美国军用配给，罐头等供应。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    我们的一身黄军装，到战地没有几天就磨烂了，形势所迫，不得已都穿起美国卡其军装。只是有一样，译员不喜欢美国大兵的船形帽子，所以多数人都设法保留一顶中式有沿的中国军帽（有帽徽），出进机关部队，提醒人们，“我是中国工作人员”。其他被服、野外装备、雨具、吊床、生活用品，也酌量由美军后勤供应（
   </span>
   S.O.S,
Service of Supply
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    ）。此时算多少有了明文章法。美军还有些奢侈品供应，如香烟和巧克力糖果等，我们没有。不过有烟瘾的人，要一点抽，也没什么大问题，东西很多。那时对于美国的物质文明已有些印象。
   </span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<br/>
</span>
</font>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    租借法案
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    关于“租借法案”一事，现在除了专门研究历史的人以外，恐怕都不大清楚了。原来二战中美国参战后，是同盟国中唯一有能力提供军事装备、物资的国家，但受援的英、苏和中国都拿不出钱来，所以美国国会专门通过一项租借法案（
   </span>
   Unleasement
Act
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;
mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    ），规定援助各国的军需用品一律按租借物资处理，不交现款，留待战后结算。上世纪七十年代我还看到过：苏联清偿一笔租借法案欠款的新闻报导。至于中国这笔帐后来怎么结算就不得而知了。
   </span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;
mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<br/>
</span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    云南境内远征军范围的译员，一般个人不配备武器，驻印军方面似乎是有枪的。联络组当然随身有长短枪。为了照顾盟军，那时候，联络组一般不安排到距敌步枪有效射程（
   </span>
   200m
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    ）以内的阵地活动，而在机关枪有效射程（
   </span>
   500m
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    ）以内就不好说了。另外，冷枪、冷炮的危险也不可避免。
   </span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
<br/>
</span>
</font>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

minor-fareast">
    译员在外事局都有编号，我参加工作较早编号在前。据我
   </span>
   1945
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年见到的，参加工作较迟的，编号已到
   </span>
   2400-2500
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    左右，所以估计在这一历史事件中，参与战地工作的译员总数应在
   </span>
   3000
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    人左右，大概误差不大。（有些人估计数目更多些）

</font>

<font size="3">
    译员与盟军的关系
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    因为是同盟抗日的缘故，美军人员和译员相处关系一般是正常融洽的，但有时也有些争吵。问题主要起因于美军个别人员的傲慢、无礼上。这批译员不是雇佣来赚美金的，而是敏感、有尊严的中国大学生。有人一怒之下，有时就不告而别，找个便车回昆明，请求外事局办事处另派工作，但也没听说过更严重的问题。
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
<br/>
</font>
</span>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    美国年轻军官，许多是在大学中通过“后备军官训练”（
   </span>
   R.O.T.C.O Reserved officers
training courses
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    ）取得了尉官资格，战争一来就上了前线，各方素质未必比中国职业军官有经验。美国军士阶层中工农劳动出身的多些，这些人语言较粗俗，知识面也窄，许多农家子弟很朴实。我在工作中接触较多的美国军官，级别较低，一般是尉级（相当中国方面营连级），连上尉和少校都很少，但有关训练教官等方面都请他们作指导。我个人考虑，如果涉及新式装备，自然另当别论，而比较普通的军事项目，恐怕多少有些礼仪的考虑。
   </span>
</font>
</p>
<p class="MsoNormal">
<font size="3">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<br/>
</span>
</font>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    前面说过，联络组的中美人士一般相处比较友好，因为一般情况连译员一共也就三五个人（少的时候只有三个人）。长期同吃同住，确实是彼此在一个壕沟中并肩作战，并且年龄也较仿佛，在难得的战斗空隙（主要在后期），都在一起玩纸牌，下国际围棋，或者拉家常，闲谈两国的风土人情。我个人感到，在老百姓、平民这一个阶层，不管什么民族都不难建立友谊的。
   </font>
</span>
</p>

<font size="3">
    中美友谊长存
   </font>
</span>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<font size="3">
    大后方云南等地，老百姓对美国“飞虎队”的援助比较熟悉，这是二战中，陆地野战之外，盟军对我国最大的援助项目，此前，日寇依靠强梁，轰炸我国重庆和昆明等地，给我造成极大损失，“飞虎队”一来，情况就不一样了，大后方老百姓都曾目睹“飞虎队”战士激战长空，击落日寇多架飞机；但当然也曾目睹我方个别飞机因为众寡不敌，被击落坠地牺牲的悲壮场面。云南有些百姓就曾亲自为他们收拾尸骨，安葬立碑（如云南省祥云县），现在南京及昆明都有大型的国际空军烈士纪念碑（依照人数为美、中、苏），对于促进国际上人民的友谊，起到了很好的作用，除了一般游客的凭吊瞻仰外，有些烈士家属在碑上摸索着找到他们自己亲属的名字，都不禁痛哭失声，她们了解到中国人是有情有义的人民，不会忘记在中国人民最艰苦的时候牺牲生命来华援助我们的外国朋友，这些烈士家属多半是当事人的姐妹、甥侄之类，因为空军战斗需要最精壮的小伙子，许多人都来不及结婚留下后代就牺牲在中国了。
   </font>
