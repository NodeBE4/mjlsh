   榜第三名，最后列
  </span>
  1951
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年度榜单的第
  </span>

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

  </span>

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

   榜的唯一一首的中文原创歌曲。完全可以说，《玫瑰玫瑰我爱你》是世界上最有名的中文原创流行歌曲。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1999
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，中国女足在亚特兰大与美国女足决战世界杯时，全场华人高唱的，也正是这首《玫瑰玫瑰我爱你》。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">

  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   《玫瑰玫瑰我爱你》的首唱者：姚莉
  </span>
  <span lang="ZH-CN">
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   比起《玫瑰玫瑰我爱你》，这首歌的曲作者陈歌辛作词作曲的另一首歌在华人世界更有名，这就是“过年超市神曲”《恭喜恭喜》，当然，它最初是为庆祝抗战胜利而写的。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   陈歌辛还是《夜上海》、《苏州河边》、《永远的微笑》、《花样的年华》曲作者，周璇所唱红的电影插曲，至少有三分之一由陈歌辛谱曲。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   和创作上的成就比起来，它们的曲作者陈歌辛的经历更为传奇。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   他一生三度入狱，第一次被当成共产党，关进人间魔窟的“
  </span>
  76
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   号”，第二次被当成汉奸，关进了国民党的大狱，第三次以“右派”的身份，进了劳改农场。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   陈歌辛是旧上海的“歌仙”，当红明星以唱他的歌为荣，到了新中国，他的音乐立刻成了被批判的“小黄歌”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   《玫瑰玫瑰我爱你》在海外累积了百万美元的版权，但当时国内的环境下，陈歌辛能说的，只有“如果给我，我会捐给国家买飞机”——当时是抗美援朝时期。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   他的歌在美国成为“洗脑神曲”时，他在国内被戴上“右派”的帽子。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   三年困难时期，他的儿子陈钢凭《梁祝》红遍中国，而一代“歌仙”的他，饿死在安徽。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   仙是不食人间烟火的，这个典型的旧上海文人，也与那个政治风云动荡的时代格格不入。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  2
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   陈歌辛的名曲都是创作于
  </span>
  40
  <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   年代，这个年代压到一切的主题是抗日、是救国，是黄河大合唱、是放下你的皮鞭，陈歌辛的那些靡靡之音与时代主题显得那么格格不入。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   心理学中有一个分支叫叙事心理学。名字很玄，但其实是说，人生经历虽然复杂，却都在寻找一个明确的主题，讲一个属于自己的故事。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在我看来，陈歌辛寻找的人生故事是“孤岛式的浪漫”－－一个无关时代，只属于自己的浪漫故事。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="108.jpg" border="0" height="300" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3566/108.jpg" width="436"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   陈歌辛和妻子金娇丽
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1937
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年之前上海，时局虽动荡，对普通中产阶段的生活并没有什么影响，好莱坞的电影同步上映，爵士音乐随着收音机进入千家万户。那时的上海是一个无关中国时局的“孤岛”、属于世界的上海。生于上海的陈歌辛，跟随流亡上海的犹太音乐家学习钢琴和作曲，已经为自己今后的“孤岛式的唯美主义”音乐风格定下了调性。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   《永远的微笑》陈歌辛第一首流行音乐，是一首写给妻子的一首永恒的情歌，他觉得妻子长得像蒙娜丽莎，有着永远的微笑。在抗战的背景下，它是一个无人可以进入的内心孤岛，纯粹是个人的情感。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在抗日的大环境里，陈歌辛也写过几首抗日歌曲，很快被汪精卫抓去，折磨了三个月。出来后，陈歌辛仿佛才知道自己注定不是个革命者，风花雪月的舞榭歌台才是他人生的宿命，他最有名的歌都是这之后创作的。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   创作《苏州河边》时，日军已全面占领上海，上海的“孤岛”时代结束，歌中唱到“不知是世界离弃我们
  </span>
  /
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   还是我们把它遗忘
  </span>
  /
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   夜留下一片寂寞
  </span>
  /
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   世上只有我们俩个”，你说是“粉饰太平”也好，你说是“心中幽怨”也罢，外面世界的“孤岛”虽已不在，内心的“孤岛”谁也进不了。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   就连《恭喜恭喜》这首庆祝抗战胜利的歌，后来也成了没有任何政治色彩的“过年洗脑神曲”。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   也难怪，它和一般的雄纠纠气昂昂的抗日歌气质相差了十条苏州河。仔细体会一下歌词“经过多少困难
  </span>
  \
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   历经多少磨练
  </span>
  \
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   多少心儿盼望
  </span>
  \
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   春天的消息”，没有“漫卷诗书喜欲狂”式的欢庆，仿佛在说，不错不错，噩梦都过去了，大家恢复正常的生活吧。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   堪称国民花样美男的陈歌辛自然也有他的风流往事。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   陈歌辛的儿子陈钢在《母亲教我的歌》中写道：“妈妈（金娇丽）是上海吴宫饭店大经理的千金小姐，而爸爸则是个风流倜傥、目空一切的穷书生。”事实上，陈歌辛当时是复旦中学的音乐老师，金娇丽是他的学生，仅
  </span>
  17
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   岁，大着肚子嫁给了他。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   未婚先孕，师生恋，即使是那个时代的新派文人，也不多见。泡在明星堆里的陈歌辛还与李红、李香兰、姚莉相继传出诽闻，至少与李香兰的纠葛是被儿子陈钢后来证实的。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="109.jpg" border="0" height="494" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3566/109.jpg" width="413"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   本文部分内容参考陈歌辛之子陈钢的《玫瑰之恋》
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1949
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年后，陈歌辛再次希望融入这个时代，先后为多部电影写了很多现在已完全想不起来的曲。写惯了十里洋场的才子佳人，突然要歌颂起劳动人民，这个气质完全不搭啊。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   《游击队歌》的作者贺绿汀在多年后回忆，这顶帽子本要给他戴上，因为上海市长陈毅的力保，就让陈歌辛“顶替”了。看上去有点冤，可是仔细一想，他就算躲得过
  </span>
  1961
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年的肉体折磨，也逃不了
  </span>
  1966
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年的精神肉体双重折磨。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   陈歌辛在旧时代是个新派文人，在新时代又成了个旧派文人。陈歌辛还是那个陈歌辛，时代已不是那个时代了。
  </span>
  <span lang="ZH-CN">
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  3
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   艺术家从来都是有两种，一种能准确把握时代脉博，用艺术融入时代，比如贺绿汀、洗星海，一种无论时代如何变化，只会在自己的“孤岛”中，表达自己的内心，比如陈歌辛。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
