    这个在严家被叫做“老三”的年轻人，在当时还有一个被人在私底下取的绰号－－“土匪”，这个绰号和他自幼抑郁寡欢、性情暴躁有关。

   <span lang="ZH-CN" style="font-family:

    战乱时的教书先生

Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    在那个烽火连天的年代，这个教书先生因为才气逼人、思维延续着父亲严复的开阔、坚持黑格尔的唯物论、对危难中的国家和民族颇多忧虑，吸引了不少进步学生的目光，但他的生活仍然保持着在郎官巷
   </span>
   27
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    号的诸多习惯，比如早晚烧香念经。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>

 <p class="MsoNormal">

    月，日寇沿浙赣线西进，直下江山、上饶、铅山，闽北局势紧张。严叔夏突然萌生了山居观变的念头，决然离开了协大，结伴买舟，沿富屯溪南下，避居古田山乡一学生家中。

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年，严叔夏转入南平省立师范专科学校教国文。其时，他的长衫换成了袖子常常被挽起的短装，开始阅读《西行漫记》、《大众哲学》等书。一向抑郁的心境也开始变得明朗。

 <p class="MsoNormal">

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    独留大陆
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">

 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
    抗日战争胜利后，福建协大迁回福州。严叔夏重回协大任教。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    大局势的暂时稳定并没有给这个分隔两地的家庭太多喜悦。妻子林夫人一家打算在这个时候迁居台湾，投奔当时在台湾颇有经济实力的祖父林维让，避开大陆的政权之争。因为在八年抗战的特殊时期体验过独立新生活，严叔夏已经不可能再回到原来的轨道，他再次背叛这个家族的选择，独留大陆。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    三年内战时期，严叔夏在福建协大这所以保守闻名的教会学校的教师中，像个异数。他继续大量阅读所能得到的有关马克思主义哲学、政治经济学的资料。在协大，他是学生运动的领袖们无所不谈的朋友。

 <p class="MsoNormal">
  <font size="3">
   1947
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年下半年，由中国民主同盟福建省负责人林植夫、周问仓介绍，严叔夏开始参加地下民盟的一些活动。福建协大的地下党员经常在他的住所存放物件和过夜。

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    年，学生运动频繁。他曾经专程拜访当时任福建省主席的李良荣，向其郑重提出希望他对学生运动采取宽容态度，不要抓人、杀人。

 <p class="MsoNormal">

Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    亲历政治
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>

 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    福州解放后，严叔夏被选为福建协和大学校务委员会主任委员。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    这个并不熟悉行政工作的
   </span>
   50
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    岁男人开始以参与者的身份接触政治斗争。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    当时的协大是一个接受美国教会津贴的私立学校，特殊的时期使这个学校同时处于两个委员会的管理之下，一个是行政当局的校政委员会，一个由全校师生选举的校务委员会。校政当局受美国教会直接控制，在对待学校改革的具体工作上，常常表现出消极、被动、观望、不合作，甚至公开敌对的态度。校务委员会则在人民政府领导下，监督和协助学校当局执行新民主主义教育方针的机构。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    两个委员会持不同立场、代表两种力量，斗争便不可避免。对于校务委员会来说，在维持正常的教学秩序的同时，团结全校的师生并不是一件容易的事情。而这个校务委员会主任委员的任务就是，在这场没有硝烟的斗争中取胜。

   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    这样的挑战，是符合逐渐快乐起来的严叔夏关于政治的理想的，而
   </span>
   1957
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年之前发生的一切也都足以证明，这个在新旧更替的年代出生、成长、成熟的文弱书生，有足够的能力完成他为自己设定的每一次转变，直至
   </span>
   1957
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年。

   </font>
