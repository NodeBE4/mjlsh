 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    从你家中的摆设也能看出你不是贪官，非常一般，普普通通，家具都相当破旧。
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    何参谋在
   </span>
   41
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    团就因为太正直挨整，受排挤。他心地坦荡，表里如一，从不溜须拍马，得罪了刘副政委（就是那个搞了十多个女知青的白发苍苍的老军人），当面挖苦过他当过国民党的兵。在
   </span>
   1970
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年开门整党期间，因为说话不注意，被人提了意见。春节时他喝醉酒哭了，发泄了一番心中的郁闷，结果被人上纲上线举报。团里硬说他耍酒疯，影响极其恶劣，给了他留党察看处分，从
   </span>
   4
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    连指导员降为
   </span>
   7
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    连连长，后又再次降为团部林业参谋。
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    他有良知，不圆滑，有棱有角，理解被冤枉的苦楚。因而面对一帮人殴打我时，才不顾后果，帮了我这个现行反革命。
   </font>
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <img alt="91.jpg" border="0" height="378" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/4177/91.jpg" width="500"/>
  </font>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    与何参谋的两个儿子
   </font>
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    何参谋的正直，不势利，在那个年代，那个环境下真是凤毛麟角，百里挑一。
   </font>
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    对一个毫无自卫能力的社会最底层的贱民态度，最能看出一个人心底的善恶。给他一瓣蒜，一块月饼都需要巨大勇气和仁爱。而何参谋却让个现行反革命分子跟他同骑一匹马，丝毫不怕被扣上同情反革命的帽子。当一群兵团战士群殴我时，立即让我返回连队，制止了他们要把我彻底打服的企图。如果他撒手不管，任凭革命群众修理我，修理到死，他也没任何责任
   </font>
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    那个年代是对阶级敌人越残酷无情越好的年代。打死个现行反革命很平常，顶多轻描淡写批评几句，不会有啥事。别说阶级敌人的命不如条狗，就是普通人的命也不值钱。就在我被打的
   </span>
   1971
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年
   </span>
   7
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    月，
   </span>
   43
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    团
   </span>
   9
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    连兵团战士安建设由于和某兵团战士闹矛盾，在那人茶缸里尿泡尿，即被众多兵团战士活活打死。这呼市小青年来兵团才
   </span>
   3
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    个月，年仅
   </span>
   17
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    岁，比我更倒霉。
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    最可贵的是何参谋自己当时的处境也不好，
   </span>
   41
   <span lang="ZH-CN" style="font-family:
宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    团领导反感他，整过他，他是冒着某种风险救了我一把！若被人举报“同情现行反革命分子马波”，让李主任抓住，肯定会更狠地整他。所以，
   </span>
   40
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    多年了，始终忘记不了他。
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    顺便说一下：兵团里的现役军人名声不佳，犯错误的很多。他们粗暴蛮横，贪污受贿，乱搞女战士，偷盗公物，打击报复，捆绑吊打知青等等屡见不鲜，可也不全都是坏人。里面也有善良的，有正义感的，有良知的，不腐化的，何参谋即其中一个。
   </font>
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <img alt="92.jpg" border="0" height="395" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/4177/92.jpg" width="550"/>
  </font>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    与何参谋家属及亲友
   </font>
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    妈的，乌拉斯台就是我的野猪林。当我被一帮凶恶的小混混毒打时，瘦小的何参谋就像鲁智深，突然从天而降，解救了我。若没有何参谋，真不知班长那伙人最后要给我打成什么样子，打伤打残甚至打死都说不定！
   </font>
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    何参谋对我有大恩啊！
   </font>
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <img alt="93.jpg" border="0" height="357" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/4177/93.jpg" width="500"/>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    与何参谋的双胞胎女儿，出生在高力罕
   </span>
   41
   <span lang="ZH-CN" style="font-family:
宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    团的爱边和爱疆
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    他“犯错误”后，调
   </span>
   7
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    连当连长，还上石头山看过我们，但那时我跟他没说过一句话。一辈子就是
   </span>
   1971
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    年
   </span>
   7
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    月
   </span>
   3
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
    日上午跟他交往了个把小时。因为救我时他是参谋，所以一直习惯称他为何参谋。
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
    给何参谋送行前，特地洗了澡，换上干净衣服，穿上儿子新买的耐克鞋。当年给赵总
   </span>
   <span lang="ZH-CN" style="font-family:宋体;
mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
    书记送行时，也特地洗了澡。平时自己很少洗澡。从小学就不爱洗澡，这毛病终生未改。但去看何参谋前很自觉地洗了。何参谋对我有救命之恩，必须洗干干净净去见。
   </span>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    这次去山西送行，还幸运地得到了何参谋的相片，十分喜悦，再也忘记不了他的相貌了。
   </font>
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    患难时刻，谢运来偷偷递给的一瓣蒜，张伟公开给的一小块月饼以及何参谋叫我这个“现行反革命分子”跟他同骑一匹马，并让我马上回连，帮我脱离了那帮恶少，都是自己永远难忘的经历。
   </font>
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    妈个蛋的，何参谋，一个普普通通的小官，却灼灼闪辉，让老鬼一辈子忘不了。
   </font>
  </span>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   <font size="3">
    何参谋永远活在老鬼心中。
   </font>
  </span>
 </p>
