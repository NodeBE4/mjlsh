   <span lang="ZH-CN" style="font-family: 宋体;">

 <p class="MsoNormal">

 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    唯一的疑点是，作为通信员的他，在李显斌偏离航道飞行时，为什么没有给基地发信号？是不是李显斌关闭了通讯电台？还是他发出了，地面没接收到？
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    因为死无对证，李显斌的文章又混淆视听，这边不知道真相。或者是，也不想知道真相，反正坏事已经出了，赶快处理了为好。反正对岸那边已经把他们宣传成「反共义士」了。伟大领袖说过一句话：
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    凡是敌人反对的我们就要拥护，凡是敌人拥护的我们就要反对。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    最后，三人一起被定为叛徒。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    「叛徒」这个词，就像一份命运的死刑判决书。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    尤其是对还没搞清楚状况的三个家族来说。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span style="mso-spacerun:yes">
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   3
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    廉宝生是天津市宁河县宁河镇西关村人，在家排行老二，还有四个兄弟和两个姐妹。他身高近一米九，长得阳刚帅气，一看就有勇士之风。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="79.jpg" border="0" height="434" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/4588/79.jpg" width="350"/>
  <br/>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    提起「廉」这个姓，我和你们的反应都一样，首先想到的就是两千年前的赵国大将廉颇，「尚能饭否」的主角。事实上，廉家祖籍正是河北邯郸－－赵国的首都，还真说不准就是廉将军的后代。清末，河北南部地少人多天灾频发，廉家人逃荒来到宁河。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    也许是遗传了祖上尚武的传统，廉家除了老二廉宝生在空军当飞行员外，还有老三在海军部队服役，是舰艇上的少尉副连长。老四入伍不久，是陆军。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    当时，军人是社会上最崇高的职业，而廉家一下子出了三位，海陆空齐全，是远近闻名的光荣家庭，可以说是人人羡慕嫉妒，或许也有恨。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    用老大廉保忠的话说，「那时红得发紫」。廉保忠虽然没有从军，但也是国家干部，正在支援边远山区，在隆化县当公社书记。隆化县也就是董存瑞牺牲的地方。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    廉宝生被定为叛徒后，这个家庭瞬间就从天上掉到了地上，而且是陷入了如地狱般惨烈的境地。一家人的命运就此改变了。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    廉宝生新婚的妻子听说丈夫叛变死亡，很快就办了离婚手续，改嫁他人。这也是可以理解的，丈夫没了不说，还背了个恶名，放哪个女人身上也扛不住。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    在海军当少尉的老三被提前退伍，转业到宁河县粮食局。最初是在保卫科工作，不久，「文革」爆发，领导害怕他也出事，干脆让他回家务农算了。就此当了一辈子农民，早早去世。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    刚入伍的老四，陆军士兵的新军装还没洗上几水，就被开除军籍，也回家务农。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    老五当时年龄还小，但此后的人生路就窄了。从政、从军是别想了，作为叛徒家庭，政审首先就通不过。最后也是做了一辈子农民。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    老大廉保忠相对要好一些。他本来是副县长的候选人，弟弟出事后自然不可能再提拔，连公社书记的职位也给撤了。但因为他能力强，没有被打发回家，调到生产指挥部去干活，抓农业生产，算是个闲职。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    对「叛徒」的家属，各级组织都得掌握材料，监督使用。「文革」开始后，阶级斗争成为社会之纲，廉家所受的歧视和打击，就更大了。廉保忠的儿子记得很清楚，他小时候上学，有什么活动让贫下中农子弟参加，他家虽然也是贫下中农，但他不能去，因为他有个叛徒叔叔。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    「叛徒」这个词一直是道德上最污的污点，记得在我上学的时候，敢有人随便说你是叛徒，那是要上去拼命的。但是在那个特殊年代，廉家人没办法，只有默默扛着。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    幸亏，廉保忠保留了体制内的职位，能争取到一些资源和机会，担负起了为弟弟平反的责任。他想尽办法，不断写信申诉。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    当时信息不发达，部队来回调整，番号经常变，想要找到廉宝生所在的部队都难。何况国内大搞运动，批来斗去，人心本就惶惶，谁也不愿沾这个事。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    一家人花了十几年的时间，一无所获。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   4
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    事情的转折，出现在
   </span>
   1983
   <span lang="ZH-CN" style="font-family: 宋体;">
    年。这与二李的境况有关。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    李显斌和李才旺到台湾后，升了官，拿了黄金。但，再也不能飞行了。毕竟，他们能飞过来，也可能再飞回去，得防着。那架「伊留申
   </span>
   -28
   <span lang="ZH-CN" style="font-family: 宋体;">
    」轰炸机也被拆卸研究，告别了蓝天。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    岸这边其实也不一样，从台湾「起义」过来的飞行员，一律安排在山西临汾的一个航校里当教官。为什么放到临汾呢？想一想到海边的距离，你就明白了。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    李显斌作为头号英雄，声名显赫一时。在一次老乡会上，他看中了
   </span>
   19
   <span lang="ZH-CN" style="font-family: 宋体;">
    岁的广播员张美云，展开追求。张小姐还在犹豫中，领导前来强行撮合，两人成了婚。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <img alt="80.jpg" border="0" height="489" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/4588/80.jpg" width="360"/>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    此后，他过上了「酒色财气」的生活。连立场都背叛的人，情感上也不可能专一，李显斌很快就出轨了。有英雄称号加持，张美云是不敢怒也不敢言。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    李才旺也和受伤住院时的一个护理员结了婚。因为是被迫投敌的，他更不受信任，生活处处受到监视。妻子后来告诉他实话，她也是派来监视他的。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    为了摆脱这种生活，李才旺一直找机会离开台湾。正好，妻子的妹妹嫁给了一个美国人。他先是托关系于
   </span>
   1975
   <span lang="ZH-CN" style="font-family: 宋体;">
    年退了伍，然后以探亲的名义去了美国，就此定居。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    在美国，他托人联系上了大陆政府，希望能得到原谅。政府同意了。
   </span>
   1983
   <span lang="ZH-CN" style="font-family: 宋体;">
    年底，他回大陆探亲，在机场发表了一则声明，说自己和廉宝生是被李显斌劫持到台湾的，无法制止事件的发生。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    中央人民广播电台播放了这条新闻，恰好被廉保忠的大女婿给听到了，立刻把消息通知全家。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    廉家兴奋极了。但他们没法直接联系李才旺，最后想办法通过外事部门，向李才旺要了一张证明。可惜，证明书很短，没有透露事件的细节。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    廉家没有放弃，继续申诉。
   </span>
   1988
   <span lang="ZH-CN" style="font-family: 宋体;">
    年，他们找到了全国人大常委会委员长万里的秘书，将申诉信递到万里手上。万里批复：「还原历史」，指示空军重新调查。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    就在这一年，李显斌在台湾也出了事。八十年代，台湾经济在混乱中复苏，他把赏金拿去投资民间的非法集资集团，结果赔得血本无归。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    这也是两岸叛逃人员命运不同的地方，来这边的飞行员，很快适应了计划经济，听组织安排，过起了悠哉的生活。而去了对岸的飞行员，却不适应市场经济，盲目投资、炒股，很多人都赔了。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    李显斌责怪是妻子出的馊主意，还闹上法庭，离了婚。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    生活陷入困境后，油腻中年男李显斌想到了要找「组织」。他给台湾立法院写了一份陈情书，说当年想来台湾的只有他一人，所以那
   </span>
   4000
   <span lang="ZH-CN" style="font-family: 宋体;">
    两黄金应该由他一人独得。何况廉宝生死亡，
   </span>
   1000
   <span lang="ZH-CN" style="font-family: 宋体;">
    两黄金事实上没有发放，更应该分给他。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    一个人能无耻到这个地步，会叛变也就不奇怪了。立法院驳回了他的申请。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    此时，大陆的空军经过反复论证，也确定廉宝生没有叛逃的动机和证据。
   </span>
   1990
   <span lang="ZH-CN" style="font-family: 宋体;">
    年，空军决定撤销对廉宝生的「叛徒」定性，更改为「因公牺牲」。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    廉家人不同意，因为「因公牺牲」和「烈士」的意义完全不一样。廉宝生在紧要关头自杀的行为如果不被定位烈士，实在太冤枉。他们又提起申诉。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    人生如一个
   </span>
   8
   <span lang="ZH-CN" style="font-family: 宋体;">
    字，循环轮回，兜兜转转，最后还是回到原点。这事的解决，还是落到了李显斌的头上。
   </span>
   1991
   <span lang="ZH-CN" style="font-family: 宋体;">
    年，他在山东的母亲病重。他认为，过去这么多年了，大陆应该不会追究，便从加拿大起飞，偷偷回来探亲。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    他想得太天真了，以为自由可以有，想有就能有。看完母亲，当他准备飞回去时，在机场被捕。最终以「投敌叛变罪」，判处有期徒刑
   </span>
   15
   <span lang="ZH-CN" style="font-family: 宋体;">
    年。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    伟大领袖说过一句话，凡是有人群的地方，就有左中右。一架飞机上的三个人，做了三个不同的选择，造就了三种不同的人生。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    在审讯中，李显斌交代了当年叛逃的细节：廉宝生的确是自杀。他用随身的佩枪，对准自己的太阳穴开了一枪。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    有了这样的关键证据，
   </span>
   1993
   <span lang="ZH-CN" style="font-family: 宋体;">
    年，廉宝生被追认为革命烈士。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <img alt="81.jpg" border="0" height="318" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/4588/81.jpg" width="540"/>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   5
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    国共双方用重金悬赏飞行员的行为，在上世纪八十年代后期不约而同地停止。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    总体说起来是因为台海局势缓和，但一个重要原因是，重赏来投奔的人员，会极大地损伤自己飞行员的士气。你想呀，飞来一个对岸的，一下子就给了相当于我几十年的工资，我还不如飞过去算了。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    幸运的是，廉宝生的老母亲在生前知道了儿子平反的消息。
   </span>
   28
   <span lang="ZH-CN" style="font-family: 宋体;">
    年过去，真相终于到来，但儿子是再也见不到了。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   1998
   <span lang="ZH-CN" style="font-family: 宋体;">
    年，廉母去世，留下遗嘱要把儿子的骨灰找回来。廉家又开始了漫长的寻找廉宝生墓地的历程。因为台海关系时好时坏，时间又过去太久，无从下手。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    就在廉家人寻找廉宝生的墓地时，殊不知，台湾也有人正在寻找他们。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    这个人叫高兴华，是台湾的一名中学老师。
   </span>
   1965
   <span lang="ZH-CN" style="font-family: 宋体;">
    年双十一那天，他才
   </span>
   6
   <span lang="ZH-CN" style="font-family: 宋体;">
    岁，正在桃园机场附近的学校读书，记得当天机场在放鞭炮。一打听，原来是“庆祝一架共军飞机投诚”。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    成年后，他对这件事很感兴趣，去访问了当年在机场值班、曾第一时间冲到飞机前的几位军人，询问了事情的发生过程。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    台湾军方当时就知道廉宝生是自杀的，但没有曝光。你们想呀，如果说出来廉宝生宁可自杀也不愿意投降，等于在褒奖解放军有血性，是替海峡那边树了一个宣传的典型，这该多么尴尬呀。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    虽然被封为「反共义士」，台湾并没有将他入祀「忠烈祠」或是碧潭的空军公墓，而只是葬在桃园公墓，也没人维护。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    不过，蒋经国这个人也算厚道，出于对一个血性军人的尊重，他写了「尚义成仁」四个字。我想，这句题词，即使放在大陆这边颁给廉宝生，也能说得通。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    廉宝生的故事，让高兴华很感动，他想应该把廉宝生的骨灰送回故里。
   </span>
   2016
   <span lang="ZH-CN" style="font-family: 宋体;">
    年，他和另外一位台湾空军退伍军人李刚，在脸书上发了一条寻找廉宝生家人的消息。经过网友的传递帮忙，终于和廉家联系上了。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    然后，李刚又通过自己的关系找到了冯世宽－－他服役时的老长官，让军方为廉宝生开具了一张死亡证书。在这份证书上，廉宝生仍被称为「反共义士」。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   2016
   <span lang="ZH-CN" style="font-family: 宋体;">
    年
   </span>
   9
   <span lang="ZH-CN" style="font-family: 宋体;">
    月
   </span>
   28
   <span lang="ZH-CN" style="font-family: 宋体;">
    日，廉宝生的骨灰回到故乡。两天后的
   </span>
   30
   <span lang="ZH-CN" style="font-family: 宋体;">
    日是「烈士纪念日」，骨灰被安葬于宁河区的烈士墓园。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    我时常在想，假如廉宝生活着，又会是什么结果呢？
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    在他死后不到一年，「文革」爆发。林副统帅主抓下的空军部队，创造了「用毛泽东思想占领天空，用毛主席语录指挥飞行」的工作方法。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    飞机的机身全部刷上毛主席语录，飞行员在和塔台联络时，首先要呼喊毛主席语录和革命口号。于是就出现了这样的场景：
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    ——起飞，沿着主席革命路线前进！
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    ——用毛泽东思想占领天空，前进！
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    ——为人民服务，我需要爬高
   </span>
   300
   <span lang="ZH-CN" style="font-family: 宋体;">
    米！
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    ——为人民服务，你可以爬高！
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    发动机在点火时，以前机械师要发口令「点火了」。成功了就回答「成功了」，不成功就回答「不成功」。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    到了文革时，把「点火了」改成「打倒刘少奇」。如果成功了，就回答「打倒了」。如果没成功，就说「继续打倒刘少奇」。有一次，一个新兵在点火时总不成功，他一紧张，便喊了声：「打不倒刘少奇」。结果成了政治事故，受到严重处分。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    飞行是一件非常严密的事情，每一秒钟都很宝贵，口令要求简短精炼。加上语录，耽搁了时间，很容易造成飞行事故。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    还好，这时期的飞行员本来就很少升空，因为他们要把大量时间用来学习政治理论。据统计，文革时期，飞行员的军事训练和政治学习时间比是
   </span>
   1
   <span lang="ZH-CN" style="font-family: 宋体;">
    ：
   </span>
   5
   <span lang="ZH-CN" style="font-family: 宋体;">
    。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    在这种形势下，就会理解有些人为什么会被国民党的糖衣炮弹给击中了，飞到海峡那边去。但廉宝生估计不会，因为他还得组织大家学习理论。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   6
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    在国共对峙的漫长历史中，廉宝生是唯一一位被双方都评为「烈士」的解放军军人。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    除了孙中山外，国共双方同时给予认可的人，不多。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    但，即使现在真相大白，他已魂兮归来，官方也不好大力宣传。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>
  </o:p>
 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    去年
   </span>
   4
   <span lang="ZH-CN" style="font-family: 宋体;">
    月，廉宝生被评为天津宁河区第三届「身边感动」十大新闻人物之一。但，只是由区电视台做了简单报道，列举了他的名字，并没有详细讲述背后的事情。天津一如既往地没有新闻。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>
   <font size="3">
   </font>

 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    我是很能理解的。不管廉宝生个人多么壮烈，毕竟是叛逃事件中的人，越宣传，越等于给自己抹黑。岸这边向来以光明使者自居，不断宣扬别人弃暗投明的例子，殊不知还有自己的人偏偏不要光明。这就尴尬了。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>

   </font>

 </p>
 <p class="MsoNormal">
  <font size="3">
   <span lang="ZH-CN" style="font-family: 宋体;">
    每个时代都有自己的牺牲者。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">
  <o:p>

 <p class="MsoNormal">

   <span lang="ZH-CN" style="font-family: 宋体;">
    在廉宝生的墓前，我献上一束花，愿他的灵魂安息。
   </span>
   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">

 <p class="MsoNormal">

   <span lang="ZH-CN" style="font-family: 宋体;">

   <o:p>
   </o:p>
  </font>
 </p>
 <p class="MsoNormal">

 <p class="MsoNormal">
