<p class="MsoNormal">
<b>
<span lang="ZH-CN" style="font-family: 宋体;">
<font size="5">
     抗战时期农村的童年记忆
    </font>
</span>
<font size="4">
<o:p>
</o:p>
</font>
</b>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
<b>
<font size="4">
     －－作者：莲花伯
    </font>
</b>
</span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   抗战，距今已七十余载，亲身经历者已不多。此文是将个人经历事情的回忆和根据党史书的框架，对当时老百姓的日常社会生活做白描式的叙述，不是从民族矛盾和阶级斗争角度进行取材和理论分析。著文的目的是提供一些真实的具体的社会历史材料
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   ，澄清当前史学研究和评论中一些误解，尤其是针对一些影视剧中对抗战情节的胡编乱造，说明历史的真相。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   我农村的家乡在冀东路南区的陡河之岸，频临渤海，土地肥沃，人口密集，蕴藏着深厚的传统文化。在抗战时期，属于敌占区也是游击区，是两面政权。人民群众即老百姓的社会生活活动，总的说遵循的还是传统文化中的道德准则和为人之道，
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   共产党在当地的活动也是如此，是与人民群众的生活准则交织融合在一起的。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   抗战时的沦陷区的社会是所谓的旧社会，有人称其为万恶的旧社会，“旧社会将人变成鬼，新社会将鬼变成人”
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   是其写照。实际情况并非如此，它是社会发展历史过程中一个阶段。人们在这样社会环境中满足生存，享受和发展的需要。我们首先介绍社会生活状
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   ，然后介绍与之相关的抗战的一些事情。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   真实的社会状况
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   当时的农村中，保留着传统文化中所谓封建的社会制度，道德观念和行为规范。具体表现在人们的社会活动，社会关系和思想意识之中。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   社会关系种类众多，分类各异。一般是采用按照性质的划分，如亲情关系，即夫妻关系
  </span>
  ;
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   血缘关系，即父母与子女的关系；兄弟姐妹的关系，堂兄弟姐妹的关系，伯叔侄关系等等
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   。亲戚关系，即由亲情关系和血缘关系结合派生出来的关系，如姑侄，姨甥，进而按照远近派生出众多的表亲…此外还有其他类型的关系。这些关系的观念与准则表现在社会生活活动的方面面。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   辈分划分和交往中的称呼
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   家庭成员和亲戚关系之间有严格的辈分划分。每个辈分成员在相互关系中都有相应的行为规范，如孝，是子女对父母的关系的首要准则，儿媳对婆母的绝对服从。悌是兄弟之间的关系准则。
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   称呼是说话者在交往中对他人不是指名道姓而是按照社会关系的指称。人们之间的称呼是由血缘关系，亲情关系，亲戚关系，友情关系以及职务关系等决定的。如子女称自己的父母为爸爸和妈妈，孙子孙女称自己的祖父母为爷爷奶奶……与个人没有亲情和友情关系的人彼此的称呼就非常复杂。涉及场合，年龄，性别，关系，职业，感情
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   等等。如父母是标准语，口语中则叫爸妈或爹妈。大妈，大娘，大姐，大嫂，大爷，大叔，可泛指不同性别和年龄的人尊称。人称代词用“你老”代替“你”。不用您。你老，他老表示尊敬礼貌，不能说我老。有表示说话者与对话者共同的人称代词咱，咱们。用名称代替人称，妈妈
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   我
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   对你说。以第三者的身份的关系称呼对方，他哥哥，他可指任何的第三者，如儿子的哥哥，妹妹的哥哥，丈夫的哥哥。辈分词很多具有多义性
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   ，
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   如大哥，可以是同族兄弟中弟妹对排行第一的兄长的称呼，在社会交往中则指比自己年龄大的男子的尊称，群体中为首者。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   男女不平等
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   重男轻女的范围很广泛。男性是家族继承者，女儿没有家产的继承权，女儿未婚早逝不得入祖坟。女儿在家一般没有大名
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   学名
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   只有小名，嫁入夫家后属于夫家的人，与一些西方民族相同，改为夫家的性，如张董氏，张夫人。过继只限兄弟之间的儿子。妻室早逝，丈夫的续弦继承前房的一切社会关系，后夫不行。入赘须改性，
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   养子有继承权。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   婚姻制度
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   被称作封建婚姻，讲究门当户对，父母之命，媒妁之言，男女授受不亲，讲究贞节和从一而终等。婚龄一般是十六七岁，通行女大于男一两岁。这样的婚姻制度是由当时的社会条件决定的，如人均预期寿命短，大致不到四十岁，要完成传宗代的任务必须早婚。自由恋爱成婚不可能，十二、三岁不到恋爱年龄。当时的农村闭塞，青年男女没有交往的机会。童养媳，听说以前有过，三、四十年代没有了。只是有的家庭因为战乱将未成年的女儿提早送入夫家。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   尽管没有法律的明文规定，盛行的是一夫一妻制。娶小
  </span>
  (
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   妾
  </span>
  )
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，有两种情况，一是妻子不生育，娶小用于生儿育女，较为多见。二是在外地事业有成的男人纳小做外室，老家的妻子管理家业和照顾子女与老人。有一种不多见得现象，兄弟二人膝下只有一子，娶两个儿媳不分大小，分属两家，看来这种婚姻主要是生育后代的问题
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   。女儿外嫁有陪嫁，也要彩礼。小时候没听说乡里有离婚的。妇女丈夫死后，即使没有子女，一般也不再嫁，仍是夫家成员。如果无以为生则携子女改嫁他人。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   离婚普及是在解放以后，最初主要是抗战或解放战争时期参加革命的人。入城后以反对封建婚姻为借口，换老婆。或是高级干部在革命队伍中又结婚，以一夫一妻制为由，与农村老家的妻子离婚。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   贫富观念
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   贫富主要是按照所拥有的财产的划分，不是生活水平，当时农村吃穿的生活水平不高，讲究勤俭持家。在一般人的心目中，贫富差别是客观的存在，历来如此。人们追求的是富，以富为荣。如何致富。普遍认为是勤劳节俭
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   。当地的富户多是经商，从军，耍手艺，做官等致的富，不是靠多种田。有一突出特点是贫富的流动性，贫富不是永恒的，不时地在转换。流传“富不过三代”的格言，的确如此。因素有二，一是出败家子，二是分家
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   ，家产由儿子均分，如果是多子家庭，两三代人就可把家产分得人均难以为富
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   。贫富差别是人的生产劳动积极性的刺激因素和社会发展的动力。富与文化相关，富人一般文化教育水平较高。
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   当地最早接受共产主义的差不多都出身富家的知识分子。带头起来来组织队伍抗日的也多是士绅富人，参加伪军的多为贫家子弟。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   对盗匪贼和乞丐的看法
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   当前文史著作中的主流观点认为，盗匪贼和乞丐是封建社会统治阶级的剥削与压迫造成的
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，盗匪是官逼民反，是对统治阶级的反抗与斗争
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   。那时乡里的人并不这样认为。对盗匪是憎恶的，他们杀人放火抢夺财物，没有人性。没听说当地真有过劫富济贫，打抱不平的侠盗义匪，那是评书说唱中的故事。至于贼虽然不打杀他人，但他们也是毫无同情心，不论对何人能偷就偷，即所谓贼偷方便。有一种被称作小偷小摸的人，经常遇到和听说，很典型。他们不是以偷为生，而是一种嗜好，如同赌博一样，官无法追究，群众也无可奈何。至于乞丐，一般年景下，本地人的乞丐很少见，我们村子里没有以乞讨为生的人，偶见外地人路过行乞，人们可怜他们，尽量给以施舍。人们对乡里人行乞不同情，是懒人不劳动，随便干点活，就能能有吃的。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   信仰
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在思想意识方面的信仰是多元的，有宗教，也有迷信和风俗习惯。信仰特点是具有包容性和融合性。与西方不同，宗教之间是不斗争的，宗教徒之间也是如此，老百姓也没有将他们对立起来。遇到丧事时，往往同时请和尚和道士来念经超度，分站两排，各念各的经。在生活中的各种准则和信条也是不分来源的。比如我家同时供奉的有道教
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   的张三爷，佛教的观音大士。至于像尉迟敬德的门神，八月十五敬奉的兔儿爷是来源于神话故事。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   包括宗教在内的许多与命运相关的人生信条，用来指导自己的言行。如好人有好报，与人方便自己方便，人之一切命中注定，恻隐之心人皆有之，把人的命运都寄托在老天爷的恩赐上。遇到惊恐之事，烧香磕头，求老天爷保佑，老天爷到底是谁在哪儿谁也不清楚
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   阶级意识
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   那时农村里的人，基本上没什么阶级斗争，剥削等社会意识，
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   那是在抗战胜利以后，反对蒋介石，尤其是土地改革时提出来的，一直持续到文化大革命
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   。近代文学与影视作品立足于宣传教育而夸大了剥削的程度和残酷性
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，与家乡的事实不符。在我们的家乡地富同佃农和雇农的关系一般是协调的，这是由当时的社会生产力的性质决定的。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在经济方面，佃农用租金租用地富的土地或是将收获按比例分成，
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   地富还要担负各种税务。雇工与雇主的合同口头协商制定。雇主负责雇工吃住，预付全年工资，如有意外工资不退。工资的数量同一般小学教师差不多。总的讲，雇工的收入不少于供养两三口人之家的需要。这是社会规定性决定的，因为这样才能维持劳动力的延续，否则作为社会的一个阶层的雇工就会停止存在，社会生产就会停顿。雇主与雇工之间的关系是互助合作的关系，
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   因为是市场经济，两者由供求关系来平衡。雇工不等于农奴，是自由人，农业劳动建立在自觉的基础上，不能对立也不能相互斗争，否则就没有劳动效率。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   这种社会意识，在在乡亲和亲友之间的关系中也表现出来，在抗日时期，在一个村子里，一户人家的亲友中，甚至一个家庭里，有当八路的，也有当汉奸和伪军的，还可能是国民党的人。在村子里，无论是何方的家属，彼此之间不会因此而相互仇视，仍然是乡里乡亲，守望相助。
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   村里的村长，保长，对乡里人遇到事情时，也不是落井下石，而是尽力保护。难道当时的农村就是这么风平浪静，没有矛盾和冲突嘛。不是的，矛盾和冲突是社会的常态。在那里，一般发生在家庭内部，邻里之间，族群之间，是利益之争，名誉之争，不是政治之争。没有听说过，两个家庭由于子女在敌对方而发生冲突。双方家庭之间甚至还会彼此照应，互通消息。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   八路军
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   八路军最初是军队的编号，后来范围扩大成了一个政权的名称。不仅指作战的部队，还包括敌后抗日根据地的地方政权的党政机关和工作人员，
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   后者的工作不是打仗，主要是宣传教育
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，税收，动员群众参军，支前，为部队提供粮食和安排住宿，收集各种信息……
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   将两者的工作性质区别开来很重要，否则会出现误解。如作战部队会不时地进行转移，而地方干部必须坚持在自己工作的地区。地方干部在敌人讨伐和清乡时必须躲开
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   ，而不是同敌人交火。有的评论者说这是游而不击。在现代的影视剧中，几乎把抗日与打仗等同起来。把八路军的地方党政人员都当做战斗员，不能见敌人就跑，
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   而是奋不顾身，与敌人枪战，击毙数个敌人，冲出重围，或是英勇牺牲。实际上党政干部用短枪是打不了仗的，应该转移。或是把枪藏起来，化妆成农民
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   ，躲过敌人的搜查。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   军队本身的职能也不相同，像区级武装队伍区小队，属于军队的编制，相当于一个排，二、三十人。主要是配合区干部做区的政治和行政工作，如打击土匪、盗贼、为非作歹的警察特务、汉奸以及交通联络，护送过往同志等。县有县大队
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   一般百十人，相当一个连，主要是维护地区的社会治安，间有小的作战任务和配合大部队作战。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   主要用于作战的队伍
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   是隶属于军分区或军区的队伍，当时主要以团为单位
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   。被称之为大部队。在当时的路南区，很少有大部队活动，因为路南区的自然环境特点
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   是一马平川，不能设伏突击。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   组织一次大战斗是要有一系列条件，评价一次交火的得失要从多方面进行估计
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   。不能只以消灭敌人的数量为标准，还要看自己的军队的损失，战斗中缴获多少武器弹药。
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   据统计缴获一挺轻机枪，要通过牺牲一二百战士的生命的大的战斗。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   敌强我弱
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   从军力方面。不能只看人数，还要看士兵的素质和武器装备。如区小队的武器主要的汉阳造，少量的七九和三八，刺刀都配不齐，自造手榴弹人均两三颗，关键是步枪子弹，每人只有两三发，最多的是五发子弹。全队共有百、八十发子弹。日军步兵
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   是清一色的三八大盖
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   出发时人均配有三百发子弹。一个步兵班的子弹有三、四千发。三八大盖
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   的性能远比汉阳造强的多。一百发子弹的队伍和三千发子弹的队伍是无法对垒的。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   由此可见，说八路军如游而不击，不打大战，其实不是不打而是打不起，没有那么多枪支弹药。主要根据地有设备简陋的兵工厂能生产子弹和手榴弹、一般的游击区没有军工厂。
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   军队的武器弹药靠的是战斗的缴获和和用根据地的物资向伪军交换或是购买。伪军除了领薪水外，还卖弹药做外快。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   敌人的报复
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   还要强调的是，即使是小的伏击，也还要考虑到群众的损失。扑杀个别作恶多端的伪军警察特务，汉奸，关系不大。打杀日本人则要慎重安排地点和机会。日本人是要报复的，
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   军队打完仗就撤离了。老百姓离不开自己的家园
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，日本人就会烧杀众多的老百姓，冀东的一些大的惨案，皆是如此。以冀东的潘家峪惨案来说，日军讨伐队包围整个村庄，除了少数人幸免以外，全村被杀光。原因很简单，哪里是八路军的集中地，在那里打过埋伏杀死过日本人。这就是当地群众付出的代价。只是我们不予报道。只说敌人的残酷。何况也说不清楚。这也是日寇策略，用此种手段对付八路军。因此军事领导在决定战斗时，
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   必须考虑到老百姓
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   ，否则老百姓就会说，我们帮你们打日本人。当日本人来杀我们的时候。你们却不保护我们，你们到底是为谁打鬼子
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   。
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   老百姓的艰辛与无奈
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   在抗战期间，在敌占区和游击区的农村里的普通老百姓的处境很难，他们是处于层层叠叠的无奈之中。首先他们要承受个人和家庭生存的压力，无论在何种情况下，都要从事生产劳动，或者外出挣钱养家，于是有的人为日本人做事，当伪军。至于给日本人交税，交粮，出劳工，虽非人之所愿，但也只能是为之。有的自命为是气节高尚的人，把这类现象看作是中国人懦弱，没有骨气。尤其是把中国人面对日本人抢杀中国人时的麻木状态，大发雷霆。这种情况下，应该怎么办？难道能像文革期间歌剧《刘胡兰》中群众用挺起的胸膛逼退敌人的刺刀那样应对
  </span>
  ?
  <o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   对日军研究中的空白
  </span>
<o:p>
</o:p>
</p>
<p class="MsoNormal">
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   对日寇和伪军的请剿和清乡，文艺作品和影视剧中的描写，几乎是一个模式，
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   日寇士兵的惨无人道烧杀抢掠，伪军的为虎作伥
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   ，老百姓的怒目圆睁
  </span>
<span lang="ZH-CN">
</span>
<span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
