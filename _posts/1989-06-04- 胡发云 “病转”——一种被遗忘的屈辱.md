  </o:p>

  </span>

   的事件。它不是一出气贯长虹辉煌磅礴的正剧，也不是一出惨烈沉痛凄切哀婉的悲剧，甚至这一出活剧的真正主人公们并未出场，他们将千百万
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   孩子们送上广阔天地这个大舞台之后，自己依然留在宫廷与都市，继续着他们成年人的政治游戏。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   “一桩事件，哪怕再重大，若已说透，便不会常挂在嘴边记在心头，整理一番束之经史高阁，也就是说，已完成了它的叙说。即便要纪念要庆
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   祝，也只是一种仪式了。上山下乡（以及与其紧密嵌接粘连的“文革”）显然还未说透，它在二十世纪中国的历史上，究竟生成了一些什么样的意
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   义？对二十世纪末页的中国及中国人产生了哪样一些影响？大约还有许多值得思考、值得评说的地方。”（《泥泞与随想》，新创作，
  </span>

  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>

  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   。）
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   “病转”，是一部知青史中最灰暗最屈辱也最残酷的一页，它没有上山下乡初期战天斗地的热情与豪迈，也没有“文革”结束后集体返城的悲
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   壮与激越，“病转”的人们，只有一副伤残病痛的身体，一颗孤独绝望的心和一摞一摞病历、化验单、诊断证明书以及从下到上各级有关部门的证
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   明材料－－如本书中说的，大大小小十一个章子……以上那些，有的是真的，有的是假的，有的是将假的终于做成了真的－－许多知青后来不得不
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   用自伤自残的方式，来换取一张回城的通行证。本书中，林菁菁在一次又一次的检查中，为终于出现了病症而激动的时候，我们看到生命与尊严
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   已坠落到了地狱的门槛前。所以，几乎所有艰难地进行“病转”的人，几乎都象鼹鼠一样，悄没声息又忍辱负重运作着它那漫长又繁难的程序，有
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   幸终于成功以后，又象鼹鼠一样悄没声息在这本原就是属于自己的城市蜷伏下来——待业
  </span>
  ,
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   打零工
  </span>
  ,
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   或努力进入那些街道小厂……我见到为招工返城而
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   摆酒设宴的，也见到为上学返城而奔走相告的，但我从未见到为“病转”返城而庆贺的－－那些必须运用“病转”才得以回城的人，基本上都是那个
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   时代的贱民，在乡下是，回城以后依然是。在很长的时间里，很少读到写“病转”的文字。大约这是一个极深的伤痛，经历过它的人害怕再去戳它
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   。有一位女性告诉我，回城许多年后她还会作恶梦，梦见有人拿一张表给她填，说又要下农村了。她哭着央求说，她已经结婚了，已经有孩子了
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   ，那人冷冷地说，有孩子也要下。她从梦中哭醒，满面泪痕，那恐惧却久久挥之不去。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   老三届一代人，一直被亲昵称呼为“祖国的花朵”“共和国的儿女”。确实，这一代人，不论他们的家庭背景，生活环境，个人禀性是如何的不同
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   ，他们都是在新中国这所大学校的严密管教中和铺天盖地的革命文化熏陶中诞生与成长起来的。本来，他们中的绝大多数人都会成为最坚定的“革
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   命接班人”。忽然有一天，生活告诉他们，在他们中间，一部分人和另一部分人是不一样的，甚至是水火不相容的，他们有的是天生的高贵者，有
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   的是天生的卑贱者，有的血是红的
  </span>
  ,
  <span lang="ZH-CN" style='font-family:
宋体;mso-ascii-font-family:"Times New Roman"'>
   心是红的
  </span>
  ,
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   有的血是黑的心也是黑的……尽管有“文革”以前升学、入团中的“阶级路线”
  </span>
  ,
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   尽管有“文革”初期的“红
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   色恐怖”，但最彻底地向他们宣示了这一天条的，便是招工。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   招工不仅是红色中国建政以来最直观最大规模的一次阶级歧视，也是红色中国建政以来最直观最大规模的一次性别歧视。在几次招工浪潮
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   过后，广袤的乡下，那些破败、孤独、空荡荡的知青屋中，剩下的几乎全是“家庭有问题”女知青，她们以柔弱之身，负载着那颗伤痕累累之心及
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   沉重得望不到头的劳作，苦熬着绝望的日子。如果说，在往日的痛苦中，她们多少还能得到家的护卫和基本的生活保障，在被抛弃于乡野的日子
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   中，她们连最基本的生存都受到威胁，更不消说那巨大精神苦痛。不少的人因此自杀或发疯。在《病转》这本书中，我们可以随处读到她们。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   这是一次全社会的卑劣的合谋。是我们民族永远不可抹去的耻辱的印记。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   文革
  </span>
  ,
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   下乡
  </span>
  ,
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   回城这三部曲，让一代“共和国的儿女”经历了一次巨大的精神炼狱，完成了一次巨大的精神转变－－不论他们曾经是红色的，还是
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   黑色的，高贵的，还是卑贱的。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   我在上面说到的那篇文章中还这样写过，上山下乡，让整整一代人改变了自己的生活方式，同时也改变了自己的精神方式……作为一九六八
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   年的全国规模的上山下乡运动，显然不是一个准备充分思虑成熟的部署，而是
  </span>
  '
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   文革
  </span>
  '
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   失控后的一种临时应变措施。它所产生的一系列或隐或显的
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   后果，也是人们始料未及的。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   在各种各样的知青类群中，《病转》写了长期以来被掩盖了的特殊的一群，它丰富补充一代知青的群像。她们是孤独的，卑微的，常年流落于城
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   乡之间，隔离于社会之外。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   《病转》没有曲折复杂惊心动魄故事，在非常平实朴素的人事中，却隐含着曲折复杂惊心动魄生活原生相。《病转》也没有华丽奇俏的辞藻
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   和新潮前卫的技巧，却处处弥漫着那个已逝去时代的灰暗沉郁的气息，那时的语言，那时的衣饰，那时的吃食，那时的社会生活氛围－－特别是
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   那些灰色人群的社会生活氛围，还有那些两毛钱买一斤粮票、五斤粮票换一斤肉票、一斤肉票买一斤半排骨的故事，让人觉得恍若隔世又记忆犹
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   新……在某种意义上说，我们还可以将这本书当作一部文革中期中国社会民俗读物来读。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   一眨眼，又是沧海桑田了。当初费尽心机忍辱负重苦苦“病转”林菁菁、洪接锋、占斯琴、先梅、应笙、可可、王珏王小珏们生活大约已发生很
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   大变化，大约早已为人妻为人母了，操劳一些的大约已生出星星华发。当初大权在握，执掌着她们命运的那些招工者，或许已退休，或许已下岗
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   ，其中某人可能就在咱们都市一角靠摆地摊聊以度日，那些同情过或冷漠过林菁菁们的医生们呢？大约也今非昔比了。过往的岁月已经远去，但
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   它不会消失。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   本书的作者高志远是我的校友，插队也在一个县－－这些，都是三十年以后、也就是这部书稿写完之后我才知道的，因此，关于她我了解
