  </o:p>

 </p>
 <p class="MsoNormal">
  <img alt="62.jpg" border="0" height="329" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3449/62.jpg" width="531"/>
  <o:p>

 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   向死而生，以死为大，尊重死者乃文明的核心内容。掘墓辱尸，乃历史上极为恶毒一招。无深仇大恨者，不至于出此狠策。而出此策者，史册多不齿。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   据《史记·伍子胥列传》载：伍子胥之父兄为楚平王所害，其发誓必倾覆楚国，以报杀亲之仇。果然事于他愿，其借吴军击败楚军主力于柏举，并展开追击，长驱攻入楚都郢，终成破楚之功。时楚平王已死，伍子胥“乃掘楚平王墓，出其尸，鞭之三百，然后已”。鞭平王之后，此类事件便很少了。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   据冯自由《革命逸史》载：李鸿章任两广总督时，“一日清晨，众聚内花厅谒李督闲谈，李子经芳侍焉。忽来廷寄上谕一电，令将康有为、梁启超二人祖墓刬毁。李督阅后，问诸幕僚应如何办理？众俱唯唯莫敢置答，独少川起言曰：‘罪人不孥，古有明训。若株连及于党人祖坟，似太残忍，尤碍各国观瞻。还请傅相三思。’”李鸿章遂作罢，卒以发掘一二无主荒坟虚报塞责。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   明末，朝廷看李自成渐成气候，大有取而代之之势，想必其祖坟上生有龙脉，便于崇祯十五年（
  </span>

   日，因批判《武训传》需要，冠县红卫兵砸开千古义丐武训的墓，掘出其遗骨，抬去游街，当众批判后焚烧成灰。也是这年，正当批判《海瑞罢官》之时，造反派到海瑞墓园“破四旧”，扒土掘坟。墓里只有几绺头发、几颗牙齿、几根残骨和几枚铜钱。那些发齿残骨，被放到一个盒子里游街示众，最后焚烧扬灰。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="66.jpg" border="0" height="184" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3449/66.jpg" width="250"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   清官不能避祸，民族英雄也不能幸免。杭州岳飞墓也被红卫兵夷平，并焚弃了遗骨。理由是岳飞镇压过农民起义，是军阀。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   帝王墓被毁者，
  </span>
  <span lang="ZH-CN">
  </span>
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   运城舜帝陵被毁，墓冢挂上了大喇叭。炎帝陵主殿及其附属建筑皆毁，刨挖陵墓内存物抢夺一空，墓丘被夷为平地。万历皇帝及后妃遗骸，其自定陵里取出，骷髅吊挂于树上，然后与皇后骨骸一同焚毁。安阳县明赵简王朱高燧的墓也被挖毁。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   名臣墓被毁者，有合肥包拯墓，有江陵张居正的墓，有杭州于谦墓。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   名将墓被毁者，有霍去病的霍陵，除香烛签筒被打烂之外，霍去病的塑像也毁于一旦。明末名臣何腾蛟墓墓被掘，祠堂也毁。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   文人墓遭毁者，有王羲之墓，占地二十亩的金庭观几被平毁。有蒲松龄墓，掘墓后发现墓中陪葬除手中一管旱烟筒、头下一迭书外，只有四枚私章，他们对蒲氏私章不屑一顾，遂其遗骨捣毁，弃之于野。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  1932
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   年，徐志摩落葬海宁东山。胡适题写了“诗人徐志摩之墓”。
  </span>
  1966
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年，红卫兵炸毁诗人墓，拖出棺木，散落骸骨。几年后，原址上建起了化肥厂。无锡阿炳墓毁于
  </span>
  1966
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年秋，
  </span>
  1974
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年平整土地，阿炳墓被彻底扒平，后来建工厂，又惨遭焚烧，焚烧后推入旁边的河中。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   抗战时，日军欲开挖位于西湖南屏山荔枝峰下的章太炎墓。一日本军官得悉后喝退士兵，并祭奠一番，此次日军再不扰。
  </span>
  1966
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年底，红卫兵将太炎墓夷为平地，把遗体曝于荒野，棺材也被劫走。周建人见状，老泪纵横道：“我相信，我们的民族是会好起来的。”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   蒋介石乃大敌，蒋虽逃离大陆，但蒋家祖坟仍在，于是位于奉化溪口的蒋介石生母墓，被上海大学生领导的宁波中学生掘开，其遗骸被弃树林。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   葬于上海万国公墓的宋庆龄父母，因是蒋介石的岳父母，被红卫兵连棺木都翻捣出来，且扬尸抛骨。宋庆龄闻讯落泪，据《宋庆龄传》载：“墓地遭破坏的照片从上海寄到北京时，工作人员第一次看到她精神上支持不住而痛哭起来。”急忙给周恩来写信请求保护。后来虽修缮了宋氏墓地，但原先墓碑上宋庆龄兄弟姐妹名字全被删掉，只剩下宋庆龄一名。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   赵登禹将军为抗日殉国的第一位师长。
  </span>
  1937
  <span lang="ZH-CN" style="font-family:
宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:
宋体;mso-fareast-theme-font:minor-fareast">
   年
  </span>
  7
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   月
  </span>
  28
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   日，将军在宛平南苑与倭血战而死，时年三十九岁，龙泉寺方丈冒死寻得将军遗体，将灵柩匿藏寺内八年。
  </span>
  1946
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年
  </span>
  7
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   月，灵柩葬于卢沟桥畔。“文革”间，将军墓被红卫兵掘翻，内无陪葬，遗骨抛掷于野。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <img alt="67.jpg" border="0" height="399" src="http://mjlsh.usc.cuhk.edu.hk/medias/contents/3449/67.jpg" width="578"/>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   将军遭毁，士兵墓也不放过。抗战时期，中国军队死亡

  148
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;
mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   万人，国民政府纪念之，在衡山建忠烈祠，安葬阵亡将士的遗骸。
  </span>
  1943
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年
  </span>
  7
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   月
  </span>
  7
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   日，在南岳忠烈祠隆重的落成大典上，主祭者薛岳将军：“抗战还在进行，牺牲在所难免。这总神位，特为今后为国捐躯忠灵而设，千秋万代，血食无替。”
  </span>
  1944
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年日军陷衡山，忠烈祠被破坏。
  </span>
  1953
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年，忠烈祠的碑文以“反动遗迹”被凿，一字不留。

  1966
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:
minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:minor-fareast">
   年，所有墓葬都被挖毁，骨灰弃之荒野，无一幸免。
  </span>

 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:

minor-fareast">
   其实，这一掘墓行为自“文革”前已相当普遍。孙殿英于清东陵盗掘乾隆和慈禧两墓，而
  </span>

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

  </span>
  1946
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:

  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">

  </span>

  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:Calibri;mso-ascii-theme-font:minor-latin;

  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   被辱者未辱，辱人者反被辱。这是一个时代的耻辱，一代人的耻辱。鲁迅曾言：“不可救药的民族中，一定有许多英雄，专向孩子们瞪眼。这些孱头们！”若将“专向孩子们瞪眼”改成“专向死人们泄愤”，确是很恰当的。那是个所谓的英雄时代，“一定有许多英雄”，英雄们在领袖的号召下，其行为多半是出格的壮举，行为出格的时代，定是荒唐的时代，中外皆然。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style="font-family:宋体;mso-ascii-font-family:
Calibri;mso-ascii-theme-font:minor-latin;mso-fareast-font-family:宋体;mso-fareast-theme-font:
minor-fareast">
   陈云曾言：“把权交给我们的子女，他们至少不会掘我们的祖坟。”其言至少有两重含义：一则掘墓荒唐，自己虽然就是掘墓人；一则制止掘墓的途径，是世袭政权。其出发点仍不在人性的穴位，没有人性这一根本，掘墓的恶性，依就会循环下去。
