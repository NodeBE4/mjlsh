  </span>

  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

  </span>

 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

   山东范县：“共产主义新乐园”
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:

  </span>
  1958
  <span lang="ZH-CN" style='font-family:宋体;

  </span>
  (
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   今属河南
  </span>
  )
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   。寿张出名后，引起了中央领导人的关注。这年
  </span>
  7
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   月中旬，刘少奇前往山东视察。
  </span>
  7
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   月
  </span>
  15
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   日，刘少奇在济南同寿张县委书记及该县部分乡社干部进行了座谈，了解有关情况。
  </span>
  7
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   月
  </span>
  17
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   日，他又驱车前往寿张，参观了台前农业社的高额丰产田。在这期间，刘少奇曾要求山东五年建成社会主义。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   既然中央领导同志提出了明确要求，山东省委自然不敢怠慢。为了贯彻刘少奇的指示，山东省委立即在寿张召开各地
  </span>
  (
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   市
  </span>
  )
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>

  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   这是一次“插红旗、拔白旗”的会议。会上，各县自报夏收小麦的产量。这年范县的小麦亩产是
  </span>
  96
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   斤，与往常相比是一个不错的年份，谢惠玉就如实地报了这个数。可是，邻近的寿张报出的产量却是每亩
  </span>
  260
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>

  </span>
  3
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   倍。范县与寿张地理位置、自然条件都大致相同，可报出的产量却远远低于寿张。这样一来，受到了主持会议的一位省委领导的点名批评，认为范县是右倾保守，挡了五年建成社会主义之道。在这全面“大跃进”的年月，谁都怕被指责为右倾，这时，坐在台下的范县负责人十分紧张，赶紧检讨。接着，他们又给在家的县委常委通电话，要县委立即组织人马来寿张参观。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>

  </span>
  5
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   亩试验田，亩产小麦
  </span>
  2013
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   斤，全县小麦总产量比
  </span>
  1957
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   年增长了
  </span>
  62%
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   。这个头一开，范县的“卫星”也就越放越大，小麦亩产量一直放到了

"Times New Roman"'>
   仅过了两天，毛泽东就看到了《宣教动态》上的这个规划，并批示印发给即将召开的八届六中全会与会人员，还提议陈伯达、张春桥、李友九等人去范县看看。这时，毛泽东已开始对“大跃进”和人民公社化运动的“左”倾错误进行局部纠正，所以对范县的规划并不欣赏。他在此间召开的郑州会议上的讲话中说：“现在有那么一种倾向，就是共产主义越快越好，最好一两年就搞成共产主义。山东省范县说两年进入共产主义，说得神乎其神，我是怀疑的。”不久，陈伯达等人到了范县了解情况，并对范县一些过左的做法作了批评。范县也因大放卫星，号称粮食获得大丰收，结果实行高征购，甚至将群众的口粮都征购走了，到
  </span>
  1959
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   年春，不要说实现共产主义，就是吃饭都成了大问题，山东省给其紧急调拨了
  </span>
  500
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   万斤粮食才勉强渡过了
  </span>
  1959
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:"Times New Roman"'>
   年的夏荒，至于那个两年实现共产主义的规划，就只能收进历史的档案馆了。
  </span>
  <o:p>
  </o:p>
 </p>
 <p class="MsoNormal">
  <span lang="ZH-CN" style='font-family:宋体;mso-ascii-font-family:
"Times New Roman"'>
   此外，其他一些地方也制订了类似的规划，例如，山东莒县制订了大战二百天向共产主义过渡的规划，河南修武县制订了三年丰衣足食、五年内建成社会主义并向共产主义过渡的规划，等等。其内容与徐水、寿张、范县的规划大同小异，结局也完全相同。这说明，当年人们对实现共产主义是何等企盼，而对什么是共产主义的认识又是何等肤浅。
  </span>
